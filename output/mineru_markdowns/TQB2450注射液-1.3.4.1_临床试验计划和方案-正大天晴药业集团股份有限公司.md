# 1. 研究背景

# 1.1 前言

盐酸安罗替尼是我国自主研发的 1.1 类新药，是一类抗血管生成的新型小分子多靶点酪氨酸激酶抑制剂。ALTER 0303 研究证实，安罗替尼对比安慰剂用于晚期非小细胞肺癌（NSCLC）三线或以上治疗，可以同时显著延长患者的 PFS 和OS。基于这一研究，2018 年 5 月，CFDA 批准其上市。盐酸安罗替尼三线及三线以上治疗小细胞肺癌二期临床试验结果已经公布在 2018 年世界肺癌大会，相比安慰剂，安罗替尼明显延长晚期小细胞肺癌患者的 PFS 和 OS。同时期开展的盐酸安罗替尼临床研究显示，其在治疗晚期软组织肉瘤、甲状腺髓样癌、分化型甲状腺癌等项目也效果颇佳。

程序性死亡因子 1 及其配体（PD-1/PD-L1）是一对免疫共刺激因子。正常情况下，PD-1通过其配体PD-L1 发挥免疫调控作用。近年来，PD-1及其配体PD-L1 因其参与肿瘤免疫逃逸机制而受到关注。PD-1/PD-L1 信号通路的激活可导致免疫抑制性肿瘤微环境形成，使肿瘤细胞逃避机体免疫监视和杀伤，而阻断 PD-1/PD-L1 信号通路可逆转肿瘤免疫微环境，增强内源性抗肿瘤免疫效应。PD-L1 高表达于各种实体恶性肿瘤，包括非小细胞肺癌、黑色素瘤、肾细胞癌、前列腺癌、乳腺癌、胶质瘤等，其表达水平因肿瘤类型不同而不同，不仅能促进肿瘤细胞的生长，还可诱导 T 淋巴细胞的凋亡。

如今，众多免疫治疗临床研究在如火如荼地进行。在机制研究中，肿瘤血管异常会促进组织乏氧、乳酸增多，从而激活免疫抑制，抑制 T细胞功能。抗血管生成药物则通过诱导血管正常化、降低免疫抑制而增强效应免疫细胞的浸润。基于此，正大天晴药业集团股份有限公司现开展本项安罗替尼联合TQB2450(PD-L1）、化疗的临床研究。

# 1.12 小细胞肺癌一线治疗现状

肺癌是全世界范围内一种高发病率、高死亡率的恶性肿瘤。近年来，我国肺癌的发病率和死亡率呈快速增长。目前基于肺癌的生物学特性、治疗和预后，世界卫生组织（WHO）将其分为两大类：非小细胞肺癌（non-small cell lung cancer，NSCLC）和小细胞肺癌（small cell lung cancer，SCLC）。SCLC 占所有肺癌病例的 $1 5 { - } 2 0 \%$ ，无论从组织来源、生物学特征、对治疗的反应和预后等方面都与非小细胞肺癌迥然不同，所以被认为是一种全身性的疾病。SCLC 侵袭性高，预后较差，5年生存率不足 $5 \%$ ，不治疗的患者平均生存期仅2-4 个月。

大多数 SCLC 患者在初诊时失去了根治性治疗机会，但是联合化疗仍是广泛期SCLC 的有效治疗方法，可以改善症状，延长生存期。广泛期 SCLC 一线化疗缓解率为$4 0 \% - 7 0 \%$ ，中位生存期为 7-11 个月。尽快初始化疗缓解率高，但多数完全缓解的患者在3个月内病情进展，远期疗效差。EP 方案是治疗广泛期 SCLC 标准的一线治疗方案。在临床应用中，为了减轻胃肠道反应、肾毒性和神经毒性，通常用 CBP 替代 DDP，但CBP 的骨髓造血功能抑制风险较 DDP 大。

对于广泛期小细胞肺癌一线治疗，也进行很多其它联合化疗方案的临床研究，但均未取代标准治疗方案。2019 年 3 月，基于 IMpower133 研究结果，FDA 批准 Atezolizumab联合化疗用于广泛期小细胞肺癌一线治疗。IMpower133 研究是一项观察 Atezolizumab联合EC 一线治疗广泛期 SCLC 的双盲、随机、安慰剂对照研究。入组了既往未接受过治疗的广泛期SCLC 患者，不要求进行 PD-L1 检测。患者 1:1 随机分配接受 4个周期的卡铂(AUC $5 \ \mathrm { \ m g / m L / m i n \ \mathrm { \ I V } }$ ，第 1 天) $^ +$ 依托铂苷 $( 1 0 0 \mathrm { \ m g / m ^ { 2 } \ I V }$ ，第 1-3 天)联合Atezolizumab $1 2 0 0 \mathrm { ~ m g ~ I V }$ ，第 1 天) 或安慰剂，21 天为一个治疗周期，然后给予Atezolizumab 或安慰剂维持治疗，直至疾病进展、不可耐受的毒性或不再有临床获益。在维持治疗期间，方案允许患者进行预防性脑照射(PCI)。但不允许患者接受根治性胸部放疗(TR)。共同主要终点为 OS 和研究者评估的 PFS。2018年世界肺癌大会公布该研究结果，Atezo 组相对安慰剂组显著延长 SCLC 患者的 OS、mPFS，两组 OS 分别为 12.3 vs10.3 个月，mPFS 分别为 5.2 vs 4.3 个月。

# 3.5维持治疗期 PCI后继续治疗标准

SCLC 患者中 $5 0 \%$ 以上会发生颅内转移。有研究表明PCI可以降低 SCLC 患者脑转移的发生率。对于广泛期 SCLC，EORTC 开展的随机对照临床试验评估了 286例对初始化疗有反应的广泛期SCLC 患者接受/不接受PCI的差异。与对照组相比，PCI组减少了脑转移的症状（ $1 4 . 6 \% \mathrm { v s . } 4 0 . 4 \%$ ），增加了 1 年生存率（ $2 7 . 1 \% \mathrm { v s . } 1 3 . 3 \%$ ）。而在另一项日本开展的评价PCI对比观察治疗广泛期 SCLC 的随机3期临床研究中，尽管 PCI能降低脑转移发生几率（ $4 8 \% \mathrm { v s . } 6 9 \%$ ， $P { < } 0 . 0 0 0 1$ ），但并无总生存获益（11.6个月 vs 13.7个月； $P { = } 0 . 0 9 4$ ），因此对于广泛期 SCLC 的 PCI治疗应慎重选择。《2019 年中国临床肿瘤学会（CSCO）原发性肺癌诊疗指南》建议 PCI应在放化疗结束后 3周左右时开始，PCI 之前应该行脑增强 MRI 检查，如证实无脑转移，可开始 PCI。PCI 的剂量为 25Gy，2周内分10次完成。

在维持治疗期间，方案允许受试者进行预防性脑照射(PCI)。但不允许受试者接受根治性胸部放疗(TR)。PCI治疗后继续维持治疗，需要符合下列标准：

在28天内完成筛选期所规定的各项检查和评估，仍旧符合各项入选标准，不符合排除标准（除外一线治疗标准）；  
未接受除研究治疗、PCI以外的其他抗肿瘤治疗；  
PCI 洗脱 2 周；  
须经过本研究中心主要研究者的审核并批准。

若受试者 PCI 治疗后继续维持治疗，后续研究流程应参照方案要求继续随访，直至符合终止治疗的标准。

# 4.2 III 期期中分析

III期研究设立独立数据监督委员会（IDMC），进行期中数据分析，评估患者的风险/受益。独立数据监督委员会包括 2名独立的肿瘤学专家和 1名独立的统计学家，其独立于申办者和研究者之外开展工作。

计划在本次研究过程中，由独立数据监督委员会进行 1次安全性分析与 2次有效性分析。第1次有效性分析主要针对PFS，约在出现总 PFS 事件的1/2(147例)时进行，同时进行1次安全性分析，主要目的是进行安全性评价和样本量再估计，若方案中估计的样本量不能保证最终分析获得足够的把握度，将对样本量进行扩充;若样本量已足够，不再对现有样本量进行任何修改，具体内容见 IDMC 章程。第 2 次有效性分析主要针对OS，在 PFS 达到总事件数时进行，但不做终止试验或样本量再估计的决策，故本次分析消耗的一类错误定为 0.0001，OS 最终分析的一类错误为 0.050。IDMC 根据分析结果向申办方推荐试验项目进行与否、是否能够提前汇总资料进行申报或修改方案后继续研究的书面建议。申办方将根据 IDMC 的建议对本项研究项目采取相应的决策，如因安全性或缺乏有效性终止试验或因药物有效而提前总结申报，整个试验结束后再补充资料。

● 当 $\mathrm { A N C } > 1 . 5 { \times } 1 0 ^ { 9 } { / \mathrm { L } }$ 且血小板 $> 1 0 0 { \times } 1 0 ^ { 9 } / \mathrm { L }$ 时，可重新开始治疗。

● 在出现2级皮肤毒性的情况下，尚未出现3级药物相关性皮肤AE的受试者可重新开始治疗。

# 6.1 血清抗 TQB2450 抗体 (ADA)检测

免疫原性监测的时间点以TQB2450注射液给药时间为准；TQB2450延迟给药时，免疫原性采血相应延迟。

在使用 TQB2450 注射液治疗时，为了预防输液反应可以在每次输注 TQB2450 注射液前30-60min 内，使用 H1阻滞剂（苯海拉明 $5 0 \mathrm { m g \ I V }$ ，或同等药物）和对乙酰氨基酚（500 至 $6 5 0 \mathrm { m g }$ 口服或 IV）。用于控制输注反应或免疫相关不良事件所需的全身糖皮质激素治疗，在下一次研究药物给药前必须逐渐减量至少 4周，且减量后治疗剂量不会抑制免疫系统功能（强的松或同等药物 ${ \leqslant } 1 0 \mathrm { m g / }$ 天）。

# 9. 疗效评估

TQB2450注射液的同类药物在临床用药期间，患者存在假性进展，本次研究的疗效评估标准依据 RECIST 1.1。同时使用 iRECIST 标准（详见附件）对疗效进行确认。即按RECIST 1.1 标准判定为疾病进展(PD)的患者，按 iRECIST 标准进一步确认，从而决定是否进一步用药观察。

从第一周期第一天开始，每2个周期评估一次疗效，直至受试者发生肿瘤影像学证实的疾病进展，该评估不会因为受试者延迟或中断治疗而改变评估频率。如果受试者因为非疾病进展原因终止研究治疗，仍需要持续接受上述频率肿瘤影像学评估直到受试者开始新的抗肿瘤治疗、或影像学证据的疾病进展、或受试者主动退出、或受试者死亡，以先出现者为准。

肿瘤的影像学评价方式将由研究者决定，可采用 CT或MRI，但在整个研究期间评价方法、机器和技术参数应保持一致；如果无禁忌症，应使用造影剂。如果在首次给药前 14 天内已进行肿瘤评价，且在同一家医院使用同样的方法，可作为基线肿瘤评价。基线肿瘤评价应包括胸部、腹部和盆腔的增强 CT或MRI（造影剂过敏除外）。筛选期需行颅脑平扫 $^ +$ 增强/增强 MRI检查和骨扫描检查（筛选期没有骨转移的患者，研究医生根据症状决定）。所有可疑的病灶部位都应该行影像学检查。对于有骨转移的患者，应使用骨扫描对病灶进行随访。对于骨转移的患者，如果没有临床症状加重，不需每次肿瘤评估进行复查，如有临床症状加重，应及时复查。对于在计划中的下次评价开始前怀疑出现疾病进展的病例，应进行一次计划外的肿瘤评估。试验期间只做有病灶部位的影像学检查，期间有可疑部位，可以加查对应部位影像。

# 10．安全性评估

# 10.1 不良事件

不良事件（AE）指受试者接受研究药物治疗时出现的任何不良的医学事件，但不一定与治疗有因果关系。因此，AE 可以是与使用研究药物在时间上相关的任何不利的和非意求的征兆（包括异常的实验室发现）、症状或疾病，而不管其是否与药物有关。

AE 也包括方案规定的干预导致的并发症，例如，组织活检等有创操作导致的并发症；以及研究者认为之前存在的疾病在 AE报告期间出现恶化（肿瘤进展除外）的，也认为是AE，但疾病进展不是 AE。

研究人员应详细记录患者所发生的任何不良事件。不良事件的记录包括：不良事件及所有相关症状的描述，发生的时间，严重程度，持续时间，采取的措施和转归情况。

# 10.2 不良事件的评价

不良事件性质和严重程度的评价标准遵照美国国立癌症研究所的常见毒性反应标准 [NCI –CTC v5.0]进行。

# 10.3 不良事件的记录

在方案规定的AE 报告期间，研究者有责任搜集所有的 AE（包括 SAE），并记录在CRF/eCRF表中。在记录 AE时，研究者应该使用正确规范的医学术语，避免口语和缩写。需记录AE的开始时间、NCI CTCAE v5.0分级的最高程度、结束时间、与研究药物相关性、与研究影响、有无伴随治疗以及恢复情况。

# 诊断 vs症状和体征

如果已经有诊断，应在 CRF/eCRF上记录诊断而非单个的症状和体征（例如记录肝衰竭，而非黄疸，转氨酶升高和扑翼样震颤）。但是如果在报告时症状和体征不能归为单个诊断，每个单独的事件都应作为 AE在CRF/eCRF上记录。如果后来诊断确立，应在CRF/eCRF上更新以记录诊断。

# 继发于其它事件的不良事件

总的来说，继发于其它事件的 AE（例如由其它事件导致或临床后遗症）应记录其原发事件，除非继发事件严重程度更重或为SAE。但是具有显著临床意义的继发事件，如果和原发事件的发生时间不同的，应在 CRF/eCRF中记录为独立的 AE；如果事件之间的关联关系不清楚，则应分别记录原发事件和继发事件。

# 持续存在的、间歇发生的或单独的不良事件（不良事件的频度）

持续存在的AE（continuous AE）是指在整个过程中没有缓解而一直存在的 AE，例如，持续5 天的上呼吸道感染。这种 AE在CRF/eCRF上只应记录一条。严重程度评估时，应记录该事件全过程中最严重的程度。

间歇发生的AE（intermittent AE）是指在整个过程中症状、体征或实验室检查指标等时有变化或缓解，但没有发生具有显著临床意义结局的 AE，例如，持续多日的恶心呕吐，期间有相对地缓解；发生高血压的受试者多次血压检测中，虽有间歇性缓解但高

血压病程相对连续。这种 AE在CRF/eCRF上可只记录一条。强度评估时，应记录该事件全过程中最严重的程度。

单独的AE（Single AE）是指逻辑上只能单独发生的，或试验中仅独立地发生一次的AE，例如，服药期间受试者发生了跌倒事故；受试者在试验期间仅发生的一次呕吐。这种 AE 在 CRF/eCRF 上只记录一条。

需要说明的是，如上述 AE已具有显著临床意义的恢复，但后来又发生了相同的AE且认为后者与前者没有病程上的连续性，则应在CRF/eCRF中分别记录事件的发生。

# 实验室检查异常或生命体征异常

所有实验室检查结果可记录在 CRF实验室结果页上。并非所有实验室检查异常/生命体征异常都应记录为 AE，研究者有责任审查所有的实验室异常结果和异常的生命体征，并进行医学判断是否应记录为 AE。任何上述异常当有显著临床意义，例如满足一条或多条以下情况时，应该记录为 AE：

伴随临床症状   
导致研究用药改变（例如，剂量调整、暂时或永久停止）  
 需要医学干预或改变合并治疗（例如，合并药物、治疗或处理的增加、暂停、终止或其他改变）  
研究者判断有显著的临床意义

如果有临床意义的实验室检查异常或生命体征异常是某个疾病或综合征的表征（例如肝功能损害导致的ALT/AST 和血胆红素升高），仅在 CRF/eCRF的不良事件表上记录诊断（即肝功能损害）。反之，则应在 CRF/eCRF的不良事件表上记录实验室检查异常或生命体征异常，并表明检查值是高于或低于正常范围。如果实验室检查异常或生命体征异常有与之对应的标准临床术语，应在 CRF/eCRF中记录临床术语（例如血钾升高到7.0mmol/L，应记录为“高钾血症”）。

# 死亡

在记录死亡事件时，如果存在导致死亡的 AE，应在CRF/eCRF中使用单个医学概念记录导致死亡的AE，并将该事件作为SAE报告；如果死亡的原因不明，应在CRF/eCRF的AE表中记录“不明原因死亡”，并先将“不明原因死亡”作为SAE报告，再进一步调查确切的死亡原因，待得知导致死亡的原因后再更新记录及 SAE报告。

# 既往存在的医学状态

受试者在试验筛选期已有的症状/体征在进入试验后，仅在出现严重程度、频率、性质的加重（所研究的疾病状况恶化除外）时才作为 AE予以记录。应在记录中体现相对于既往状态的改变，例如“头痛频率增加”，“高血压加重”等。

# 住院、住院时间延长

以下情况导致住院或者住院时间延长，不应该报告为 SAE：

 根据方案要求计划的住院或住院时间延长（例如为了给药、疗效评估等）  
 因参加研究前已存在且未发生改变的医学状况而住院，例如在参加研究前已安排的择期手术或治疗参加研究后受试者如期住院接受手术或治疗，则不认为是不良事件。然而，如果研究中已有疾病的状况恶化（如比原计划要提前进行手术或治疗），那么由于疾病恶化而需要住院进行手术或治疗，该状况的恶化将被认为是 SAE。

# 手术

如果明确了手术所治疗的疾病，则应将该疾病记录为 AE，而非手术本身（例如，受试者经历了腹股沟疝修补术，则应记录“腹股沟疝”，而非“腹股沟疝修补术”）；但如果手术原因不明确，则可记录手术名称为 AE（例如，受试者经历腹腔探查术，可记录“腹腔探查术”为AE）。

# 妊娠

在研究中，如果女性受试者或男性受试者的女性伴侣出现妊娠，应立即通知研究者。研究者应在得知妊娠事件后 24小时内报告给申办方。女性受试者妊娠的，应立即停用研究药物。研究者应与受试者讨论继续妊娠的风险和可能对胎儿造成的影响。对受试者的监测应持续至妊娠结束。在末次研究药物给药30日内发生的妊娠都应报告给研究者。

无论是人工还是自然流产都应该作为 SAE记录和报告。服用过研究药物的女性受试者或男性受试者的女性伴侣所生的婴儿，如有任何先天异常或出生缺陷，都应该作为SAE记录和报告。

# 疾病进展

如果发生的事件明确与原发肿瘤预期的进展模式一致，则不应视为 AE。仅仅由于该疾病进展导致的住院也不应作为 SAE。若症状不能明确完全由于疾病进展引起，或与预期的肿瘤进展模式不吻合，相关临床症状可记录为 AE，符合SAE 的应报告SAE。

# 10.4 不良事件的随访

研究者应随访所有的AE，直至下列任何一种情况发生，

AE缓解或改善到基线水平；

研究者确认预期不会出现进一步改善；  
受试者死亡；  
受试者已失去联络；  
研究者确认AE与研究治疗无关；  
受试者开始新的抗癌治疗；  
不再收集临床或安全性数据，或数据库最终关闭。

每个AE的最终结局（包括 AE缓解或死亡的日期）必须在 CRF/eCRF上记录。

# 11. 生活质量评价

通过问卷从多方面收集患者的生活质量进行评估。健康相关生活质量问卷的评估在本项目中作为一项探索性的内容，旨在尽量早期开始收集药物应用的各种信息。

健康相关生活质量问卷需要在规定的时间点完成。患者应该在进行各种临床检查和临床评估之前，在被告知任何新的疾病信息以及开始任何新的治疗之前完成问卷，以避免答案受到医生或者新病情的影响。问卷将作为原始资料保存。

研究者将审核患者的问卷并评估患者填写问卷时的临床状况，如发现有临床评估中未发现的不良事件，也需记录在CRF中。

欧洲五维健康量表 EQ-5D 由问卷和效用值换算表两部分组成。问卷调查结果可以用来描述人群的健康状况和获得 EQ-VAS 得分，使用效用值换算表则可进一步获得EQ-5D 指数得分。

问卷又可分为 EQ-5D 健康描述系统和 EQ-VAS 两个部分。EQ-5D 健康描述系统包括五个维度：行动能力（Mobility）、自己照顾自己能力（Self-Care）、日常活动能力（Usual Activities）、疼痛或不舒服（Pain/Discomfort）、焦虑或抑郁（Anxiety/Depression）。每个维度又包含三个水平：没有任何困难、有些困难、有极度困难。EQ-VAS 是一个长20 厘米的垂直的视觉刻度尺。顶端为 100 分代表“心目中最好的健康状况”，底端 为0分代表“心目中最差的健康状况”。

# 12．数据管理及资料保存

# 12.1 数据管理

# 12.1.1 数据录入

本试验采用电子数据采集（EDC）系统进行数据采集。数据录入人员应根据《eCRF填写指南》将原始资料信息及时、准确、完整、规范地录入 EDC。只有经过 EDC 操作培训合格的研究者或CRC 等人员被授权后才能执行数据录入工作。

# 12.1.2 数据核查

监查员根据源文件及时对EDC 数据进行核查，确保数据准确、完整、一致、规范。数据管理员根据数据核查计划对 EDC 数据进行审核，以确保的数据逻辑正确、完整及规范。监查员与数据管理员发现问题及时在EDC 发起质疑，研究者或 CRC 及时回答质疑或数据更正，监查员与数据管理员确认无误后关闭质疑。

# 12.1.3 数据清理

数据清理包括 EDC 系统自动核查、人工核查与 SAS 程序清理等。研究者或 CRC对被质疑的数据进行确认，确认后修改数据或回复质疑。监查员或数据管理员对修改后的数据再次进行核查，若质疑解决则关闭质疑，若质疑仍存在，监查员或数据管理员需再次质疑以获得进一步确认，直至质疑最终解决。

# 12.1.4 电子签名

数据核查结束后，由研究者最终确认已录入数据的真实性和准确性，在 EDC 系统中对数据进行电子签名。进行电子签名的研究者需在实施前签署《电子签名声明》，声明电子签名与书面手写签名具有同等的法律效应。

# 12.1.5 数据库锁定

所有试验数据录入完成、数据核查与审核完成、电子签名完成、医学核查完成、数据核查报告定稿后，召开数据库锁定前会议，共同确认锁定相关事宜。数据管理员确认数据库锁定清单的工作完成后，执行数据库锁定程序。数据库锁定后原则上不再解锁。

# 12.1.6 数据移交

数据库锁定后，数据管理员导出数据集提交统计分析人员，导出受试者eCRF电子病例提交各研究中心存档。

# 13. 统计分析

# 13.1统计分析数据的选择

●意向治疗分析集(Intention to treat Analysis Set, ITT)：按照意向性分析（ITT）原则，对所有随机化的病例进行疗效分析。治疗分组按随机分组进行分析。

ITT为主要分析集，用于所有有效性和基线特征分析。

● 符合方案集(Per-protocol Set)：所有疗程完成 2个周期及以上、至少有 1次影像学评价者、符合试验方案、依从性好、试验期间未用禁止用药、完成病例报告表规定填写内容的病例。

PPS 将用于主要有效性指标和关键的次要有效性指标的符合方案分析。在数据库锁定前，由主要研究者、统计师及申办者在数据审核会共同协商确定最终的符合方案集。

● 安全性分析集(Safety Analysis Set)，所有入组病例，至少使用过一次试验用药，并有用药后安全性记录的全部患者，均属于安全性分析集。

SS 数据集用于所有安全性分析。治疗分组按实际分组进行分析。

本试验中，基线资料的分析对 ITT进行，主要疗效指标的分析同时对 ITT 和PPS 进行，以 ITT结论为准，对实验室检查资料及不良事件和不良反应资料采用 SS 分析。

# 13.2 统计分析计划

# 13.2.1 一般原则

对于连续型变量，将列出未缺失的受试者例数、均数、标准差、中位数、最小值和最大值。最小值和最大值的小数位数将与数据库中记录保持一致。均数、标准差、中位数将比数据库中记录的原始数据多保留一位小数。

对于分类变量，将以频数表的形式（频数和百分数）列出。百分数将保留 2位小数。

# 13.2.2 检验水准

研究中设立 3 组：试验组 1（TQB2450+安罗替尼 $^ +$ 卡铂 $^ +$ 依托泊苷）、试验组 2（TQB2450 空白 $^ +$ 安罗替尼 $^ +$ 卡铂 $^ +$ 依托泊苷）、对照组（TQB2450 空白 $^ +$ 安罗替尼空白$^ +$ 卡铂 $^ +$ 依托泊苷）。研究的主要疗效指标为无进展生存期(PFS)和总生存期(OS)，其中PFS 更为重要，OS 次之。研究采用序贯检验方法，先对试验组 1 vs 对照组的 PFS 进行假设检验，若PFS 有统计学意义，再对试验组 2 vs 对照组的PFS进行假设检验，若有统计学意义，再按顺序对试验组1 vs 对照组的 OS 和试验组2 vs 对照组的 OS 进行假设检验，将一类错误控制在 0.05。

本研究考虑在 PFS 事件数达到 1/2 时进行一次安全性和有效性期中分析，目的是进行安全性评价和样本量再估计，该估计过程将由 IDMC 负责。采用 O'Brien-Fleming 消耗函数法校正一类错误，两次比较消耗的一类错误分别为0.003和0.049。OS 在PFS 最终分析时进行一次期中分析，消耗的一类错误定为 0.0001，OS 最终分析的一类错误为0.050。最终分析的检验顺序及一类错误如下图所示：

# 13.3研究人群

# 13.3.1 病例分布情况

各中心受试者入组情况、脱落病例、完成预计疗程、提前中止的描述：例数、百分比；

各数据集的病例分布情况：汇总各分析人群的受试者分布情况，汇总各分析人群剔除受试者的原因。

对被各分析人群剔除的受试者进行列表（包括方案违背情况以及剔除原因）

脱落、提前中止病例逐一进行描述：用药情况、提前退出试验的原因等。

# 13.3.2 方案违背与偏离

汇总与描述方案违背和偏离的病例情况。主要和次要方案违背的定义见附录。

# 13.3.3 两组受试者特征分析

年龄：采用均数、标准差、最大值、最小值、中位数描述。

性别：计算构成比。

身高、体重：采用均数、标准差、最大值、最小值、中位数描述。

肿瘤病史：TNM 分期、抗肿瘤治疗史、吸烟史计算构成比。

基线生命体征（体温、心率、呼吸、血压）：基线定义为用药前最近一次非缺失测量值。

基线体力状况ECOG 评分：采用均数、标准差、最大值、最小值、中位数描述。

# 13.4 有效性评价

# 13.4.1 主要疗效指标的分析

 无进展生存期（PFS）

采用 Kaplan-Meier 法估计中位 PFS，并绘制生存曲线图。

PFS的组间比较采用分层log-rank检验，同时采用COX比例风险模型估计组间HR。模型中考虑ECOG评分(0/1分)、有脑转移(是/否)、有肝转移(是/否)作为协变量。

对PFS 进一步根据分层因素进行亚组分析。

 总生存期（OS）

采用 Kaplan-Meier 法估计中位 OS，并绘制生存曲线图。

OS组间的比较采用分层log-rank检验，同时采用COX比例风险模型估计组间HR。模型中考虑ECOG评分(0/1分)、有脑转移(是/否)、有肝转移(是/否)、出组后治疗(是/否)作为协变量。

结合我国广泛期小细胞肺癌治疗现状，对照组受试者进展出组后可能接受其他抗肿瘤治疗，对本研究总生存期的估计产生较大影响，故考虑将出组后治疗作为协变量纳入到对OS 的分析模型中，并以此作为结论的依据。

对OS 进一步根据分层因素进行亚组分析。

# 13.4.2 次要疗效指标的分析

 客观缓解率（Objective response rate, ORR）：

计算各组客观缓解例数（ $\mathrm { P R + C R }$ ）占各组总病例数的比率及 $9 5 \% \mathrm { C I }$ 。ORR的95%CI基于F分布的精确二项方法计算。组间ORR的比较采用分层Mantel-Haenszel检验，分层因素同本方案4.3部分。

 缓解持续时间 ( Duration of response, DOR)

采用 Kaplan-Meier 法估计中位 PFS 及其 $9 5 \% \mathrm { C I }$ ，并绘制生存曲线图。

DOR 的组间比较采用分层log-rank 检验。仅对缓解者进行分析。

 疾病控制率（DCR）：

计算疾病控制例数（ $\mathrm { C R + P R + S D }$ ）占总病例数的比率及 $9 5 \% \mathrm { C I }$ 。DCR 的 $9 5 \% \mathrm { C I }$ 基于F分布的精确二项方法计算。组间 DCR 的比较采用分层Mantel-Haenszel 检验，分层因素同本方案4.3部分。

# 13.5安全性评价

# 13.5.1 药物暴露量与依从性情况

对各组药物暴露量采用均数、标准差、最大值、最小值、中位数描述。

总结受试者暴露于研究药物治疗的情况，各组病人完成周期数的情况、治疗期间剂量调整情况、治疗期间累计调整剂量次数的情况等。

对治疗期内研究药物治疗时间，研究药物服用总剂量及日平均剂量和研究药物剂量依从性进行统计描述。

研究药物剂量依从性将基于 eCRF记录的每天实际服用的研究药物总剂量和方案规定研究药总剂量进行计算。

各研究药物治疗时间、研究药物服用总剂量及日平均剂量和研究药物剂量依从性的比较采用单因素方差分析，依从性分类的比较采用卡方检验或 Fisher 精确概率法。

# 13.5.2 不良事件

不良事件数据，根据开始编码时现行版本的 MedDRA进行编码后，在统计分析中进行处理。

 汇总不良事件、首次用药前的不良事件、治疗期间不良事件、治疗期间非预期不良事件、治疗期间重要不良事件、治疗期间特别关注的不良事件、治疗期间3级及以上不良事件、治疗期间严重不良事件、治疗期间与研究药物相关的不良事件、治疗期间与研究药物相关的 SAEs、治疗期间导致剂量调整、治疗永久停止、试验终止、患者死亡的不良事件的例次、例数和发生率，按照 SOC、PT 分类汇总。  
 按照PT分类汇总治疗期间发生率 $5 5 \%$ 的不良事件、药物相关不良事件。  
 按照PT分类汇总治疗期间 CTCAE分级为 3级或4级的药物相关不良事件。  
 按照PT分类汇总治疗期间发生率 $2 1 0 \%$ 的不良事件、药物相关不良事件。

将按系统器官分类（SOC）和首选术语（PT）将不良事件总结于频率表中。分系统、症状/体征计算发生率（例数的计数：至少发生过一次某一种不良事件的受试者例数）。

按照SOC、PT分类分别计算 NCI CTCAE 5.0 版各级严重程度及与研究药物的相关性的例次、例数和发生率（重复出现相同 AE，以最严重的一次参与例数和发生率的严重程度分析）。

提供发生不良事件的受试者清单、SAE 的受试者清单。

必要时用Fisher确切概率法，比较各组不良事件发生率及与研究药物有关的不良事件发生率。

特别关注的不良事件中位首次发生时间。

# 13.5.3 生命体征

采用均数±标准差、最大值、最小值、中位数描述各组治疗前后的测量值和变化值。

# 13.5.4 实验室检查指标

血常规、血生化、甲状腺功能、凝血功能、淀粉酶、脂肪酶采用均数±标准差、最大值、最小值、中位数描述各组治疗前后的测量值和变化值，必要时组间比较采用单因素方差分析，组内比较采用配对 t 检验。采用交叉分类表描述治疗前后正常、异常的变化情况。

尿常规：采用交叉分类表描述治疗前后正常、异常的变化情况。

便常规：采用交叉分类表描述治疗前后正常、异常的变化情况。

描述异常变化的受试者中的“异常、并有临床意义”的比例，其中异常有无临床意义由研究者判断。

# 13.5.5 心电图

心电图：根据研究者判断的正常、异常，描述治疗前后正常、异常的变化情况。

心率、PR 间期、QRS间期、QT间期和QTc 采用均数±标准差、最大值、最小值、中位数描述两组用药前后的测量值和变化值。组内比较采用配对 $t$ 检验，组间比较采用单因素方差分析。心电图总体评估结果采用交叉分类表描述用药前后正常、异常的变化情况。描述异常变化的受试者中的“异常有临床意义”的比例，其中异常有无临床意义由研究者判断。列表呈现给药后异常清单。

# 13.5.6 体格检查

描述治疗前后正常、异常的变化情况。

# 13.5.7 合并用药

汇总研究期间合并用药（包括筛选期合并用药的任何改变、筛选后新增的合并用药以及随访期的合并用药）的使用情况，各药物的使用频率。必要时组间合并用药发生率的比较采用卡方检验或者 Fisher精确概率法。

将采用 WHODrug（2018 英文版）进行 ATC 分类编码，按照解剖学主族和治疗学亚组基于SS 人群进行频数汇总描述。

# 附件二 美国退伍军人医院肺癌研究组（VALG）分期标准

根据 SCLC 生物学特点及预后，1973 年美国退伍军人医院肺癌研究组 VALG 的两分期为局限期（limited-stage disease，LD）和广泛期（extensive-stage disease，ED）。

局限期定义为：病变限于一侧胸腔，且能被纳入一个放射治疗野内。

广泛期定义为：病变超过一侧胸腔，且包括恶性胸腔和心包积液，或血行转移。

# 附件五 免疫实体瘤疗效评价标准（iRECIST）

----评估免疫疗法的临床试验中使用的疗效评估指南（附录）（节选）

# 1. iRECTIST 疗效评估

免疫疗法可能会促进免疫细胞的浸润，导致肿瘤体积一过性增大，或让本无法检测到的病变被检测到。本标准和RECIST 1.1 在许多方面完全一致，但对肿瘤负荷增大或出现新病灶等可能并未反映肿瘤真实进展情况的案例评估进行了调整。

下文将对关键的不同之处进行说明。所有使用iRECIST评估的疗效指标前会有前缀“i”。iRECIST各时间点疗效和最佳整体疗效将分开记录。

# 1.1 确认进展

与RECIST 1.1不同，iRECIST 需要对疾病进展进行确认，同时引入iUPD（未确认的进展）和iCPD（确认的进展）标准。确认进展的扫描最早可在 iUPD 后第 4 周，并不晚于 iUPD 后第 8 周进行。

如果肿瘤负荷对比末次测量结果仍有增大，则判定为 iCPD，具体证据参考以下一条或多条：

 符合RECIST 1.1定义的靶病灶、非靶病灶或新病灶的疾病进展（和最低点相比）基础上，肿瘤负荷继续增大：

• 靶病灶恶化，表现为测量值之和的绝对值至少增大 $5 \mathrm { m m }$ ；  
• 非靶病灶进展明确，表现为肿瘤负荷增大；  
• 既往发现的新病灶增大（新靶病灶测量值之和绝对值至少增大 $5 \mathrm { m m }$ ）或发现其他新病灶。

 既往未发现肿瘤进展的其他病灶（靶病灶、非靶病灶或新病灶）符合RECIST 1.1 进展标准，包括发现其他新病灶

如果iUPD 未在下一次评估中得到确认，则将记下相应的疗效评估结果（如果仍符合 iUPD 且没有恶化，记为 iUPD，如果和基线相比符合 iSD、iPR 或 iCR 的标准，则记为相应的结果）。如表 2所示，在iUPD的下次疗效评估未达到 iCPD的情况下，既往 iUPD并不影响在之后的评估时间点或在最佳整体疗效时记录iCR、iPR或 iSD。

# 1.2 新病灶

新病灶应按照RECIST 1.1的标准进行测量和评估（最多 5个病灶，每个器官不超过 2个，最长径至少 $1 0 \mathrm { m m }$ （淋巴结病灶短径至少 $1 5 \mathrm { m m } )$ ）），且明确记为新靶病灶及新非靶病灶以区别于基线的靶病灶和非靶病灶。

新病灶应符合新靶病灶和新非靶病灶的定义，以记为 iUPD（或 iCPD）。对此类靶病灶的测量不应计算在基线时定义的靶病灶测量值之和中。这些新靶病灶的测量结果应记录在单独的 CRF中。

以下情况可对新病灶的疾病进展进行确认：在 iUPD 之后至少 4 周（不多于 8 周）进行的影像学评估发现新靶病灶测量值之和的绝对值增加至少 $5 \ \mathrm { m m }$ 或新非靶病灶的体积增大（不需要明确的增大）或出现其他新病灶。

所有患者将按表3规定对开始治疗到治疗结束的免疫最佳整体疗效（iBOR）进行定义。

# 2. 缓解和疾病稳定时间（RECIST 1.1 和 iRECIST）

缓解时间将从符合CR/PR或iCR/iPR标准（以先发生为准）的时间开始计算直到复发或疾病进展，研究期间的最小测量值（包括基线）同时记为参考。
