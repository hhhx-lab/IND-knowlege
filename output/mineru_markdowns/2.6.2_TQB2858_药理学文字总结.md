# 2.6.2.1 概要

本品是一种双功能的融合蛋白（研发代号 TQB2858），临床拟用于晚期恶性实体/血液肿瘤患者静脉输注给药。

本品为 PD-L1/TGF- $\beta$ 双功能融合蛋白，由两个部分构成的。一端是能够识别结合 PD-L1 的抗体结构（Y），是本司的 TQB2450，处于临床 III 期的 PD-L1 抗体；另一端是可以结合 TGF- $\beta$ 的 TGF- $\beta$ 受体 II 型融合蛋白（Trap），可以捕获游离的TGF-β1。PD-L1 表达在多种肿瘤组织上，与 T 细胞上的受体 PD-1 结合负向调控免疫应答。多种因素可诱导肿瘤细胞上 PD-L1 的表达，同时表达的 PD-L1 有利于肿瘤的发生和生长。转化生长因子- $\cdot \beta$ （transforming growth factor- $\beta$ ，TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ ）是一种多功能细胞因子，共有 3 个亚型，即 TGF-β1、TGF- $\cdot \beta 2$ 、TGF- $\cdot \beta 3 _ { \circ }$ 。在晚期肿瘤阶段，肿瘤微环境中癌细胞、间质成纤维细胞和内皮细胞等大量释放的 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 能够使癌细胞具有侵袭性、扩散性、干细胞特性以及对治疗产生耐药性。本品能够同时阻断 PD-1/PD-L1 通路及TGF- $\beta$ 通路从而促进免疫细胞对肿瘤细胞的杀伤作用。

因此，在靶向中和肿瘤微环境的 TGF- $\beta$ 基础上抑制 PD-1/PD-L1 通路，可以使 T 细胞恢复活性，增强免疫应答，更有效地提高抑制肿瘤发生和发展的效果。

体外药效学结果表明：本品选择性与人 $\mathrm { T G F } \beta$ 结合、人 PD-L1 结合，明显抑制TGFβ1 刺激的荧光素酶报告基因表达，明显促进 SEB 刺激或 MLR 中 $\mathrm { C D 4 ^ { + } T }$ 细胞分泌IFN-γ；无明显 ADCC；本品在体外于 PBMC 共同孵育，细胞因子产生情况与同型对照组 hIgG1 相当。总体而言，本品对 TGFβ 的体外活性与 IgG1-TGFβ 相当，对 PD-L1 的体外活性与 TQB2450 相当。

体内药效学结果表明：本品对小鼠结肠癌 MC38/hPD-L1 皮下移植瘤生长有明显抑制作用，且呈现剂量依赖性。

安全药理试验伴随在食蟹猴 4 周重复给药毒性试验，考察了单次静脉注射本品对食蟹猴体温、呼吸频率及心血管系统的影响，结果表明食蟹猴单次静脉注射本品对其体温、呼吸频率及心血管系统均无损害作用。

# 3. 本品与高表达人 PD-L1 的 MC38/hPD-L1 细胞结合

将不同浓度的本品与高表达人 PD-L1 的 MC38/hPD-L1 细胞共同孵育，检测 OD 值，计算结合的 EC50。结果显示：本品与高表达人 PD-L1 的 MC38/hPD-L1 细胞明显结合，$\mathrm { E C } _ { 5 0 }$ 及 $\mathrm { E } _ { \mathrm { m a x } }$ 分别为 $2 4 7 . 5 \pm 0 . 1 \mathrm { n g / m l }$ 及 $0 . 9 8 \pm 0 . 1 0 \mathrm { O D } _ { \circ }$ 。

# 4. 本品对 SEB 刺激 PBMC 分泌 IFN-γ 的作用

将 SEB 与不同浓度的本品或 TQB2450 与 PBMC 共同孵育 72h，检测上清液中 IFN-γ 的含量。结果显示：本品显著促进 SEB 刺激 PBMC 分泌 IFN-γ，推测本品通过结合PD-L1，抑制 PD-L1 活性，最终促进 IFN-γ 分泌。

# 5. 本品对抗 CD3/CD28 抗体刺激 PBMC 分泌 IFN-γ 的作用

将抗人 CD3/CD28 抗体与不同浓度的本品或 TQB2450 与 PBMC 共同孵育 72h，检测上清液中 IFN-γ 的含量。结果显示：本品显著促进抗人 CD3/CD28 抗体刺激 PBMC分泌 IFN-γ。

# 6. 本品对混合淋巴细胞反应（MLR）中 IFN-γ 分泌的影响

将不同浓度的本品与 TQB2450 加入将 $\mathrm { C D 4 ^ { + } T }$ 细胞与成熟的 DC 细胞混合体系，作用 5 天后检测上清液中 IFN-γ、IL-2 的含量。结果显示：本品能显著促进 MLR 中 IFN-γ及 IL-2 的分泌。

# 7. 本品对人 TGFβ1 刺激的报告基因 luc2P 表达的作用

采用报告基因法，检测不同浓度的本品对TGFβ1刺激的luc2P表达的抑制作用。

结果显示：本品显著抑制 TGFβ1 刺激 luc2P 的表达， $\mathrm { I C } _ { 5 0 }$ 分别为 $4 0 8 . 2 \pm 8 . 3 \mathrm { n g / m l _ { \circ } }$ 。

# 8. 本品的 ADCC

本品无明显的 ADCC 作用。

# 9. 本品注射液对人外周血单个核细胞体外细胞因子释放的影响

采用液相法及固相法评估不同浓度下本品与人 PBMC 共孵育 24h、48h 后细胞因子的产生情况，结果表明：细胞因子产生情况与同型对照组 hIgG1 相当。

# 2.6.2.2.2 体内药效学

# 1. 本品对小鼠结肠癌MC38/hPD-L1皮下移植瘤的疗效

50只PD-L1人源化小鼠皮下接种 3105MC38/hPD-L1细胞/只，构建小鼠结肠癌模型，待肿瘤长到 $5 0 { \sim } 7 0 \mathrm { m m } ^ { 3 }$ ，根据肿瘤体积分组，10 只/组，静脉注射（IV）药物，每 2天 1 次（Q2D），共 6 次，注射体积 $0 . 1 \mathrm { m L } / 1 0 \mathrm { g }$ 体重，考察本品对肿瘤生长的影响，具体指标为 T/C%或肿瘤生长抑制率 $\%$ （ $\mathrm { \Delta T G I \% }$ ）。

本品 (3.7、 $1 2 . 3 \mathrm { m g / k g }$ ，IV， $\mathrm { Q 2 D } { \times } 6 $ ) 对 MC38/hPD-L1 皮下移植瘤生长有明显抑制作 用 ， 抑 瘤 率 分 别 为 $5 6 \%$ 和 $6 9 \%$ ， 呈 现 剂 量 依 赖 性 ； 对 照 药 物 $\mathrm { I g G 1 \mathrm { - } T G F \beta }$ (2.3mg/kg，IV， $\mathrm { Q 2 D { \times } 6 ) }$ ) 对 MC38/hPD-L1 皮下移植瘤的抑瘤率为 $4 \%$ ，没有明显疗效；TQB2450(10mg/kg ， IV ， $\mathrm { Q 2 D } { \times } 6 $ ) 与 IgG1-TGFβ (2.3 mg/kg ， IV ， $\mathrm { Q 2 D } { \times } 6 $ ) 合 用对 MC38/hPD-L1 小鼠皮下移植瘤的抑瘤率为 $7 1 \%$ ，与等摩尔剂量组本品（ $\cdot 1 2 . 3 \mathrm { m g / k g }$ ，IV， $\mathrm { Q 2 D } { \times } \mathrm { 6 } ,$ ）抑瘤效果相当；荷瘤小鼠对以上药物均能够较好耐受，没有明显体重下降等症状发生。

# 2.6.2.4 安全药理学

# 2.6.2.4.1本品对食蟹猴心血管系统的影响

本试验为伴随“食蟹猴静脉注射 TQB2858 注射液 4 周毒性试验”进行的安全药理学研究，本试验使用 40 只食蟹猴，分为 4 组，每组 10 只，雌雄各半，分别静脉注射20、60、200mg/kg 本品及空白对照，每周给药 1 次，连续 4 周，共给药 5 次，停药恢复 4 周。每天观察各组猴一般状况；首次给药前及给药后 1、8、24、72、168 小时，采用 Temp-14 热敏电阻温度计检测肛温；采用大动物无创生理信号遥测系统检测呼吸频率及 II 导联心电图；采用无创血压计测量血压（收缩压、舒张压、平均动脉压）。定性分析心电图波形，并以 ecgAUTO 软件分析每个时间点处 30 秒时段内稳定连续的心

电图，指标包括心率、R-R 间期、P 波时间、P-R 间期、QRS 波时间、Q-T 间期、校正Q-T 间期。

试验结果显示，在本试验条件下，食蟹猴单次皮下注射 20、60、 $2 0 0 ~ \mathrm { m g / k g }$ 的TQB2858 注射液，对各组猴体温、呼吸频率、II 导联心电图及血压均未见明显影响。

# 2.6.2.6 讨论和结论

综合本研究的结果，本品选择性与人 TGFβ 结合、人 PD-L1 结合，明显抑制TGFβ1刺激的荧光素酶报告基因表达，明显促进SEB刺激或MLR 中 $\mathrm { C D 4 ^ { + } T }$ 细胞分泌IFN-γ；无明显ADCC；本品在体外于PBMC共同孵育，细胞因子产生情况与同型对照组 hIgG1 相当。本品对小鼠结肠癌 MC38/hPD-L1 皮下移植瘤生长有明显抑制作用，且呈现剂量依赖性。食蟹猴单次静脉注射本品对食蟹猴的体温、呼吸频率及心血管系统均无损害作用。
