# ".$ 品种开发背景

在肿瘤微环境中，PD-L$/PD-$ 通路和 TGF- $\beta$ 信号通路在肿瘤免疫逃逸中起了十分重要的作用。PD-$/PD-L$ 是一对负性免疫共刺激分子，PD-L$ 也被称为白细胞分化抗原#-,(CD#-,)或 B- 同源物 $(B--H$)，是 B- 家族成员，PD-$ 的配体，属于 I 型 $4 0 \mathrm { k D a }$ 跨膜蛋白，具有负向调节免疫应答的作用。PD-L$ 主要表达于 T 细胞、B 细胞、巨噬细胞和树突状细胞(dendritic cell, DC)上，在活化后细胞上的表达能够进行上调。肿瘤细胞表面表达的PD-L$ 通过与肿瘤浸润淋巴细胞表面的 PD-$ 分子结合，抑制淋巴细胞的功能，是导致肿瘤发生免疫逃逸的重要原因之一。因此对 PD-$ 与 PD-L$ 之间相互作用的研究有望为肿瘤免疫逃逸机制以及肿瘤的靶向治疗提供重要的实验依据。

TGF- $\beta$ 是由两个结构相同或相近的、分子量 $#.&kDa 亚单位借二硫键连接的双体。TGF- $\beta$ 信号通路在肿瘤发生发展的过程中具有两面性：早期抑制肿瘤生长，而在晚期则表

现为促进肿瘤生长。肿瘤早期，TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 信号通路的减弱主要是因为抑制性Smad-和另外两个负性调控因子（SKI 样因子和 TGF- $\cdot \beta$ 诱导因子）高表达所引起的负反馈，后两者是Smad#/% 依赖的转录复合体的共抑制因子；肿瘤晚期，该通路的高活性状态使肿瘤细胞骨架、细胞黏附、基质重塑和迁移相关的基因表达发生改变，参与调节肿瘤的上皮细胞-间充质转化（EMT），在肿瘤浸润、转移中发挥着重要作用。TGF-βRII 能直接与游离配体 TGF-β$或TGF-β%结合，启动整个信号传导通路，结合配体的TGF-βRII连接并磷酸化I型受体TGF-βRI，进而磷酸化下游底物 Smad 蛋白。因此，TGF-βRII 在上述信号途径中起到至关重要的作用，是 TGF- $\beta$ 发挥调控作用的中心环节。

TGF- $\beta$ 能上调抗原递呈细胞（APC）上的PD-L$，而APC细胞可以释放可溶性的 PD-L$到肿瘤微环境中，通过 PD-L$ 与其受体的相互作用直接抑制细胞毒性 T 淋巴细胞，或者诱导细胞毒性 T 淋巴细胞释放抑制性可溶性分子。因此，在抑制 PD-$/PD-L$ 通路的基础上，靶向中和肿瘤微环境中的TGF- $\beta$ 可以使 T细胞恢复活性，增强免疫应答，更有效地提高抑制肿瘤发生和发展的效果，成为肿瘤治疗的一个新方案。

双功能融合蛋白是指有两个功能结构域的融合蛋白，由于其特异性和双功能性，现已成为生物工程治疗药物领域的研究热点，在肿瘤免疫治疗及自身免疫病等领域中具有广阔的应用前景。

EMD Serono Inc 和 Merck KGaA 合作研发的 MSB11$$%&2C(M--.#,)为目前全球首个正在临床试验阶段的 PD-L$ 和 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 双功能融合蛋白。M--.#, 用于治疗晚期实体瘤的 I 期研究正在进行中，近期报道在非小细胞肺癌、HPV 相关肿瘤队列显示出了初步疗效，继而开展了针对的多项在研的 III 期临床研究。多家跨国制药公司在研发针对 PD-L$ 的人源化的单克隆抗体，它通过阻断PD-L$/PD-$ 之间结合，最大限度的提高患者自身对肿瘤的免疫系统反应，从而达到对肿瘤细胞进行杀伤的目的。截止目前，市场上一共有了五种PD-$/PD-L$ 抗体药物，包括已经上市的默沙东的 PD-$ 抗体 Keytruda、BMS 的 PD-$ 抗体Opdivo、罗氏的 PD-L$ 抗体 Tecentriq、辉瑞/默克的 PD-L$ 抗体 Bavencio 和阿斯利康的 PD-L$抗体 Durvalumab。PD-$/PD-L$ 通路作为肿瘤免疫治疗的一个新靶点，抗 PD-L$ 抗体和其他抗肿瘤药物的联合治疗也可能为未来的抗肿瘤治疗带来前所未有的新前景。目前尚未有TGF- $\beta$ 抑制剂上市，近年来陆续有 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 抑制剂进入动物实验及I/II 期临床研究阶段，包括:抗 TGF- $\beta$ 的单克隆抗体技术(如赛诺菲公司的 Fresolimumab 等)，抑制 TGF- $\beta$ 信号通路的小分子激酶抑制剂(如礼来公司的Galunisertib 等)，反义RNA 或者RNA 干扰策略和肿瘤疫苗等。探索的治疗策略有 TGF- $\cdot \beta$ 抑制剂单药，联合抗 PD-$/L$ 药物，以及联合化疗、放

疗、肿瘤疫苗及其他药物。

TQB#.&. 注射液是正大天晴药业集团股份有限公司开发的抗 PD-L$/TGF-βRII 双功能融合蛋白，其分子形式包括抗 PD-L$ 单抗和 TGF-βRII 胞外结构域截短形式两个部分，具体是由抗 PD-L$ 抗体重链的末端氨基酸连接 TGF-βRII 胞外区而成。TQB#.&. 可以阻断PD-$/PD-L$ 通路，并中和肿瘤微环境中的 TGF- $\cdot \beta$ 。PD-$ 和 TGF- $\beta$ 负信号的共同抑制可带来比任一单独途径抑制更加有效的抗肿瘤免疫应答，从而达到增加抗肿瘤疗效的目的。

筛选患者时应对患者既往史、现病史、用药史问询，排除： $\textcircled { 1 } \mathrm { H I V }$ 感染；或其他获得性、先天性免疫缺陷疾病； $\textcircled{2}$ 需使用免疫抑制剂、或全身、或可吸收的局部激素治疗以达到免疫抑制目的，并在首次给药前 - 天内仍需继续使用的患者（糖皮质激素每日剂量 $\mathrm { < 1 0 ~ m g }$ 泼尼松或其他等疗效激素除外）； $\textcircled{3}$ 无法控制的全身性细菌性、真菌性或病毒活动性感染。

临床试验期间监测血常规，并关注临床症状（如发热、咳嗽）。

根据检查结果的异常程度，必要时考虑暂停试验药物，根据临床症状和体征，采取对症支持治疗，如发生感染，采取抗感染治疗，并密切监控。

# Ø 输液反应

在试验过程中需要注意观察受试者的输液/过敏反应，尤其是急性免疫介导的不良反应（包括细胞因子风暴），并对医护工作人员进行相关的过敏反应急救培训；研究治疗期间如发生相关输液反应，请根据方案中给出的相应治疗指导原给予相应的处理（《由免疫检查点抑制剂治疗导致的免疫相关不良事件处理建议》及《中国临床肿瘤学会（CSCO）免疫检查点抑制剂相关的毒性管理指南#1$2》）。

# Ø 皮肤毒性事件

研究开始前告知受试者皮肤不良反应，让受试者对风险有所了解并且一旦出现及时与研究者联系；研究治疗期间定期监测相关检查，如受试者发生了皮肤相关不良事件，请根据方案中给出的相应治疗指导原则给予相应的处理（《由免疫检查点抑制剂治疗导致的免疫相关不良事件处理建议》及《中国临床肿瘤学会（CSCO）免疫检查点抑制剂相关的毒性管理指南 #1$2》）。

# Ø 呼吸系统异常（如：肺炎）

筛选患者时应对患者既往史、现病史、手术史进行详细询问，筛选时排除有特发性肺纤维化、组织性肺炎病史（例如，闭塞性细支气管炎）、药物诱导性肺炎病史、特发性肺炎病史，或筛选期胸部CT 扫描发现活动性肺炎证据。

临床试验中定期对患者肺部进行相关检查（必要时影像学检查），及时发现肺部病变。临床试验中如果出现呼吸困难，应仔细分析原因，评估症状严重程度，酌情进行处理。

临床试验中如果出现肺炎、肺水肿、肺出血等情况，应及时暂停用药进行相关医学处理。根据实际情况，考虑调整剂量使用或永久停药。

# Ø 肌肉骨骼异常

筛选时排除存在肿瘤骨转移所导致的严重骨损伤；或最近+ 个月内发生的或预计近期很可能发生的重要部位病理性骨折等。

建议患者适当体育锻炼，戒烟酒，补充钙剂及维生素 D。

试验期间如出现骨损伤，应立即给予医学处理和相应治疗；可以适当使用止痛药物控制患者的疼痛症状，对于剧烈疼痛的患者排除其他原因后可考虑暂停用药观察。

# Ø 胃肠道反应

临床试验期间观察胃肠道症状，如排便频率和大便性状的变化（如稀便、水样便等）。对于 $-# 级的消化道相关 AE 予以支持性治疗措施（如止吐药物、止泻药物等），对于 %级及以上的应考虑暂停药物，待恢复正常或基线水平后再给药；若多次出现，应考虑调整剂量使用，必要时永久停药。

# Ø 凝血功能异常

患者筛选时进行凝血功能和血小板计数检查，排除： $\textcircled{1}$ 凝血功能异常：部分活化凝血酶原时间（aPTT）或凝血酶时间（TT） ${ > } 1 . 5 \times \mathrm { U L N }$ ，或国际标准化比（INR） ${ > } 1 . 5$ （需要抗凝治疗的受试者 ${ > } 2 . 5 $ ）； $\textcircled{2} 6$ 个月内发生过动/静脉血栓事件，如脑血管意外(包括暂时性缺血性发作、脑出血、脑梗塞)、深静脉血栓及肺栓塞等；

临床试验中，定期监测凝血功能指标，并关注是否有皮肤出血瘀斑等出血症状；若相关指标异常达到 # 级及以上 AE 标准，综合评估，必要时应考虑暂停药物，予以支持性治疗措施，待恢复至正常或基线水平后再给药；若凝血功能异常反复出现，应考虑调整剂量使用，必要时永久停药。

# Ø 肝功能异常

患者筛选时进行肝功能检查，排除肝功能异常： $\textcircled{1}$ 无肝转移时，ALT、AST 或 $\mathrm { A L P } { > } 2 . 5$ $\times$ 正常值参考范围上限（ULN）；肝转移时，ALT、AST 或 $_ { \mathrm { A L P > 5 } \times \mathrm { U L N } }$ ； $\textcircled{2}$ 血清总胆红素 ${ \mathrm { > } } 1 . 5 { \times } \mathrm { U L N }$ （Gilbert 综合症患者 ${ \bf \ddot { \tau } } { > } 3 \times \bf { U L N } ,$ ）；

临床试验中应定期对患者进行肝功能检查，若相关指标异常达到#级及以上AE标准，综合评估，必要时应考虑暂停药物，予以支持性治疗措施（给予保肝药物），待恢复至正

常或基线水平后再给药；若肝功能异常反复出现，应考虑调整剂量使用，必要时永久停药。

# Ø 血糖升高及甲状腺功能异常

筛选患者时询问患者既往史、现病史，排除已知患有严重的垂体/肾上腺/甲状腺功能障碍，及药物控制不佳的糖尿病。

临床试验期间定期检测血糖、甲状腺功能；对相关指标异常达到 # 级及以上 AE 标准，综合评估，必要时应考虑暂停药物，予以治疗（如使用或调整降糖药物），待恢复至正常或基线水平后再给药；若异常反复出现，应考虑调整剂量使用，必要时停用受试药物。

# Ø 电解质异常

筛选患者时对血电解质检测，记录基线期数据。

临床试验中，定期监测电解质水平。对于出现电解质异常的患者应积极予以临床措施进行纠正（如补充电解质）。根据检查结果的异常程度，必要时调整剂量使用或停用受试药物。

# Ø 外周血细胞改变

患者筛选时进行血常规检查，排除骨髓造血功能不足者（$, 天内未输血状态下）：绝对中性粒细胞计数（ANC） ${ < 1 . 5 \times 1 0 ^ { 9 } \mathrm { / L } }$ ，血小板 ${ < } 1 0 0 \times 1 0 ^ { 9 } / \mathrm { L }$ ，血红蛋白 ${ < } 9 0 ~ \mathrm { g / L }$ ；

临床试验中应定期对患者进行血常规检查，若相关指标异常达到 # 级 AE 标准，按需考虑给予支持性治疗措施（如给予集落刺激因子、EPO 等）；若相关指标异常达到 % 级 AE 标准，给予支持性治疗，根据医学评估可考虑暂停受试药物，待恢复至 $ 级或基线水平后再给药；调整剂量使用，必要时永久停药。

# A.% 首次人体试验设计

# A.%." 研究人群

《抗肿瘤药物临床试验技术指导原则》指出：“在临床上已经具备公认有效的标准治疗方法的情况下，肿瘤患者应当采用标准治疗方法作为一线治疗，标准治疗失败或复发的时候，患者才能参加试验药物的临床试验。因此，出于伦理的要求，通常新的抗肿瘤药物首先在对标准治疗无效或失败的患者中进行，在获得对三线或二线治疗的肯定疗效后，再逐步向一线治疗推进。对某些药物，根据其作用机制，预期与一线标准治疗联合可能获得协同效果，可能进行与一线标准治疗联合方案的临床试验，此时可选择初治患者进行。”

综上所述，首次人体试验拟选择晚期恶性实体/血液肿瘤患者。

# Ø 入选标准

# 满足以下所有入选标准的受试者才能入组本试验：

1) 经组织和/或细胞学确诊，标准治疗失败或缺乏有效治疗方法的晚期恶性实体/血液肿瘤患；

2) 年龄：18~75 周岁；  
3) ECOG 评分：0~1 分；  
4) 预计生存期超过 3 个月；  
5) 主要器官功能正常，即符合下列标准：

a) 血常规检查（筛选前 14 天内未输血且未接受生长因子支持）：血红蛋白（ $\mathbf { \mathrm { . H G B } } ) \geq 8 0$ g/L；中性粒细胞计数（NEUT） $\ge 1 . 5 { \times } 1 0 ^ { 9 } / \mathrm { L }$ ；血小板计数（PLT） ${ \geq } 9 0 { \times } 1 0 ^ { 9 } / \mathrm { L }$ ；  
b) 生化检查：谷丙转氨酶（ALT）及谷草转氨酶（AST） $\leq 2 . 5 { \times } \mathrm { U L N }$ （原发肝胆肿瘤或肿瘤肝脏转移者：ALT、 ${ \sf A S T } { \leq } 5 { \times } { \sf U L N } )$ ）；总胆红素（TBIL） ${ \underline { { < 1 . 5 } } } { \times } \mathrm { U L N }$ （Gilbert 综合症患者： $\mathrm { T B I L } { \le } 3 { \times } \mathrm { U L N } )$ ）；肌酐（CRE） $\leq 1 . 5 { \times } \mathrm { U L N }$ 或肌酐清除率 $\geq 6 0 ~ \mathrm { m l / m i n }$ ；  
c) 凝血功能：活化部分凝血酶时间（APTT）、国际标准化比值（INR）、凝血酶原时间（PT） $\leq 1 . 5 { \times } \mathrm { U L N }$ ；  
d) 左心室射血分数（LVEF） $\ge 5 0 \%$ ；

6) 育龄期女性患者在研究入组前的 7 天内血清或尿 HCG 检查必须为阴性，且必须为非哺乳期；患者应同意在研究期间和研究期结束后6个月内必须采用避孕措施。  
7) 患者自愿加入本研究，签署知情同意书，依从性好。

# Ø 排除标准

# 满足以下任意一条排除标准的受试者将不能入组本研究：

1) 合并疾病及病史：

a) 2 年内出现过或当前同时患有其它恶性肿瘤。以下两种情况可以入组：经单一手术治疗的其他恶性肿瘤，达到 R0 切除且未见复发转移；治愈的子宫颈原位癌、非黑色素瘤的皮肤癌、鼻咽癌和表浅的膀胱肿瘤 [Ta (非浸润性肿瘤)，Tis (原位癌) 和T1 (肿瘤浸润基膜)]；  
b) 由于任何既往治疗引起的高于 CTC AE 1 级以上的未缓解的毒性反应，不包括脱发、外周感觉神经障碍；  
c) 研究治疗开始前 28 天内接受了重大外科治疗或明显创伤性损伤（不包括穿刺活检）；  
d) 长期未治愈的伤口或骨折；  
e) 6 个月内发生过动/静脉血栓事件，如脑血管意外(包括暂时性缺血性发作、脑出血、脑梗塞)、深静脉血栓及肺栓塞等；  
f) 具有精神类药物滥用史且无法戒除或有精神障碍者；

g) 存在任何重度和/或未能控制的疾病的受试者，包括：

1. 血压控制不理想（收缩压 $\geq 1 5 0 \mathrm { m m H g }$ 或舒张压 $\geq 1 0 0 \ \mathrm { m m H g }$ ）；  
2. 患有 ${ \ge } 2$ 级心肌缺血或心肌梗塞、心律失常（包括男 QTc $\ge 4 5 0 \mathrm { m s }$ (男），QTc${ \geq } 4 7 0 \mathrm { m s }$ (女））及 ${ \geq } 2$ 级充血性心功能衰竭（纽约心脏病协会（NYHA）分级）；  
3. 活动性或未能控制的严重感染(≥CTC AE 2 级感染)；  
4. 肝硬化、活动性肝炎*；

*活动性肝炎（乙肝参考：HBsAg 阳性，且 HBV DNA 检测值超过正常值上限；丙肝参考：HCV 抗体阳性，且 HCV 病毒滴度检测值超过正常值上限）；注：符合入组条件的，乙肝表面抗原阳性或核心抗体阳性的受试者、丙型肝炎患者，需持续抗病毒治疗，以防止病毒激活。

5. 肾功能衰竭需要血液透析或腹膜透析者；  
6. 有免疫缺陷病史，包括 HIV 阳性或患有其它获得性、先天性免疫缺陷疾病，或有器官移植史者；  
7. 糖尿病控制不佳（空腹血糖（FBG）＞10mmol/L）；  
8. 尿常规提示尿蛋白 $\geq + +$ ，且证实 24 小时尿蛋白定量 ${ \displaystyle \sum 1 . 0 ~ } \mathrm { g }$ 者；  
9. 患有癫痫并需要治疗者；

2) 肿瘤相关症状及治疗：

a) 研究治疗开始前 4 周内曾接受过手术、化疗、放疗或其它抗癌疗法（从末次治疗结束时间开始计算洗脱期）；既往曾接受过局部放疗的，如果满足以下条件可以入组：放疗结束距研究治疗开始超过4周（脑部放疗为超过2周）；且本次研究选择的靶病灶不在放疗区域内；或靶病灶位于放疗区域内，但已确认进展。  
b) 研究治疗开始前 2 周内接受过 NMPA 批准药物说明书中明确具有抗肿瘤适应症的中成药（包括复方斑蝥胶囊、康艾注射液、康莱特胶囊/注射剂、艾迪注射液、鸦胆子油注射剂/胶囊、消癌平片/注射剂、华蟾素胶囊等）治疗；  
c) 既往接受过针对 TQB2858 注射液同作用靶点的免疫双抗治疗药物；  
d) 未能控制的，仍需反复引流的胸腔积液、心包积液或腹水（研究者判断）；  
e) 停用脱水剂及类固醇药物后症状控制稳定时间小于4周的脑转移者；

3) 研究治疗相关：

a) 研究治疗开始前 28 天内减毒活疫苗接种史或者研究期间计划行减毒活疫苗接种；  
b) 既往有对大分子药物严重过敏史，或对 TQB2858 注射液已知成分过敏；  
c) 研究治疗开始前 2 年内发生过需要全身性治疗（例如使用缓解疾病药物、皮质类固醇或免疫抑制剂）的活动性自身性免疫疾病。替代疗法（例如甲状腺素、胰岛

素或者用于肾上腺或垂体机能不全的生理性皮质类固醇等）不视为全身性治疗；

d) 诊断为免疫缺陷或正在接受全身性糖皮质激素治疗或任何其他形式的免疫抑制疗法。(剂量 $\mathrm { > } 1 0 \mathrm { m g / }$ 天泼尼松或其他等疗效激素)，并首次给药 2 周内仍在继续使用的；

 3 级疲劳、乏力、恶心或其他可控制的体质症状不判定为 DLT；
