# 3 物料控制

# 3.1 细胞基质的来源、历史和构建

# 3.1.1 概述

通过分子克隆手段将编码抗 PD-L1-TGFβRⅡ 基因克隆到真核表达载体，采用电穿孔技术导入CHO-S细胞系，经过有限稀释法结合单克隆成像技术进行单克隆筛选，获得高效、稳定表达抗 PD-L1-TGFβRⅡ 融合蛋白的重组细胞株 P311-14G11，该细胞株的细胞状态良好，细胞饱满而充实，增殖正常，细胞倍增周期为24小时左右，生长情况和目标蛋白表达水平都非常出色，被最终定为 TQB2858 的生产用工程细胞株。

# 3.1.2 表达载体的构建

轻链序列来源于正大天晴药业集团股份有限公司 TQB2450 项目（抗 PD-L1 单克隆抗体），委托金斯瑞生物科技股份有限公司进行轻链基因的合成，合成时分别在基因的两 端 添加酶 切 位 点 HindIII、ApaI， 并 将合 成的 基因 片段克 隆至pCDNA3.1(+)质粒中，获得质粒 pCDNA3.1(+)-0412-2 LC。

# 3.1.3.1 宿主细胞

# 5.1 细胞培养及纯化工艺研究
