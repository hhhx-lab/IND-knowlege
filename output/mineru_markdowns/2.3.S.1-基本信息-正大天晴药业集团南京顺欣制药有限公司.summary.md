# 2.3.S.1-基本信息-正大天晴药业集团南京顺欣制药有限公司.md 摘要

TQB2858注射液是一种双特异性融合蛋白药物，通用名称为TQB2858注射液（英文：TQB2858 Solution for Injection），曾用名TQB2814注射液。其分子式为C₇₈₆₂H₁₂₁₅₀N₂₀₉₀O₂₄₈₄S₇₈（无翻译后修饰），分子量178162 Da，由1634个氨基酸组成，包括两条轻链（214 aa）和两条重链（603 aa）。结构上，重链1-446位及轻链为PD-L1抗体区域，447-466位为连接子（Linker），467-603位为TGFβRII胞外区。重链N末端环化，非糖基化状态下理论含有4个N-糖基化位点（N296、N514、N538、N598），其中N296、N514和N538经鉴定确证糖基化，N598未发生糖基化，但文献显示其对TGF-β结合活性无影响，风险可控。理论上包含28对二硫键（14种形式），经实验确证8对（包括2对轻链内、4对重链内、1对轻重链间、2对重链间），其余6对位于TGFβRII区，连接复杂，方法开发中。氨基酸序列详见文档，轻链序列从DIQMTQSPSS...至NRGEC，重链从QITLKESGPT...至NPD。

TQB2858通过重组DNA技术在哺乳动物细胞中表达，经病毒灭活纯化，呈澄明液体（可带轻微乳光），对光和温度敏感，需低温避光储存。其基本性质为同时靶向PD-L1和TGF
