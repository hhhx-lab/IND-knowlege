# 1.7.1 临床试验期间增加新适应症

本品为未在国内外上市销售的生物制品，属于治疗用生物制品注册分类 1类。

TQB2450 注射液（ $1 0 \mathrm { m L } { : } 1 0 0 \mathrm { m g }$ ）于 2017 年 01 月申请临床试验研究，临床申请受理号为：CXSL1700018，已于 2017 年 10 月获得临床批件，临床批件号：2017L04914。此后，TQB2450 注射液于 2019 年 04 月提出新增规格（20 mL： $6 0 0 \mathrm { m g }$ ）的补充申请，补充申请受理号为：CXSB1900008，该申请已于2019年12月获得临床批件，临床批件号：2019B04525。

此次 IND 拟申请 TQB2450 注射液联合盐酸安罗替尼胶囊、化疗（注射用卡铂、依托泊苷注射液）适用于初治的广泛期小细胞肺癌的治疗适应症。
