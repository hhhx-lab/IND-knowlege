# 2.3.P.8 稳定性

# 1 稳定性总结和结论

本 研 究 对 TQB2858 两 批 中 试 生 产 规 模 的 注 射 液 （ 批 号 ：

BP20200502、BP20200503）进行了加速和长期稳定性考察，且计划选取一批中试

生产规模的注射液（批号：BP20200502）进行影响因素试验和稀释稳定性试验。目

前为止，已开展本品的加速和长期稳定性研究，已获得加速和长期稳定性研究 0

月及加速稳定性研究1月数据。

注射液加速稳定性试验的试验温度定为 $2 5 ^ { \circ } \mathsf { C } { \pm } 2 ^ { \circ } \mathsf { C } .$ 、相对湿度为 $6 0 \% \pm 1 0 \%$ ，

考察6个月，分别在试验期间第0、1、2、3、6个月末取样；长期稳定性试验的试验

温度定为 $2 { \sim } 8 ^ { \circ } \mathrm { C }$ ，考察 30 个月，分别在试验期间第 0、3、6、9、12、18、24、30 个月

取样。检验项目包括 $\mathrm { p H }$ 值、蛋白质含量、纯度（分子排阻色谱法、CE-SDS 还原电

泳）、生物学活性等。

加速稳定性研究结果表明，在温度 $2 5 ^ { \circ } \mathrm { C } { \pm } 2 ^ { \circ } \mathrm { C } ,$ 、相对湿度 $6 0 \% \pm 1 0 \%$ （加速稳定

性试验）条件下，TQB2858 两批注射液（批号：BP20200502、BP20200503）放置 1

个月，高分子杂质含量增长约 $0 . 4 \% .$ 、降解物含量增长约 $1 . 7 \% { \sim } 1 . 8 \%$ ，其余检测指

标均无明显变化。
