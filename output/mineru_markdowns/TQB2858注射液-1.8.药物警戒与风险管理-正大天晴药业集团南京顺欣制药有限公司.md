# ".$.&."." 试验方案中的风险管理与控制

排除有“潜在安全性风险”的患者入组，包括：未能控制的严重感染、免疫系统疾病、脑部疾病的患者，经药物治疗无法控制的高血压患者，肝功能损伤以及明显骨髓抑制的患者。  
• 根据 TQB#%H% 的潜在安全性风险，制定剂量限制性毒性（DLT）标准。  
• DLT观察期内安全性审核：每个剂量组当受试者完成第$ 治疗周期给药后，将评估该剂量水平下试验药物的安全性、耐受性和药代动力学特性，以决定是否进入下一剂量组。  
• 制定科学、合理、操作性强的安全性监测方案，包括，定期进行生命体征观测，以及血尿常规、便常规、凝血功能、甲状腺功能、心电图和超声心动图检测等。  
• 针对 DLT 评价窗以外的可能发生的安全性风险，拟定“剂量调整、停止用药及伴随用药”的原则。

# ".$.&.".% 拟定可预期性不良反应的处理措施

借鉴同类产品的安全性概况及本品非临床安全性结果，制定了临床试验过程中的预防处理措施。具体如下：

# Ø 免疫系统异常

筛选患者时应对患者既往史、现病史、用药史问询，排除： $\textcircled{1}$ HIV 感染；或其他获得性、先天性免疫缺陷疾病； $\textcircled{2}$ 需使用免疫抑制剂、或全身、或可吸收的局部激素治疗以达到免疫抑制目的，并在首次给药前 I 天内仍需继续使用的患者（糖皮质激素每日剂量 ${ < } 1 0$ mg 泼尼松或其他等疗效激素除外）； $\textcircled{3}$ 无法控制的全身性细菌性、真菌性或病毒活动性感染。

临床试验期间监测血常规，并关注临床症状（如发热、咳嗽）。

根据检查结果的异常程度，必要时考虑暂停试验药物，根据临床症状和体征，采取对症支持治疗，如发生感染，采取抗感染治疗，并密切监控。

# Ø 输液反应

在试验过程中需要注意观察受试者的输液/过敏反应，尤其是急性免疫介导的不良反应（包括细胞因子风暴），并对医护工作人员进行相关的过敏反应急救培训；研究治疗期间如发生相关输液反应，请根据方案中给出的相应治疗指导原给予相应的处理（《由免疫检查点抑制剂治疗导致的免疫相关不良事件处理建议》及《中国临床肿瘤学会（CSCO）免疫检查点抑制剂相关的毒性管理指南 #K$+》）。

# Ø 皮肤毒性事件

研究开始前告知受试者皮肤不良反应，让受试者对风险有所了解并且一旦出现及时与研究者联系；研究治疗期间定期监测相关检查，如受试者发生了皮肤相关不良事件，请根据方案中给出的相应治疗指导原则给予相应的处理（《由免疫检查点抑制剂治疗导致的免疫相关不良事件处理建议》及《中国临床肿瘤学会（CSCO）免疫检查点抑制剂相关的毒性管理指南#K$+》）。

# Ø 呼吸系统异常（如：肺炎）

筛选患者时应对患者既往史、现病史、手术史进行详细询问，筛选时排除有特发性肺纤维化、组织性肺炎病史（例如，闭塞性细支气管炎）、药物诱导性肺炎病史、特发性肺炎病史，或筛选期胸部CT 扫描发现活动性肺炎证据。

临床试验中定期对患者肺部进行相关检查（必要时影像学检查），及时发现肺部病变。临床试验中如果出现呼吸困难，应仔细分析原因，评估症状严重程度，酌情进行处理。

临床试验中如果出现肺炎、肺水肿、肺出血等情况，应及时暂停用药进行相关医学处理。根据实际情况，考虑调整剂量使用或永久停药。

# Ø 肌肉骨骼异常

筛选时排除存在肿瘤骨转移所导致的严重骨损伤；或最近* 个月内发生的或预计近期很可能发生的重要部位病理性骨折等。

建议患者适当体育锻炼，戒烟酒，补充钙剂及维生素 D。

试验期间如出现骨损伤，应立即给予医学处理和相应治疗；可以适当使用止痛药物控制患者的疼痛症状，对于剧烈疼痛的患者排除其他原因后可考虑暂停用药观察。

# Ø 胃肠道反应

临床试验期间观察胃肠道症状，如排便频率和大便性状的变化（如稀便、水样便等）。

对于$-# 级的消化道相关AE 予以支持性治疗措施（如止吐药物、止泻药物等），对于& 级及以上的应考虑暂停药物，待恢复正常或基线水平后再给药；若多次出现，应考虑调整剂量使用，必要时永久停药。

# Ø 凝血功能异常

患者筛选时进行凝血功能和血小板计数检查，排除： $\textcircled{1}$ 凝血功能异常：部分活化凝血酶原时间（aPTT）或凝血酶时间（TT） ${ > } 1 . 5 { \times } \mathrm { U L N }$ ，或国际标准化比（INR） ${ > } 1 . 5$ （需要抗凝治疗的受试者 ${ > } 2 . 5 { \mathrm { ~ } } ,$ ）； $\textcircled{2} 6$ 个月内发生过动/静脉血栓事件，如脑血管意外(包括暂时性缺血性发作、脑出血、脑梗塞)、深静脉血栓及肺栓塞等；

临床试验中，定期监测凝血功能指标，并关注是否有皮肤出血瘀斑等出血症状；若相关指标异常达到 # 级及以上 AE 标准，综合评估，必要时应考虑暂停药物，予以支持性治疗措施，待恢复至正常或基线水平后再给药；若凝血功能异常反复出现，应考虑调整剂量使用，必要时永久停药。

# Ø 肝功能异常

患者筛选时进行肝功能检查，排除肝功能异常： $\textcircled{1}$ 无肝转移时，ALT、AST或 $\mathrm { A L P } { > } 2 . 5$ $\times$ 正常值参考范围上限（ULN）；肝转移时，ALT、AST 或 $_ { \mathrm { A L P > 5 } \times \mathrm { U L N } }$ ； $\textcircled{2}$ 血清总胆红素 ${ \mathrm { \ddot { \gamma } l . 5 } } \times \mathrm { U L N }$ （Gilbert 综合症患者 ${ \mathrm { 5 3 } } \times { }$ ULN）；

临床试验中应定期对患者进行肝功能检查，若相关指标异常达到#级及以上AE标准，综合评估，必要时应考虑暂停药物，予以支持性治疗措施（给予保肝药物），待恢复至正常或基线水平后再给药；若肝功能异常反复出现，应考虑调整剂量使用，必要时永久停药。

# Ø 血糖升高及甲状腺功能异常

筛选患者时询问患者既往史、现病史，排除已知患有严重的垂体/肾上腺/甲状腺功能障碍，及药物控制不佳的糖尿病。

临床试验期间定期检测血糖、甲状腺功能；对相关指标异常达到 # 级及以上 AE 标准，综合评估，必要时应考虑暂停药物，予以治疗（如使用或调整降糖药物），待恢复至正常或基线水平后再给药；若异常反复出现，应考虑调整剂量使用，必要时停用受试药物。

# Ø 电解质异常

筛选患者时对血电解质检测，记录基线期数据。

临床试验中，定期监测电解质水平。对于出现电解质异常的患者应积极予以临床措施进行纠正（如补充电解质）。根据检查结果的异常程度，必要时调整剂量使用或停用受试药物。

# Ø 外周血细胞改变

患者筛选时进行血常规检查，排除骨髓造血功能不足者（$, 天内未输血状态下）：绝对中性粒细胞计数（ANC） ${ < } 1 . 5 \times 1 0 ^ { 9 } / \mathrm { L }$ ，血小板 ${ < } 1 0 0 { \times } 1 0 ^ { 9 } / \mathrm { L }$ ，血红蛋白 ${ < } 9 0 ~ \mathrm { g / L }$ ；

# ".$.&.".& 建立服务药物临床试验阶段的药物警戒系统

药物警戒是贯穿于药品全生命周期的研究与活动，是始于药物进入人体、涵盖新药研发的全部临床试验阶段、直至药品上市后在更大规模人群中使用，持续性的对药物的安全性进行监测和风险获益评估，以保障患者的用药安全。因此，正大天晴药业集团股份有限公司在开展 I 期临床试验之前，已经制定了服务药物临床试验阶段的药物警戒系统的搭建和工作规划，在临床试验过程中将药物警戒贯穿始终，收集和记录临床试验中发生的不良事件，并按照国家法规要求和法定流程及时记录和快速上报严重不良事件。

# !.#.$.!.$.! 关键术语及定义

# 不良事件

不良事件（Adverse Event, AE）是指任何发生在病人或药物临床试验受试者中的不利事件。它不一定与药物治疗有因果关系。不良事件可以是一种不良的未预期的征象（包括异常的实验室检查等）、症状或与药品使用中有时间相关性的疾病，不考虑是否与药品有因果关系。

# 药物不良反应

药物不良反应（Adverse Drug Reaction，ADR）是指新药在获准批准前的临床试验中或新适应症批准之前，尤其治疗剂量未建立之前，任何有害和未预期的药品反应（任何剂量）都认为是药物不良反应。“药物反应”一词指的是其与不良事件存在至少可能相关。

# 非预期的药物不良反应

非预期的药物不良反应（Unexpected Adverse Drug Reaction）是指不良反应的性质、严重程度、后果或频率，不同于先前临床试验方案或其他相关资料（如研究者手册等文件）所描述的预期风险。研究者手册作为主要文件提供用以判断某不良反应是否预期或非预期

的安全性参考信息。如： $\textcircled{1}$ 急性肾衰在研究者手册中列为可能的不良反应，但试验过程中出现间质性肾炎，即应判断为非预期不良反应； $\textcircled{2}$ 肝炎在研究者手册中列为可能的不良反应，但试验过程中发生爆发性肝炎，即应判断为非预期不良反应。

# Ø 严重不良事件或反应

严重不良事件或反应（Serious Adverse Event or Adverse Drug Reaction，SAE or SADR）是指以下不利/不良的医学事件（在任何剂量下）：

（$）导致死亡；  
（#）危及生命；（注意：严重中的“危及生命”的定义是指严重病人即刻存在死亡的风险，并非是指假设将来发展严重时可能出现死亡。）  
（&）需要住院或现有住院时间延长；  
（,）导致永久或显著残疾/功能丧失；  
（H）导致先天性畸形或出生缺陷；  
（*）其他重要的医学事件。

重要医学事件：在某些情况下，需要基于医学和科学判断来决定是否适用快速报告，例如某些重要的医学事件虽然不会立即危及生命、导致死亡或住院，但基于医学判断，可能危害患者或受试者，或需要治疗干预以防止出现上述情况或结果，这些事件通常被认为是严重不良事件。例如，在急诊室的重要治疗或在家发生的过敏性支气管痉挛，未住院的恶液质或惊厥、产生药物依赖成瘾。

# !.#.$.!.$.& 临床试验中安全性信息的处理

# Ø 一般不良事件（AE）收集、记录

临床监查员或项目经理协助、督促、监查研究者对临床试验中发生的一般不良事件进行收集和记录。记录的内容包括：AE名称、起止时间、严重程度、严重性、处理、转归、因果关系、随访等。

# Ø 严重不良事件报告及医学评价

严重不良事件发生后，研究者应立即告知临床监查员或项目经理。临床监查员或项目经理还需协助研究者填写《严重不良事件报告表》并按照公司的规定（需考虑 SUSAR 上报CDE的时限设置合理的报告时限）报告至临床医学部。报告后获得的随访信息应及时提供。

临床医学部收到严重不良事件报告后，对严重不良事件进行接收确认、录入严重不良事件数据库、信息审查及医学评价。医学评价内容：预期性、严重性、严重程度、因果关系、病例医学描述。医学评价后的报告表提供给药物警戒组。

# Ø 可疑的非预期严重不良反应上报 CDE

临床医学部对严重不良事件评价后，符合可疑的非预期严重不良反应上报标准的，经内部审核审批后，按照要求上报至国家药品审评中心，上报要求如下：

$） 采用 MedDRA 编码，符合 ICH E#B（R&）数据要求。  
#）对于致死或危及生命的非预期严重不良反应，在首次获知后尽快报告，但不得超过I 天，并在随后的% 天内报告、完善随访信息。

# Ø 妊娠事件报告

根据ICH 及相关法规要求，除不良事件外，申办方尚需收集妊娠报告。因此需在研究方案中要求报告受试者或其配偶是否有妊娠情况发生。

临床监查员或项目经理协助、督促和监查研究者对此类情况的收集和报告。妊娠情况发生后，临床监查员或项目经理填写妊娠报告表并按照规定报告。妊娠报告还需要随访至妊娠结局（如：妊娠终止、分娩）。如在妊娠期间发生以下情况者，按照严重不良事件/不良事件进行管理：胎儿/新生儿先天异常或畸形（严重不良事件）；自发性流产（严重不良事件）；因医学原因终止妊娠（不良事件/严重不良事件）。

# Ø 其他需要快速报告的安全性信息

临床医学部还需收集其他潜在严重安全性风险的信息。一般情况下，对于明显影响药品风险获益评估的信息或可能考虑药品用法改变，或影响总体药品研发进程的信息，均属于此类情况，包括： $\textcircled{1}$ 对于已知的、严重的不良反应，其发生率增加，判断具有临床重要性； $\textcircled{2}$ 对暴露人群有明显的危害，如在治疗危及生命疾病时药品无效； $\textcircled{3}$ 在新近完成的动物试验中的重大安全性发现（如致癌性）。

临床医学部项目组在获知以上信息后，填写报告表并按照规定报告。这类报告在临床医学部做出医学评价和科学判断后，尽快按照规定上报至国家药品审评中心。
