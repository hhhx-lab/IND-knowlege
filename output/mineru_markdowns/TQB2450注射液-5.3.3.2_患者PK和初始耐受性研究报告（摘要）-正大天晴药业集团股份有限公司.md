# 剂量递增方案

本品起始剂量 $\mathrm { { 1 m g / k g } }$ ，后续依次给予3、10、20、30mg/kg。在每一个剂量等级，若均未出现 DLT，则进入下一剂量等级；若出现 1 例DLT，则该剂量等级追加入组 3例患者（男女兼有）；若这 3例中均未出现DLT，则进入下一剂量等级；若这 3例中出现 1例或1例以上 DLT，则剂量爬坡必须中止。如某一剂量组有2 例患者出现DLT，则终止试验。

# 剂量扩展剂量

固定剂量给药：1200mg/次。

# 给药方案

按规定的剂量，使用生理盐水 $2 5 0 \mathrm { m L }$ 稀释后静脉输注（每次输液使用校验过的输液泵），输注时间 $6 0 \ \mathrm { m i n }$ （±5min）。每三周静脉滴注一次，21 天为一个治疗周期。

# DLT 定义

受试者接受连续治疗 1个周期（21天）内出现与药物有关的以下分级的毒性反应（按 NCI CTC AE4.03 毒性评价标准）：

 3级或以上的非血液学毒性，除外以下毒性反应：

 3级恶心、呕吐、腹泻，在下一周期给药前不能恢复至 $\leqslant 1$ 级视为DLT；  
受试者在基线期存在 2 级 AST、AST 和/或碱性磷酸酶（AKP）升高的情况下，肝功能指标升高至 ${ > } 1 0 \times \mathrm { U L N }$ 时视为 DLT；

 4级血液学毒性。

# MTD 定义

MTD定义为小于 $3 3 \%$ 患者出现剂量限制性毒性（DLT）的前一个剂量。

# 剂量调整

允许延迟用药，但不进行剂量调整。

# 安全性分析

基于安全性数据集，对临床观察得到的安全性指标进行安全性和耐受性评估，安全性指标包括不良事件与严重不良事件、生命体征、实验室检查、12-导联心电图等。

# PK浓度分析

基于 PK 浓度分析数据集，按照计划采样时间点，分剂量组对受试者的血药浓度进行描述性统计。统计量包括：n、BQL的个数（n of BQL）、Mean、SD、Median、Min、Max、变异系数（%CV）、几何均值（GeoMean）、对数标准差（SDlog），并绘制平均血药浓度-时间曲线（线性和半对数）。

# PK参数分析

基于 PK 浓度分析数据集，根据实际采样时间，采用 Phoenix WinNonlin8.2版本或最新版本的软件的非房室模型进行 PK 参数计算，再进行 PK 参数的统计分析。

基于 PK 参数分析数据集，分剂量组对受试者的 PK 参数进行描述性统计汇总。统计量包括：n、BQL 的个数（n of BQL）、Mean、SD、Median、Q1、Q3、Min、Max、变异系数（%CV）、几何均值（GeoMean）、对数标准差（SDlog）、几何变异系数（ $\% C V _ { \mathrm { { b } } } )$ ）。

# 药效动力学分析

基于药效动力学分析集，根据计划采样时间，分剂量组描述受体占位和细胞因子。其中连续型变量描述性分析的统计量包括：n、Mean、SD、Median、Min、Max、变异系数（%CV）。分类变量采用频数、百分比和 $9 5 \%$ 可信区间进行描述，可信区间基于 Clopper-Pearson 方法计算。

# 安全性结果

# TEAE

40 例（ $1 0 0 . 0 0 \%$ ）受试者至少发生了 1 次 TEAE。

# TRAE

所有级别的 TRAE 发生率为 37例（ $9 2 . 5 0 \%$ ），3级以上不良反应发生率为$2 7 . 5 0 \%$ 。
