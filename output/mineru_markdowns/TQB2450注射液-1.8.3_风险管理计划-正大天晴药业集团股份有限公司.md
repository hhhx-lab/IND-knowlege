# 2）药物的药理作用及作用机制

安罗替尼是多靶点的酪氨酸激酶抑制剂，作用靶点为 VEGFR（VEGFR-1、VEGFR-2和 VEGFR-3）、PDGFR、FGFR 和 c-Kit 等。安罗替尼可以通过同时抑制血管新生和肿瘤生长达到抑制肿瘤进展的效果。

TQB2450 是一种靶向程序性死亡配体-1（PD-L1）的人源化单克隆抗体。程序性死亡分子-1及其配体1（PD-1 /PD-L1）是一对负性免疫共刺激分子，正常情况下，组织细胞表面的 PD-L1 与淋巴细胞表面的 PD-1 结合后，可抑制淋巴细胞功能，诱导活化的淋巴细胞凋亡，从而在自身免疫耐受及防止自身免疫性疾病中发挥重要作用。多种肿瘤细胞表面也表达PD-L1，肿瘤细胞表达的 PD-L1，可与肿瘤浸润淋巴细胞表面的 PD-1分子结合，抑制淋巴细胞的功能及细胞因子的释放，并诱导淋巴细胞凋亡，从而抵抗淋巴细胞的杀伤作用，最终导致肿瘤发生免疫逃逸。TQB2450 可阻止 PD-L1 与 T 细胞表面的 PD-1 和 B7.1 受体结合，使T细胞恢复活性，从而增强免疫应答，具有治疗多种类型肿瘤的潜力。

# 1）临床前毒理学研究

安罗替尼的一般毒理学研究：SD 大鼠连续 26 周经口给予安罗替尼 0.2、0.8 和 3.0$\mathrm { m g / k g }$ ，停药恢复 6 周，未见不良反应剂量（NOAEL）为 $0 . 8 \mathrm { m g / k g }$ ，按体表面积计算，约为临床用药剂量（ $1 2 ~ \mathrm { m g / }$ 人）的 0.65 倍；在 $3 . 0 ~ \mathrm { m g / k g }$ 出现明显毒性反应，毒性靶器官为牙齿和肾脏。Beagle 犬连续 39 周经口给予安罗替尼 0.02、0.08 和 $0 . 3 2 \mathrm { m g / k g }$ ，停药恢复6周， $\mathrm { N O A E L } { < } 0 . 0 2 \mathrm { m g } / \mathrm { k g }$ ，按体表面积计算，约为临床用药剂量（ $1 2 \mathrm { m g / }$ 人）的 0.05 倍，主要毒性反应为小动脉/微动脉动脉炎及其继发性改变。

# 1.8.3.1.3 风险管理与控制计划

为保障临床研究期间受试者的安全性权益，计划从以下方面对安全性风险进行管理和控制：

# 1）试验方案中的风险管理与控制

 排除有“潜在安全性风险”的患者入组，包括：明显胃肠道异常的患者、经药物治疗无法控制的高血压患者、肝肾功能损伤、以及明显骨髓抑制的患者。  
 根据抗 PD-L1 单抗和抗血管生成药物潜在的安全性风险，以及早期 TQB2450 注射液联合盐酸安罗替尼胶囊临床研究安全性数据，制定科学、合理、操作性强的安全性监测方案，包括但不限于定期进行生命体征观测，血尿常规、血生化、凝血功能、便常规、甲状腺功能、心肌酶谱、12-导联心电图和超声心动图等检查。

# 2）拟定可预期性不良反应的处理措施

#  给药方案的调整

# 总体原则：

按NCI CTCAE v5.0 标准进行毒性分级。考虑到本次研究为联合用药，能明确与不良事件无关的研究药物，可以维持用药方案不变。  
● 同时发生不同严重程度级别的重度毒性时，应根据观察到最高级别进行调整。  
当发生毒性反应时，研究者可参考以下规则对给药进行延迟或剂量调整。延迟或剂量调整的原因，采取的支持疗法和结果记录在受试者病历和 eCRF中。  
● 若经研究者判定给药调整与建议的调整规则不同时，与申办方协商沟通后，由研究者参照临床实践指南或既往临床经验进行相应调整，相关原因记录在受试者病历中。  
如果因一种药物引起的毒性反应需永久终止用药时，可继续使用另一种药物治疗直至疾病进展或者出现不可耐受的毒性反应（以先出现者为准）。

# 暂停/延迟给药和剂量调整：

● TQB2450 注射液延迟给药规定：

TQB2450 注射液为固定剂量，因 TQB2450 相关的不良事件可延迟给药但不能剂量调整，延迟给药最长时间 12周（自上一次给药时间计算）。若毒性反应在 12周以内恢复至基线水平时，可以继续接受 TQB2450 治疗；若毒性反应超过 12 周才恢复至基线水平，则根据研究者判断其获益风险比，与申办方讨论后决定是否恢复 TQB2450 治疗。如果因为TQB2450 的免疫相关毒性反应需要永久终止用药时，可继续接受安罗替尼治疗直至疾病进展或者出现不可耐受的毒性反应（以先出现者为准）。TQB2450注射液所致的不良事件的处理建议参见《中国临床肿瘤学会（CSCO）免疫检查点抑制剂相关的毒性管理指南（2021）》。本次研究期间，TQB2450 注射液允许延迟用药，但不进行剂量调整。

# 盐酸安罗替尼胶囊暂停给药和剂量调整规定：

每周期盐酸安罗替尼胶囊在服药期内（1~14 天），发生与安罗替尼有关的不良反应，需要暂停给药时，暂停时间最长不能超过 5 天。超过 5 天仍无法继续用药者，则本周期剩余安罗替尼不再使用。在下一个周期开始用药时，如因不良反应延迟用药，延迟用药时间最长不得超过4周，超过 4周仍然无法服用（包括降低剂量）安罗替尼的受试者，需要永久停用安罗替尼（非安全性原因导致的事件可以除外）。

如果需要，盐酸安罗替尼胶囊可以进行剂量调整。任何受试者如果需减少剂量，将在其后的周期继续接受减量的剂量治疗。盐酸安罗替尼胶囊的起始剂量是 $1 2 \mathrm { m g }$ 时，可下调剂量至 $1 0 \mathrm { m g }$ 或 8 mg（ $1 2 \mathrm { m g } {  } 1 0 \mathrm { m g } {  } 8 \mathrm { m g }$ 依次下调，不允许跨剂量调整），若 $8 \mathrm { m g }$ 剂量水平下无法耐受的受试者需终止研究。对于盐酸安罗替尼胶囊经下调剂量至 $1 0 ~ \mathrm { m g }$ 或 8mg 的受试者，用药一段时间后，若研究者判定疾病存在进展可能，受试者安全性稳定后，则可上调剂量1 次。每例受试者最多只能进行 1 次剂量上调，且不能跨剂量组调整，同时在病历中详细记录调整的原因。

#  可预期不良反应的处理措施

由 TQB2450 注射液引发与免疫相关的不良事件类型和治疗建议参考《中国临床肿瘤学会（CSCO）免疫检查点抑制剂相关的毒性管理指南（2021）》。若出现少见的心脏毒性、肾脏毒性、神经毒性、血液毒性等，研究者应与申办方医学人员及时沟通，根据药物特点制定合适的处理措施。

以下简要介绍安罗替尼等抗血管生成药物常见的可预期性试验药物相关的不良事件及处理措施：

非血液系统毒性

非血液系统毒性可能有：恶心、呕吐、腹泻、疲劳、虚弱、高血糖等，较严重的可能有腹泻、高血糖。当发生不良反应时，须根据症状对症处理，同时按照剂量下调标准适时调整剂量，甚至退出试验。

# 【腹泻】

腹泻如程度严重或持续时间过长有可能导致脱水、电解质紊乱和败血症，从而威胁生命，因而出现腹泻时需及时处理。受试者首次出现软便、溏便或排便频次增加时，应立即通知研究医生。

出现 1~2 级腹泻症状时，可以给予对症治疗，补充水分。如果 2 级腹泻持续超过 48h，应暂停安罗替尼给予，直至症状缓解至 1 级及以下，再以低剂量重启治疗。

如果出现3级腹泻，应暂停安罗替尼治疗，并给予对症治疗，直至症状缓解至1级及以下，再以低剂量重启治疗；如果 3 级症状持续 14 天且未缓解至 1 级及以下，则给予最佳支持治疗并终止安罗替尼给予。

# 【恶心、呕吐】

出现恶心、呕吐的患者，在第一周期可进行对症治疗，而在后续周期中可预防性使用止吐药。

# 【消化道出血】

消化道出血，包括大便潜血（ $^ { + + }$ ）以上、呕血或鲜血便等，应密切监控，必要时尽早对症处理。上消化道出血者，应禁食，并给予止酸、保护胃粘膜、止血（止血环酸、立芷雪等）、输血及支持治疗，必要时可以使用奥曲肽等；下消化道出血者，应给予止血、输血及支持治疗等；出血无法控制者，需请外科协助处理。

# 【肝功能损伤】

当出现 $\textcircled{1}$ 血清ALT或 $_ { \mathrm { A S T } > 8 } { \times } \mathrm { U L N }$ ； $\textcircled{2}$ ALT 或 $_ { \mathrm { A S T } > 5 \times \mathrm { U L N } }$ ，持续2周； $\textcircled{3}$ ALT 或$_ { \mathrm { A S T } > 3 \times \mathrm { U L N } }$ ，且 TBIL>2×ULN 或 $\mathrm { I N R } { > } 1 . 5$ ； $\textcircled{4}$ ALT 或 $_ { \mathrm { A S T } > 3 \times \mathrm { U L N } }$ ，伴逐渐加重的疲劳、恶心、呕吐、右上腹疼痛或压痛、发热、皮疹和（或）嗜酸性粒细胞增多（ $5 5 \%$ ），应暂停安罗替尼给药。研究者应根据肝损伤的临床类型选用适当的药物对症治疗。

# 【手足红肿综合征】

一旦出现皮肤干燥、红肿、瘙痒、疼痛或皮疹，可加强皮肤护理，保持皮肤清洁，避免继发性感染；避免压力或摩擦；使用润肤霜或润滑剂，局部使用含尿素和皮质类固醇成分的乳液或润滑剂；必要时局部使用抗真菌或抗生素治疗，及时请皮肤科专家会诊。

# 【高血压】

高血压是 VEGFR 抑制剂常见的一类不良反应。一般来说，已知高血压的受试者在进入研究前，高血压应控制在收缩压 ${ < } 1 5 0 \ \mathrm { m m H g }$ 或舒张压 ${ < } 1 0 0 \ \mathrm { m m H g }$ 。试验期间，每次血压检测应选择固定体位（坐位或仰卧）测量。一旦出现高血压症状，应在第一次测量后至少5分钟后进行第二次测量，进行确认。

如果临床试验期间出现高血压，可参考以下原则进行治疗。

1）针对首次出现的高血压（如无须给予立即治疗），建议在 1周内进行第二次随访，确认是否属于新发生的高血压或高血压是否加重，再决定是否给予对症治疗。  
2）如果收缩压大于 $1 5 0 \ \mathrm { m m H g }$ 但小于 $1 6 0 \ \mathrm { m m H g }$ ，或舒张压大于 $1 0 0 \ \mathrm { m m H g }$ 但小于$1 1 0 \mathrm { m m H g }$ ，应给予新的抗血压药物治疗或增加原抗血压药物剂量。  
3） 如果收缩压大于 $1 6 0 \ \mathrm { m m H g }$ 或舒张压大于 $1 1 0 \ \mathrm { m m H g }$ ，应采取 $\textcircled{1}$ 下调给药剂量； $\textcircled{2}$ 给予新的抗血压药物治疗或增加原抗血压药物剂量； $\textcircled{3}$ 如果经对症治疗后，收缩压仍大于 $1 5 0 \ \mathrm { m m H g }$ 或舒张压仍 $1 0 0 \ \mathrm { m m H g }$ ，则应进一步下调剂量或暂停给药； $\textcircled{4}$ 如果经对症治疗后，收缩压仍大于 $1 6 0 \mathrm { m m H g }$ 或舒张压大于 $1 1 0 \mathrm { m m H g }$ ，则应暂停给药。  
4）如果出现高血压危象或高血压脑病，应立即终止试验药物并给予对症治疗。

# 血液系统毒性

目前没有证据表明安罗替尼会引起显著的血液系统毒性。一旦发生4级血液系统毒性反应，应立即暂停试验药物治疗，根据临床诊疗常规给予必要的处理，并密切随访，及时对受试者进行再评估。

# 3）建立服务药物临床试验阶段的药物警戒系统

药物警戒是贯穿于药品全生命周期的研究与活动，是始于药物进入人体、涵盖新药研发的全部临床试验阶段、直至药品上市后在更大规模人群中使用，持续性的对药物的安全性进行监测和风险获益评估，以保障患者的用药安全。因此，正大天晴药业集团股份有限公司在开展 I 期临床试验之前，已经制定了服务药物临床试验阶段的药物警戒系统的搭建和工作规划，在临床试验过程中将药物警戒贯穿始终，收集和记录临床试验中发生的不良事件，并按照国家法规要求和法定流程及时记录和快速上报严重不良事件。

#  关键术语及定义

# 不良事件

不良事件（Adverse Event, AE）是指任何发生在病人或药物临床试验受试者中的不利事件。它不一定与药物治疗有因果关系。不良事件可以是一种不良的未预期的征象（包括

异常的实验室检查等）、症状或与药品使用中有时间相关性的疾病，不考虑是否与药品有因果关系。

# 药物不良反应

药物不良反应（Adverse Drug Reaction，ADR）是指新药在获准批准前的临床试验中或新适应症批准之前，尤其治疗剂量未建立之前，任何有害和未预期的药品反应（任何剂量）都认为是药物不良反应。“药物反应”一词指的是其与不良事件存在至少可能相关。

# 非预期的药物不良反应

非预期的药物不良反应（Unexpected Adverse Drug Reaction）是指不良反应的性质、严重程度、后果或频率，不同于先前临床试验方案或其他相关资料（如研究者手册等文件）所描述的预期风险。研究者手册作为主要文件提供用以判断某不良反应是否预期或非预期的安全性参考信息。如： $\textcircled{1}$ 急性肾衰在研究者手册中列为可能的不良反应，但试验过程中出现间质性肾炎，即应判断为非预期不良反应； $\textcircled{2}$ 肝炎在研究者手册中列为可能的不良反应，但试验过程中发生爆发性肝炎，即应判断为非预期不良反应。

# 严重不良事件或反应

严重不良事件或反应（Serious Adverse Event or Adverse Drug Reaction，SAE or SADR）是指以下不利/不良的医学事件（在任何剂量下）：

（1）导致死亡；  
（2）危及生命；（注意：严重中的“危及生命”的定义是指严重病人即刻存在死亡的风险，并非是指假设将来发展严重时可能出现死亡。）  
（3）需要住院或现有住院时间延长；  
（4）导致永久或显著残疾/功能丧失；  
（5）导致先天性畸形或出生缺陷；  
（6）其他重要的医学事件。

重要医学事件：在某些情况下，需要基于医学和科学判断来决定是否适用快速报告，例如某些重要的医学事件虽然不会立即危及生命、导致死亡或住院，但基于医学判断，可能危害患者或受试者，或需要治疗干预以防止出现上述情况或结果，这些事件通常被认为是严重不良事件。例如，在急诊室的重要治疗或在家发生的过敏性支气管痉挛，未住院的恶液质或惊厥、产生药物依赖成瘾。

#  临床试验中安全性信息的处理

一般不良事件（AE）收集、记录

临床监查员或项目经理协助、督促、监查研究者对临床试验中发生的一般不良事件进行收集和记录。记录的内容包括：AE名称、起止时间、严重程度、严重性、处理、转归、因果关系、随访等。

严重不良事件报告及医学评价

严重不良事件发生后，研究者应立即告知临床监查员或项目经理。临床监查员或项目经理还需协助研究者填写《严重不良事件报告表》并按照公司的规定（需考虑 SUSAR 上报CDE的时限设置合理的报告时限）报告至临床医学部。报告后获得的随访信息应及时提供。

临床医学部收到严重不良事件报告后，对严重不良事件进行接收确认、录入严重不良事件数据库、信息审查及医学评价。医学评价内容：预期性、严重性、严重程度、因果关系、病例医学描述。医学评价后的报告表提供给药物警戒组。

可疑的非预期严重不良反应上报 CDE

临床医学部对严重不良事件评价后，符合可疑的非预期严重不良反应上报标准的，经内部审核审批后，按照要求上报至国家药品审评中心，上报要求如下：

$\textcircled{1}$ 采用 MedDRA 编码，符合 ICH E2B（R3）数据要求。  
$\textcircled{2}$ 对于致死或危及生命的非预期严重不良反应，在首次获知后尽快报告，但不得超过7 天，并在随后的 8 天内报告、完善随访信息。

注：正大天晴药业集团股份有限公司人员（一般为临床监查员或项目经理）首次获知当天为第 0 天。

$\textcircled{3}$ 对于非致死或危及生命的非预期严重不良反应，首次获知后尽快报告，但不得超过 15 天。  
$\textcircled{4}$ 首次报告后，持续跟踪非预期严重不良反应，以随访报告的形式及时报送有关新信息或对前次报告的更改信息等，报告时限为获得新信息起的 15天内。

妊娠事件报告

根据 ICH及相关法规要求，除不良事件外，申办方尚需收集妊娠报告。因此需在研究方案中要求报告受试者或其配偶是否有妊娠情况发生。

临床监查员或项目经理协助、督促和监查研究者对此类情况的收集和报告。妊娠情况发生后，临床监查员或项目经理填写妊娠报告表并按照规定报告。妊娠报告还需要随访至妊娠结局（如：妊娠终止、分娩）。如在妊娠期间发生以下情况者，按照严重不良事件/不良事件进行管理：胎儿/新生儿先天异常或畸形（严重不良事件）；自发性流产（严重不良

事件）；因医学原因终止妊娠（不良事件/严重不良事件）。

# 其他需要快速报告的安全性信息

临床医学部还需收集其他潜在严重安全性风险的信息。一般情况下，对于明显影响药品风险获益评估的信息或可能考虑药品用法改变，或影响总体药品研发进程的信息，均属于此类情况，包括： $\textcircled{1}$ 对于已知的、严重的不良反应，其发生率增加，判断具有临床重要性； $\textcircled{2}$ 对暴露人群有明显的危害，如在治疗危及生命疾病时药品无效； $\textcircled{3}$ 在新近完成的动物试验中的重大安全性发现（如致癌性）。

临床医学部项目组在获知以上信息后，填写报告表并按照规定报告。这类报告在临床医学部做出医学评价和科学判断后，尽快按照规定上报至国家药品审评中心。
