# 2.3.S.7 稳定性

# 1 稳定性总结和结论

本 研 究 对 TQB2858 两 批 中 试 生 产 规 模 （ 1000L ） 的 原 液 （ 批 号 ：20200303、20200405）进行了加速和长期稳定性考察，且计划选取一批中试生产规模的原液（批号：20200303）进行影响因素试验。目前为止，已开展本品的加速和长期稳定性研究，已获得加速和长期稳定性研究0月及加速稳定性研究1月数据。

原液加速稳定性试验的试验温度定为 $2 { \sim } 8 ^ { \circ } \mathrm { C }$ ，考察 6 个月，分别在试验期间第0、1、2、3、6个月末取样；长期稳定性试验的试验温度定为- $. 4 0 ^ { \circ } \mathrm { C } { \pm } 5 ^ { \circ } \mathrm { C }$ ，考察24个月，分别在试验期间第 0、3、6、9、12、18、24 个月取样。检验项目包括 pH 值、蛋白质含量、纯度（分子排阻色谱法、CE-SDS还原电泳）、生物学活性等。

加速稳定性：在 $2 { \sim } 8 ^ { \circ } \mathrm { C }$ （加速稳定性试验）条件下，TQB2858两批原液（批号：20200303、20200405）放置 1 个月，质量均无明显变化。
