# 1、研究背景

肺癌是全世界范围内一种高发病率、高死亡率的恶性肿瘤。近年来，我国肺癌的发病率和死亡率呈快速增长。目前基于肺癌的生物学特性、治疗和预后，世界卫生组织（WHO）将其分为两大类：非小细胞肺癌（non-small cell lung cancer，NSCLC）和小细胞肺癌（small cell lung cancer，SCLC）。SCLC 占所有肺癌病例的 $1 5 { - } 2 0 \%$ ，无论从组织来源、生物学特征、对治疗的反应和预后等方面都与非小细胞肺癌迥然不同，所以被认为是一种全身性的疾病。SCLC 侵袭性高，预后较差，5年生存率不足 $5 \%$ ，不治疗的患者平均生存期仅 2-4个月。

大多数SCLC 患者在初诊时失去了根治性治疗机会，但是联合化疗仍是广泛期SCLC 的有效治疗方法，可以改善症状，延长生存期。广泛期 SCLC 一线化疗缓解率为$4 0 \% - 7 0 \%$ ，中位生存期为 7-11个月。尽快初始化疗缓解率高，但多数完全缓解的患者在3个月内病情进展，远期疗效差，EP 方案是治疗广泛期 SCLC 标准的一线治疗方案。

盐酸安罗替尼是我国自主研发的 1.1类新药，是一类抗血管生成的新型小分子多靶点酪氨酸激酶抑制剂。ALTER 0303 研究证实，安罗替尼对比安慰剂用于晚期非小细胞肺癌（NSCLC）三线及以上治疗，可以同时显著延长患者的 PFS 和OS。基于这一研

究，2018年5月获NMPA批准上市。盐酸安罗替尼三线及三线以上治疗小细胞肺癌二期临床试验结果已经公布在 2018年世界肺癌大会，相比安慰剂，安罗替尼明显延长晚期小细胞肺癌患者的PFS 和OS。同时期开展的盐酸安罗替尼临床研究显示，其在治疗晚期软组织肉瘤、甲状腺髓样癌、分化型甲状腺癌等项目也效果颇佳。

TQB2450 注射液为一种全新序列的创新抗 PD-L1 全人源化单克隆抗体，由中美冠科和正大天晴药业集团股份有限公司合作开发，属于创新型单抗，注册分类为治疗用生物制品1类。程序性死亡因子 1及其配体（PD-1/PD-L1）是一对免疫共刺激因子，正常情况下，PD-1通过其配体 PD-L1发挥免疫调控作用。近年来，PD-1及其配体PD-L1 因其参与肿瘤免疫逃逸机制而受到关注。PD-1/PD-L1 信号通路的激活可导致免疫抑制性肿瘤微环境形成，使肿瘤细胞逃避机体免疫监视和杀伤，而阻断 PD-1/PD-L1信号通路可逆转肿瘤免疫微环境，增强内源性抗肿瘤免疫效应。PD-L1 高表达于各种实体恶性肿瘤，包括非小细胞肺癌、黑色素瘤、肾细胞癌、前列腺癌、乳腺癌、胶质瘤等，其表达水平因肿瘤类型不同而不同，不仅能促进肿瘤细胞的生长，还可诱导 T淋巴细胞的凋亡。

# 2、研究目的

机制研究中，肿瘤血管异常会促进组织乏氧、乳酸增多，从而激活免疫抑制，抑制T细胞功能；安罗替尼作为一个多靶点的小分子酪氨酸激酶受体抑制剂，主要通过抑制血管内皮细胞生长来阻止新生血管的生成，诱导血管正常化、降低免疫抑制而增强效应免疫细胞的浸润。

# 8、参加研究可能的风险

# 疗效性风险：

虽依托泊苷、卡铂以及盐酸安罗替尼胶囊作为新型小分子多靶点酪氨酸激酶抑制剂治疗晚期非小细胞肺癌已经上市，但 TQB2450 注射液目前还处于临床试验阶段，未经国家相关部门批准上市。因此尚不能保证您的肿瘤能得到控制和缓解。

# 抽血的风险

从胳膊抽血的风险包括短暂的不适和/或青紫。尽管可能性很小，也可能出现感染、出血过多、凝血或晕厥的情况。

# 计算机断层扫描(CT)风险：

X 线是无痛的。增强 CT 通过静脉注射使用造影剂（或染料）可对注射部位造成轻微的烧灼感。也可以导致口中有金属味道。一些人群会主诉身体内有热感。这些感觉都正常并且会在几秒钟内消失。罕见的反应为一些人可能会对造影剂产生危及生命的过敏反应。

# 磁共振成像(MRI)扫描风险：

由于MRI扫描使用了强力磁体，如果您体内有任何金属物体，例如骨钉、颅骨板或

支架，您不应接受MRI扫描。如果您体内有起搏器或除颤器等医疗器械，您也不应接受MRI 扫描。

# 药物不良反应风险

TQB2450 注射液可能引起的不良反应包括：肺炎、胃肠道不良反应（腹泻、腹痛、便血等）、皮炎（斑疹、丘疹等）、肝炎（ALT、AST、总胆红素升高等）、内分泌不良事件（下垂体炎、甲状腺功能减退、甲状腺功能亢进、I型糖尿病）、风湿病/肌肉骨骼免疫相关不良事件（炎症性关节炎等）、贫血、肾炎、输注部位反应（触痛或伴发热、红斑、瘙痒、水肿等）等。

TQB2450注射液在输注后可能出现输液反应，表现为输液后发热、寒战、头晕或者呼吸困难等症状。在每次输注前研究医生将给您使用预防输液反应发生的药物，并在每次输液过程中密切监测您的情况，在出现的输液反应时立即采取相应的治疗措施。

盐酸安罗替尼胶囊不良反应包括手足综合征、蛋白尿、甲低、乏力、肝功能升高、腹泻、血脂异常、胆红素升高，发生率略低的不良反应有白细胞降低、血淀粉酶升高、肌酐升高、头晕、声音嘶哑、胸腔积液加重、肌肉酸痛、耳鸣、牙痛、发热、气胸等。

卡铂的主要不良反应包括骨髓抑制伴血小板减少、白细胞减少、中性粒细胞减少和贫血；传染性并发症；恶心和呕吐；耳毒性；和周围神经病。

依托泊苷主要不良反应包括可逆性的骨髓抑制，包括贫血、白细胞及血小板减少；食欲减退、恶心、呕吐、口炎、腹泻等消化道反应等；

参加本项研究的用药过程中，您可能会发生以上不良反应，也有可能出现尚未发现的或未能预知的不良反应，可能会很严重，甚至危及生命。整个研究过程中，任何改变和任何新近明确的不良反应都会由您的负责医生及时告知您。

# 9、参加研究可能的受益

考虑到本品的同类药物 Tecentriq 的前期研究的结果，参加本次临床研究后，您有可能从中受益，但也有可能达不到预期的效果。

研究过程中，当疗效评价为受益，不良反应可耐受情况下，您将可以持续使用研究药物，直至负责医生认为您不适合继续用药或疗效评价为疾病进展时结束。

也许您没能从本次研究中直接受益，但因为有您的参与，在对此疾病和相关药物的研发中，有您做出的积极贡献，本研究协作组向您表示由衷的感谢。

# 10、研究的补偿

研究过程中，申办方将免费提供研究药物。研究期间与试验相关的各项检查费用由

申办方承担，如感染筛查（包括乙肝、丙肝、HIV）、血常规、尿常规、便常规、血生化、血清癌胚抗原CEA、神经烯醇化酶 NSE、超声心动图、心电图检查及影像学检查等。

您参加研究前同时伴随的其他疾病所需要的治疗和检查，将不在免费范围内。

此外，对于方案规定的每次访视申办方将提供 200元交通补贴，对于免疫原性血样采集，每个采血点将提供 200元营养补贴。因本次研究产生的补贴，会在您完成研究后由研究医生/护士发放给您，即使您中途退出研究或未完成研究，您也可以得到相应比例的补贴。

# 11、本次研究之外的备选疗法

参加这项研究对于治疗您的疾病或病情并不是必须的。对于您的疾病状况还存在其他的治疗方法可供选择，如：依托泊苷 $^ +$ 顺铂（EP方案）等治疗。其获益和风险将与具体的治疗方案有关，您可能从中获得疾病的控制或缓解，但同时也可能会出现一些治疗常见的毒副反应。您的研究医生会向您介绍具体的治疗方案并解释接受其他治疗方案时会发生哪些好的和坏的情况。他/她会回答您的所有疑问。

# 10、退出研究

研究过程中，按照方案设计如果觉得您不适合再继续用药，负责医生会向您主动解释原因并中止本次药物研究。

您参加本项研究完全自愿，您有权选择不参加本次研究，研究过程中也有权随时退出，您的后续治疗不会受到任何影响，仍可继续得到其他治疗；如果您准备退出研究，请及时告诉您的负责医生。为了您的安全，他将对您进行一次全面检查。

一旦选择参加本次研究，我们希望您在无特殊原因的情况下，按时用药并按约定的时间进行访视和检查。

# 12、研究的赔偿

在研究过程中出现的不适或意外情况，不管是否与研究有关，均请您即刻告知您的负责医生，他将对此作出判断和专业医疗处理。经确认不良事件与本研究有关，并由此所产生的损伤，申办方正大天晴药业集团股份有限公司将按照中国的法律法规承担治疗的费用及相应的经济补偿。申办方已为本次研究购买了药物临床试验责任保险。

您参加研究前同时伴随的其他疾病所需的治疗和检查，将不在赔偿范围之内。

# 13、个人信息的保密

我们从本研究中收集到的信息将作为机密保存。为了保护您的身份，在研究文件

中任何关于您的信息都将使用一个编号，而不是使用您的姓名。所收集和合并的所有受试者信息中，任何可以帮助识别您的身份的信息都将被去除，保证有关信息不能链接到一个特定的研究受试者。

一旦您参加研究，正大天晴药业集团股份有限公司、研究医生和其他研究需要的工作人员（包括研究项目所指定的数据管理、分析或研究监查与质量控制人员）都将有权利检查您的医学记录和相关数据。这样做的目的是分析研究结果并确保研究根据方案或法规要求执行。此外，机构审查委员会、伦理委员会或国家食品药品监督管理总局的代表将可能在相关中国法律或法规指定的范围内稽查这些相关记录以确保研究得到正确的执行和收集到可靠的数据。所有这些人都将对您作为一名研究的受试者有保密的责任。并且可以提供您身份的任何信息将不会被披露给医院/机构以外，除非被法律要求。

研究的数据将由独立的部门进行检查和分析，并最终根据相关法律或法规在安全的场所保存这些数据。研究期间收集到的所有医疗信息都将以书面报告的形式放在一起，其中将不会提及您的姓名，只可能使用您的拼音首字母缩写，出生日期和您的个人研究编号。

本研究的结果可能在医学杂志中发表或出于科学目的进行分享或由正大天晴药业集团股份有限公司用于产品研究或改进，但您的身份和个人信息在任何时候都将不会被披露。
