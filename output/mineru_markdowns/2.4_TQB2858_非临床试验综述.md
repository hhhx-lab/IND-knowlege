# 2.4.1 非临床试验策略概述

本品是一种双功能的融合蛋白（研发代号 TQB2858），临床拟用于晚期恶性实体/血液肿瘤患者静脉输注给药。拟定的临床起始剂量为 30mg/人 q3w，后续按剂量递增方案进行剂量爬坡，直至确定 MTD 或研究者认为无需继续剂量爬坡为止。

本品为 PD-L1/TGF- $\beta$ 双功能融合蛋白，由两个部分构成的。一端是能够识别结合 PD-L1 的抗体结构（Y），是本司的 TQB2450，处于临床 III 期的PD-L1 抗 体 ； 另 一 端 是 可 以 结 合 TGF- $\cdot \beta$ 的 TGF- $\beta$ 受 体 II 型 融 合 蛋 白（Trap），可以捕获游离的 TGF-β1。PD-L1 表达在多种肿瘤组织上，与 T 细胞上的受体 PD-1 结合负向调控免疫应答。多种因素可诱导肿瘤细胞上 PD-L1 的表 达 ， 同 时 表 达 的 PD-L1 有 利 于 肿 瘤 的 发 生 和 生 长 。 转 化 生 长 因 子 -β（transforming growth factor- $\beta$ ，TGF- $\cdot \beta$ ）是一种多功能细胞因子，共有 3 个亚型，即 TGF- $\cdot \beta 1$ 、TGF- $\cdot \beta 2$ 、TGF- $\cdot \beta 3$ 。在晚期肿瘤阶段，肿瘤微环境中癌细胞、间质成纤维细胞和内皮细胞等大量释放的 TGF-β 能够使癌细胞具有侵袭性、扩散性、干细胞特性以及对治疗产生耐药性。本品能够同时阻断 PD-1/PD-L1 通路及 TGF- $\beta$ 通路从而促进免疫细胞对肿瘤细胞的杀伤作用。

因此，在靶向中和肿瘤微环境的 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 基础上抑制 PD-1/PD-L1 通路，可以使 T 细胞恢复活性，增强免疫应答，更有效地提高抑制肿瘤发生和发展的效果。

本部分资料将对本品的非临床研究的药理学、药代动力学、毒理学试验进行综合性评价。

本次提交的药理学研究资料包括主要药效学（体外药效学及体内药效学）、安全药理学。其中安全药理学试验遵循国家药品监督管理局（NMPA）的 GLP

法规。体外药效学结果表明：本品选择性与人 TGFβ 结合、人 PD-L1 结合，明显抑制 TGFβ1 刺激的荧光素酶报告基因表达，明显促进 SEB 刺激或 MLR 中$\Sigma \Sigma 4 ^ { + }$ T 细胞分泌 IFN-γ；无明显 ADCC；本品在体外于 PBMC 共同孵育，细胞因子产生情况与同型对照组 hIgG1 相当。总体而言，本品对 TGFβ 的体外活性与 IgG1-TGFβ 相当，对 PD-L1 的体外活性与 TQB2450 相当。体内药效学结果表明：本品对小鼠结肠癌 MC38/hPD-L1 皮下移植瘤生长有明显抑制作用，且呈现剂量依赖性。安全药理试验伴随在食蟹猴 4 周重复给药毒性试验，考察了单次静脉注射本品对食蟹猴体温、呼吸频率及心血管系统的影响，结果表明食蟹猴单次静脉注射本品对其体温、呼吸频率及心血管系统均无损害作用。

药代动力学方面，考察了食蟹猴单次静脉注射本品后体内的药代动力学特征。同时，考虑到重复给药药代动力学试验剂量与 4 周重复给药毒性试验剂量接近，因此重复给药的药代动力学结果参考毒代动力学结果。单次给药药代动力学试验为 Non-GLP 条件，重复给药和药代动力学生物分析方法学验证遵循GLP 法规（与毒代动力学采用同一套方法验证）。

# 2.4.4.3 重复给药毒性

本试验共使用 40 只食蟹猴，分为 4 组，每组 10 只，雌雄各半，分别为对照组（TQB2858 注射液（空白））以及本品 20、60、 $2 0 0 \mathrm { m g / k g }$ 组。每周 1 次、连续给药 4 周（共 5 次），停药恢复观察 6 周。

给药结束时，本品 60、 $2 0 0 \mathrm { m g / k g }$ 组可见轻度以 RBC、HGB、HCT 降低为主的贫血样改变以及 RET、RET%升高， $2 0 0 \mathrm { m g / k g }$ 组还可见骨髓涂片轻度红细胞系、晚幼红细胞比例升高。组织病理学检查骨髓及脾脏未见相关改变。TGF-β1 具有促进红系祖细胞增殖、分化，以及促进血红蛋白生成的生理作用。结合TGF-β1 的生理作用及供试品的作用机制，上述改变与本品药理作用相关，其中RET、RET%、红细胞系、晚幼红细胞升高为代偿性改变。至恢复期结束，200$\mathrm { m g / k g }$ 组 1 只雌猴 HGB、HCT 仍未见明显恢复，红细胞形态学见血红蛋白含量降低。上述症状恢复较慢，可能与该只猴给药后药理学效应（如 TGF- $\cdot \beta 1$ 的变化情况）维持时间较长及恢复期结束时仍可检测到一定的药物浓度相关联。此外，各给药组其余猴上述指标均可见不同程度恢复。

# 2.4.5 综合评估和结论

根据 ICH S6《生物技术药物的临床前安全性评价》指导原则，我们对本品主要药效学、安全药理学、药代动力学、单次给药毒性、重复给药毒性、溶血性、刺激性等方面进行了非临床研究。

药理学方面，进行了体外药效学、体内药效学及安全药理学研究。体外药效学结果表明：本品选择性与人 TGFβ 结合、人 PD-L1 结合，明显抑制 $\mathrm { T G F } \beta 1$ 刺激的荧光素酶报告基因表达，明显促进 SEB 刺激或 MLR 中 $\mathrm { C D } 4 + \mathrm { T }$ 细胞分泌IFN-γ；无明显 ADCC；本品在体外于 PBMC 共同孵育，细胞因子产生情况与同型对照组 hIgG1 相当。总体而言，本品对 TGFβ 的体外活性与 IgG1-TGFβ 相当，对 PD-L1 的体外活性与 TQB2450 相当。体内药效学结果表明：本品对小鼠结肠癌 MC38/hPD-L1 皮下移植瘤生长有明显抑制作用，且呈现剂量依赖性。安全药理试验结果表明食蟹猴单次静脉注射本品对其体温、呼吸频率及心血管系统均无损害作用。

药代动力学方面，考察了食蟹猴单次静脉注射本品后体内的药代动力学特征。重复给药的药代动力学伴随在毒理实验中进行。单次静脉输注 $\mathrm { 1 - 6 0 ~ m g / k g }$ 本品后，血清药物浓度水平随给药剂量增加而增加，各给药组主要药代动力学参数无明显性别差异，暴露量与剂量成比例增加，各剂量组给药后均检出抗药抗体。多次给药后首、末次给药后各剂量下未见明显性别差异，暴露量增加与剂量增加比例基本一致，未见明显蓄积。

毒理方面，食蟹猴单次静脉注射本品 MTD 为 $1 0 0 0 \mathrm { m g / k g _ { \circ } }$ 。食蟹猴每周 1 次、连续 4 周（5 次）静脉注射 $2 0 { - } 2 0 0 ~ \mathrm { m g / k g }$ 的 TQB2858 注射液，HNSTD 为 200mg/kg。主要毒性反应为包括 TGF- $\cdot \beta 1$ 降低所致的轻度外周血 RBC、HGB、HCT降低以及 PD-L1 免疫相关的胸腺淋巴细胞减少及多器官（脑膜、甲状腺和肾脏）单核细胞浸润，毒性反应均为药理学作用相关，文献和同类药物均可见，恢复期结束可见一定程度恢复。

本品在体外无溶血或红细胞聚集作用，无明显的局部刺激性。
