# 2 分析方法

# 2.1 相对结合活性

本方法采用的是酶联免疫吸附测定（ELISA）方法。将 PD-L1 蛋白、TGF-β 蛋白分别包被在微孔板上，清洗封闭，然后将参比品和供试品稀释至特定浓度，与包被蛋白孵育结合，清洗后加入 HRP 标记羊抗人 IgG 抗体孵育，洗涤微孔板以除去未结合的反应物，加入底物（TMB）进行显色反应，用终止液终止，在 $4 5 0 \mathrm { n m } /$ 650nm的波长处测定吸光度。参比品、供试品和包被蛋白的结合量与吸光度值呈正相关。通过四参数 Logistic 曲线拟合获得参比品和供试品 EC50 值，并计算参比品和供试品 EC50 的比值，得出供试品抗 PD-L1 端与 PD-L1 蛋白、TGF- $\beta$ 受体端与 TGF- $\beta$ 蛋白的相对结合活性。采用该方法完成了 TQB2858 三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2.2 外观

取供试品5瓶，去除容器标签，擦净容器外壁，裸眼观察。采用该方法完成了TQB2858 三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2.10 分子排阻色谱法

分子排阻色谱法是根据待测组分的分子大小进行分离的一种液相色谱技术。其分离原理是凝胶色谱柱的分子筛机制。药物分子进入色谱柱后，不同组分按其

分子大小进入相应的孔径内，大分子物质在色谱柱中的滞留时间较短，出峰较早小分子物质在色谱柱中滞留时间较长，出峰较晚。本法依据《中国药典》（2015年版 ） 通 则 0514 ， 色 谱 柱 为 ACQUITY UPLC Protein BEHSEC（200Å， $1 . 7 \mu \mathrm { m }$ ， $4 . 6 ^ { * } 3 0 0 \mathrm { m m }$ ）色谱柱或性能相当的色谱柱；流动相为50mmol/L 磷酸盐缓冲液-200mmol/L 氯化钠，pH7.0；流速为每分钟 $0 . 3 \mathrm { m l }$ ；检测波长为 $2 8 0 \mathrm { n m } _ { \circ }$ 。用流动相将供试品稀释至每 lml 中约含蛋白质 $1 0 \mathrm { m g }$ ，作为供试品溶液，取供试品溶液 $2 \mu \mathrm { l }$ （ $2 0 \mu \ g$ 蛋白质）注入液相色谱仪。采用面积归一化法计算高分子杂质和免疫球蛋白单体含量。采用该方法完成了TQB2858三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2.11 CE-SDS 还原电泳

依据《中国药典》（2015年版）通则3127，通过加入还原剂将抗体分子中的二硫键打开，进而分离蛋白的轻链（LC）、重链（HC）以及降解物等杂质。该方法使用面积归一化法计算峰面积百分比，进而计算降解物含量。采用该方法完成了TQB2858 三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2.12 聚山梨酯 80 含量

本方法采用反相和阴离子交换混合模式色谱柱对聚山梨酯 80进行分离，采用低 pH 值的有机相作为流动相进行洗脱，供试品中的蛋白质在低 pH 值下带正电荷，与色谱柱存在静电排斥被洗脱下来，一些中性糖组分因为亲水性也会被洗脱，聚山梨酯80吸附在色谱柱上，随有机相增加被洗脱，通过蒸发光散射检测器检测。依据《中国药典》（2015年版）通则0512，色谱柱为反相和阴离子交换混合模式色谱柱（如：Waters Oasis MAX色谱柱）；流动相A为 $2 \%$ 甲酸水溶液，流动相B为2%甲酸-异丙醇溶液；采用蒸发光散射检测器检测。用水将聚山梨酯80对照品分别稀释至 0.30mg/ml、0.25mg/ml、0.20mg/ml、0.16mg/ml 和 $\mathrm { 0 . 1 0 m g / m l }$ ，作为标准线性溶液，直接取供试品作为供试品溶液。取标准线性溶液和供试品溶液各$3 0 \mu \mathrm { l }$ ，注入液相色谱仪，记录峰面积。采用外标法计算供试品中聚山梨酯80的含量 。 采 用 该 方 法 完 成 了 TQB2858 三 批 注 射 液 （ 批 号 ：BP20200101、BP20200502、BP20200503）的放行。

# 2.13 生物学活性

1 PD-L1

本实验采用报告基因的方法。根据 PD-1/PD-L1免疫疗法的作用机制来针对PD-1或PD-L1设计特定的抑制剂，阻止PD-1和PD-L1的识别过程，部分恢复T细胞功能，通过阻断 PD-1/PD-L1 信号通路使肿瘤细胞死亡。本实验以 CHO-PD-L1-CD3L 作为靶细胞；Jurkat-PD-1-NFAT 作为效应细胞。PD-1 与 PD-L1 结合可以阻断 CD3 下游信号的转导，从而抑制萤光素酶的表达，当加入 PD-1/PD-L1 抗体时，这种 Block 效应被反转，萤光素酶可以表达，从而检测到萤光信号。采用该方法完成了 TQB2858 三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2 TGF-β

本方法采用的是酶联免疫吸附测定（ELISA）方法。将 TGF- $\beta$ 蛋白包被在微孔板上，清洗封闭，然后将参比品和供试品稀释至特定浓度，与包被蛋白孵育结合，清洗后加入HRP标记羊抗人IgG抗体孵育，洗涤微孔板以除去未结合的反应物，加入底物（TMB）进行显色反应，用终止液终止，在450nm/650nm的波长处测定吸光度。参比品、供试品和包被蛋白的结合量与吸光度值呈正相关。通过四参数 Logistic 曲线拟合获得参比品和供试品 EC50 值，并计算参比品和供试品 EC50

的比值，得出供试品 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 受体端与 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 蛋白的相对结合活性。采用该方法完成了 TQB2858 三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2.14 蛋白质含量

本方法采用紫外-可见分光光度法对TQB2858样品进行蛋白质含量测定。由于蛋白质分子中含酪氨酸、色氨酸和苯丙氨酸等苯环结构，在 $2 8 0 \mathrm { n m }$ 波长附近有最大吸收峰，根据朗伯-比尔定律，蛋白溶液在某一固定紫外波长的吸光度值（A）与蛋白质的消光系数（ε）、蛋白质含量（C）以及光透过路径（L）存在一定关系：A=εCL。因此，蛋白溶液的蛋白质含量可以通过固定波长的吸光度值、比色皿的光径和该蛋白物质的消光系数计算获得。本法依据《中国药典》（2015年版）通则0401，用磷酸盐缓冲液（量取200mmol/L 磷酸氢二钠溶液 81ml、200mmol/L 磷酸二氢钠溶液 $1 9 \mathrm { m l }$ ，加水约 $8 0 0 \mathrm { m l }$ ，混匀，用磷酸调节pH值至 $7 . 3 { \pm } 0 . 1 $ ，加水至$1 0 0 0 \mathrm { m l }$ ，混匀）将供试品稀释至每lml中约含蛋白质 $0 . 5 \mathrm { m g }$ ，作为供试品溶液；以磷酸盐缓冲液作为空白，测定供试品溶液在波长 $2 8 0 \mathrm { n m }$ 处的吸光度，以消光系数为1.30AU/mg/ml计算供试品溶液的蛋白质含量，再乘以稀释倍数即得。采用该

方 法 完 成 了 TQB2858 三 批 注 射 液 （ 批 号 ：

BP20200101、BP20200502、BP20200503）的放行。

# 2.16 细菌内毒素检查

依据《中国药典》（2015年版）通则1143，内毒素与鲎试剂混合后可以引起鲎试剂发生一系列的反应，最终激活凝固酶，使鲎试剂中的凝固蛋白原变为凝固蛋白，产生凝集反应。凝集反应的程度与内毒素的含量是相关的。内毒素测定时首先需要确定最大有效稀释倍数。计算公式为：MVD（最大有效稀释倍数） ${ = } _ { \mathrm { C L } } / \lambda$ ，其中L为产品需控制的内毒素限值，c为样品溶液的浓度，λ为鲎试剂的灵敏度。将待测样品按上述方法计算的MVD进行稀释后与鲎试剂在 $3 7 ^ { \circ } \mathrm { C }$ 的条件下孵育1小时，若混合液凝结坚实，可以达到翻转不掉落的程度，结果记为阳性 $^ { 6 6 } + \boldsymbol { ^ { 5 } }$ ，表明内毒素含量高于给定的限值，否则结果为阴性“-”，表明内毒素含量低于给定

的 限 值 。 采 用 该 方 法 完 成 了 TQB2858 三 批 注 射 液 （ 批 号 ：

BP20200101、BP20200502、BP20200503）的放行。
