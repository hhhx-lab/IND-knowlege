# 一、基本情况

肿瘤免疫疗法是当前肿瘤治疗领域中最具前景的研究方向之一，《Science》杂志将其评为 2013 年十大科学突破的第一位。PD-1/PD-L1 抑制剂就是其中备受瞩目的一类药物。PD-1（Programmed Death-1）和 PD-L1（Programmed death-ligand 1）都是免疫反应节点调节因子，在体内能够抑制 T细胞的浓集和表达。PD-1/PD-L1 抑制剂通过阻断 PD-1/PD-L1 信号通路，使 T 细胞恢复活性，从而增强免疫应答，具有治疗多种类型肿瘤的潜力。

目前已上市的 PD-1 抑制剂有百时美施贵宝公司的 Opdivo 和美国默克的 Keytruda。除上述两个已上市药物外，还有众多处于 II期或 I期临床试验阶段的 PD-1抑制剂。

Opdivo（Nivolumab）是一种对 PD-1 有抑制作用的单克隆抗体，由 Ono 和 Medarex(隶属于百时美施贵宝)合作开发，用于肿瘤和慢性病毒感染的治疗。2014 年9月2日在日本首次上市，被批准用于黑色素瘤，这是全球第一个 PD-1 抑制剂； 2016 年 2 月 2日，日本批准 Opdivo 用于不可手术切除的晚期或复发性非小细胞肺癌。目前已批准的适应症还有肾癌、霍奇金淋巴瘤，处于 III 期临床阶段的适应症有头颈部癌、胃癌、脑癌、小细胞肺癌、肝癌等。

Keytruda（Pembrolizumab）是一种对 PD-1 有抑制作用的单克隆抗体，由美国默克公司开发上市，用于肿瘤的治疗。2014年10月 28日在美国首次上市，被批准用于黑色素瘤，是在美国上市的第一个 PD-1 抑制剂。目前已批准的适应症还有非小细胞肺癌，处于 III期临床阶段的适应症有肾癌、膀胱癌、尿道癌、胃癌等。

目前已上市的 PD-L1 抑制剂只有罗氏制药公司的 Tecentriq（atezolizumab）。Tecentriq是一种对 PD-L1 有抑制作用的单克隆抗体，用于治疗肿瘤。2016 年 5 月，被加速批准Tecentriq 用于治疗最常见类型的膀胱癌——尿路上皮膀胱癌，该药也由此成为 FDA 批准的首个抗PD-L1抗体；除了疗效，Tecentriq 和化疗相比严重副作用发生率低一半，对于这些年龄较大患者来说这个优点非常重要。

本产品为一种全新序列的创新抗 PD-L1全人源化单克隆抗体，由中美冠科与本公司合作开发。属于创新型单抗，注册分类为治疗用生物制品 1类。开发剂型为注射液，根据研究结果暂定规格为 $1 0 \mathrm { m l : 1 0 0 m g }$ 。本项目简称为 TQB2540注射液。

# 3、作用机制

PD-L1可能是在肿瘤细胞和/或肿瘤浸润免疫细胞上表达，可以抑制肿瘤微环境中抗肿瘤免疫应答反应。T 细胞和抗原呈递细胞上，PD-L1 与PD-1和 B7.1 受体结合会抑制细胞毒性T细胞活化、T细胞增殖和细胞因子产生。

本品是一种可与 PD-L1 结合的单克隆抗体，可阻断 PD-L1 与 PD-1 和 B7.1 受体相互作用。这释放了 PD-L1/PD-1 介导的免疫应答的抑制，包括无诱导抗体依赖的细胞毒效应的抗肿瘤免疫应答的激活。在同源鼠肿瘤模型研究中，阻断 PD-L1 活性可降低肿瘤生长。

# 体外结果：

TQB2450 与人 PD-L1 蛋白结合，其 $\mathrm { E C } _ { 5 0 }$ 为 $2 1 . 3 \mathrm { n g / m L }$ ，Tecentriq®同样与人 PD-L1蛋白结合， $\mathrm { E C } _ { 5 0 }$ 为 $2 7 . 6 \mathrm { n g / m L }$ ，两者结合活性基本相当。

TQB2450 及 Tecentriq®均显著诱导 CD4+ : DC 细胞 MLR 中 $\mathrm { I F N } { - } \gamma$ 的分泌，其作用呈明显的剂量依赖性， $\mathrm { E C } _ { 5 0 }$ 分别为 $3 5 . 0 \pm 1 1 . 3 ~ \mathrm { n g / m L }$ 及 $2 6 . 6 \pm 1 . 4 ~ \mathrm { n g / m L }$ ，两者刺激活性基本相当，说明 TQB2450 与 DC 表达的 PD-L1 结合后，抑制 PD-L1/PD-1 信号通路，从而刺激 $\mathrm { C D 4 ^ { + } T }$ 细胞中 $\mathrm { I F N } { - } \gamma$ 的分泌。

在Treg与DC 的混合反应中， $\mathrm { I F N } { - } \gamma$ 基本没有分泌，TQB2450也没有刺激活性；在无 Treg 的 $\mathrm { C D 4 ^ { + } C D 2 5 ^ { - } \Delta T }$ 细胞与 DC 的混合反应中，IFN-γ 有显著的分泌，而 TQB2450可以显著刺激 $\mathrm { I F N } { - \gamma }$ $\cdot \gamma$ 的分泌；若将 Treg、 $\mathrm { C D } 4 ^ { + } \mathrm { C D } 2 5 ^ { - } \mathrm { T }$ 细胞与DC 同时混合反应，可发现Treg明显抑制 $\mathrm { C D } 4 ^ { + } \mathrm { C D } 2 5 ^ { - } \mathrm { T }$ 细胞与DC 混合后 IFN-γ分泌，而TQB2450 则显著逆转Treg 的抑制活性。总体而言，TQB2450 与对照抗体 Tecentriq®活性相当。

# 2、临床前药代动力学研究概述

食蟹猴单次静脉滴注不同剂量本品（1，10， $6 0 \mathrm { m g / k g } \mathrm { , }$ ）后，血清药物暴露水平呈明显剂量依赖，呈线性药代动力学特征。多次给药（ $1 0 ~ \mathrm { m g / k g }$ ，每周 1 次，连续 4 次）后，达峰时间，半衰期等与单次给药无统计意义差别，而峰浓度，药物暴露水平较单次给药增高；此给药间隔（168hr）下蓄积因子为 $1 . 7 3 \pm 0 . 6 5$ 。

本品放射性分布按 AUC 高低排序依次为：血清、肺、肝、骨髓、生殖腺、心脏、肾上腺、脾、肾、膀胱、淋巴结、小肠、颌下腺、脂肪、胸腺、眼球、大肠、尿液、胰腺、肌肉、脑。血流灌注丰富的组织器官的放射性较高，如：肺、肝、骨髓、生殖腺、心脏等；而血流灌注不丰富器官的放射性分布较少，如脂肪、眼球、肌肉等。脑浓度很

低，提示药物不易透过血脑屏障。

本品排泄速率较慢，主要经尿排泄，少量经粪排泄。

# 3.4 生殖毒性

本品并未进行生殖毒性研究。已上市 PD-L1药物的非临床研究证明，通过保持母体对胎仔的免疫耐受来维持妊娠是 PD-1/PD-L1 通路的主要功能之一。阻断妊娠啮齿类动物模型的PD-L1信号通路可破坏母体对胎仔的耐受性，导致胎仔丢失增加。妊娠期间给予本品有潜在的风险，包括流产或死胎的风险增加。基于本品的作用机制，胎仔暴露于本品可增加发生免疫介导紊乱或改变正常免疫应答的风险。 根据 ICH《S9：抗肿瘤药物非临床评价指导原则》，本品通过文献评估存在引起生殖发育毒性风险，无需进行生殖发育毒性试验。

# 2、临床安全性文献总结

2019 年 4 月，JAMA oncol 上发表了一项综合分析 PD-1 和 PD-L1 单药治疗多种肿瘤的临床研究中治疗相关性不良反应的回顾和荟萃分析[9]。该荟萃分析，在 PubMed、Web of Science、Embase 和 Scopus 中搜索从 2017 年 10 月 1 日到 2018 年 12 月 15 日的数据，共纳入125项临床试验，涉及 20128例患者的历史数据，从纳入的每一项研究中提取试验名称、分期、癌症类型、所用 PD-1 和 PD-L1 抑制剂、剂量递增、剂量用表、患者人数、所有不良事件的数量以及不良事件报告数据的标准，并用贝叶斯多元回归模型进行数据分析，治疗相关不良事件的发生率以及不同药物和癌症类型之间的差异。

不同癌症类型平均不良事件发生率的亚组分析：不同癌症类型之间，所有级别和 3级或更高级不良事件的总体平均发生率没有差异。我们没有进一步研究特定的不良事件在特定癌症类型中是否更常见（例如，肺癌中的肺炎或胃肠癌中的结肠炎），这是未来分析的潜在焦点。

药物和剂量平均不良事件发生率的亚组分析：对于 nivolumab，不同给药方案之间的所有等级或3级或更高的不良事件的平均发生率没有发现统计学上显着的差异。对于pembrolizumab 和 atezolizumab 也是如此。
