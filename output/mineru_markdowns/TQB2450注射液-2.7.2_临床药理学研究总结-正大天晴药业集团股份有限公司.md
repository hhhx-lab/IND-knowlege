# 2.7.2.1 背景和综述

本品是抗PD-L1单克隆抗体注射液，临床给药方式为与生理盐水配伍稀释后静脉输注，生物利用度高。本品未进行相关人体生物材料研究。本品的药代动力学（PK）、药效动力学（PD）和免疫原性研究来自一项在晚期恶性肿瘤患者中开展的 I 期耐受性和药代动力学研究，项目代号为TQB2450-I-01。由于本品为针对肿瘤高表达的抗原的单克隆抗体，所以未开展健康受试者的PK 和PD 研究。

# 2.7.2.2单项研究结果总结

TQB2450-I-01 研究为一项在晚期恶性肿瘤患者中开展的 I 期耐受性和药代动力学研究，给药方案为 1、3、10、20、30 mg/kg、 $1 2 0 0 \mathrm { m g / }$ 次，每3 周静脉滴注一次, 21 天为一个治疗周期。数据截止2021 年5月1日，入组 40例晚期恶性肿瘤患者进行剂量递增研究和扩展研究。目前已出组 39例，尚余1例患者在组。

#  药效动力学

体外外周血单个核细胞（PBMC）的受体占位率（ ${ \mathrm { R O } } \%$ ）试验，选取 $0 . 0 0 3 { \sim } 0 . 3 \mu \mathrm { g / m L }$ 剂量，3~4个供体，流式检测。结果显示， $0 . 0 6 2 5 \mu \mathrm { g / m L }$ 的 TQB2450 可使不同供体的 PBMC的 $\mathrm { R O \% }$ 均 ${ > } 9 0 \%$ （均值 $9 9 . 0 8 \%$ ）。

I 期临床研究中（TQB2450-I-01）， $3 – 3 0 \mathrm { m g / k g }$ 和 $1 2 0 0 \mathrm { m g }$ 固定剂量共计 26例受试者检测了给药后 $\mathrm { R O \% }$ 情况。给药21天后， $\mathrm { R O \% }$ 基本都在 $8 0 \%$ 以上，基本判定药物可维持饱和至下次给药前，以保证药效持续发挥，也说明了 3周给药一次的可行性。

# 2.7.2.4 特殊研究

#  免疫原性

I期临床研究中（TQB2450-I-01），入组 40例受试者，抗药抗体（ADA）检测阳性率为 $3 2 . 5 \%$ （13/40），随着剂量的增加，ADA 阳性率降低。ADA 阳性率的高低和检测方法的灵敏性及特异性密切相关，并且受多种因素的影响，包括分析方法、样本的处理方法、样本的收集时间、合并用药，以及患者的其他基础疾病等。因此，不同产品的 ADA 阳性率不适合直接比较。

ADA阳性受试者的血药浓度降低，随着剂量的增加影响程度减小。

ADA 阳性受试者未见异常安全性信号。
