# TQB2450 注射液说明书

请仔细阅读说明书并在医师指导下使用

# 【药品名称】

通用名称：TQB2450 注射液

商品名称：暂无

英文名称：TQB2450 Injection

汉语拼音：TQB2450 Zhusheye

# 【成份】

活性成份：重组抗 PD-L1 人源化 IgG1 单克隆抗体。

辅料：组氨酸、蔗糖、聚山梨酯 80、盐酸、注射用水。

# 【性状】

本品为澄清至微乳光，无色至淡黄色液体。

# 【适应症】

本品联合盐酸安罗替尼胶囊、化疗（注射用卡铂、依托泊苷注射液），适用于初治的广泛期小细胞肺癌的治疗。

# 【规格】

$6 0 0 \mathrm { m g }$ （20 ml）/瓶。

# 【用法用量】

本品应在有抗肿瘤药物使用经验医生的指导下使用。

# 用法用量：

TQB2450 注射液： $1 2 0 0 \mathrm { m g / }$ 次，每个治疗周期第 1 天静脉输注。

盐酸安罗替尼胶囊： $1 2 \mathrm { m g / }$ 次，每日 1次，早餐前空腹口服，连续服用2周停用1周。无特殊情况，建议每天固定时间服用；用药期间如出现漏服，确认距下次用药时间短于12小时，则不再补服。

每 21天为一个治疗周期，治疗直至疾病进展或出现不可耐受的不良反应，用药期间不能进行其它抗肿瘤治疗。

联合用药的用法用量、剂量调整将根据临床试验确定，最终根据临床试验结果

确认添加。

# 使用方法：

给药前应观察药品是否存在颗粒物质和变色，如果观察到溶液是云雾状、变色，或有可见颗粒存在，则应弃用，勿摇晃小瓶。

配制输注溶液：从含有 $0 . 9 \%$ 氯化钠溶液的 $2 5 0 ~ \mathrm { m L }$ 输液袋中弃置与应当抽取的TQB2450 注射液等体积的生理盐水，向输液袋中加入对应体积的 TQB2450 注射液，轻轻倒置容器，不要摇晃，确保溶液充分混合。

静脉输注：使用带有滤膜（ $0 . 2 2 \mu \mathrm { m }$ ）输液器完成静脉输注，输注完毕之后应立即使用 $2 0 \mathrm { m L } 0 . 9 \%$ 氯化钠溶液进行冲管，从开始输注到冲管结束的静脉输注时间控制在60 min（±10min）。如果发生输液反应，需根据方案采取恰当的干预措施。避免混合其他药物使用，避免静脉推注。

输注溶液的贮存：由于本产品不含防腐剂，一旦制备好输注溶液应立即给药。如配置好的输注溶液不能立即使用，在室温放置不超过 6 小时，包括室温贮存和输注给药时间；在 $2 ^ { \circ } \mathrm { C } { - } 8 ^ { \circ } \mathrm { C }$ （36°F-46°F）冰箱放置应不超过 24小时，不要冻结，不要摇晃。

# 特殊人群：

本品尚未开展临床试验，对特殊人群的用量未知。

# 【不良反应】

本说明书描述了在临床试验中观察到的判断为可能由 TQB2450 引起的不良反应及其发生率。不良反应包括由研究者判定为与 TQB2450肯定有关、很可能有关、可能有关、可能无关以及无法判定的不良事件。由于各临床试验不是在相同条件下开展，一个临床试验中观察到的不良反应发生率与其他临床试验观察到的不良反应发生率不宜直接比较，也未必能准确预测临床实践中的真实发生率。

# 【药理毒理】

# 药理作用：

PD-L1 在肿瘤细胞和/或肿瘤浸润免疫细胞上表达，PD-L1 与 T 细胞上的 PD-1和 B7.1 受体结合，会抑制细胞毒性 T 细胞活化，T 细胞增殖和细胞因子产生。

本品是一种抗PD-L1单克隆抗体，可阻断PD-L1与PD-1和B7.1受体相互作用，释放了 PD-L1/PD-1 介导的免疫应答的抑制，包括无诱导抗体依赖的细胞毒效应的抗肿瘤免疫应答的激活。在同源鼠肿瘤模型研究中，阻断 PD-L1活性可降低肿瘤生长。

# 毒理研究：

食蟹猴静脉给予本品 50、100和 $2 0 0 \mathrm { m g / k g }$ ，每周1次，连续 4周，停药恢复 8周。结果显示，给药期及停药恢复期内，本品各组猴一般状态、体重、摄食量、体温、II 导联心电图、血压、呼吸频率、眼科检查、血液学、血生化、尿常规、骨髓

检查、补体、循环免疫复合物、淋巴细胞亚群、激素（T3、T4、TSH）、细胞因子（IFN- $\gamma$ 、TNF- $\mathbf { \nabla } \cdot \mathbf { q }$ 、IL-2、IL-6、IL-10）、脏器重量及系数、大体解剖观察及组织病理学检查均未见明显异常改变。食蟹猴静脉给予本品 20、60 和 $2 0 0 ~ \mathrm { { \ m g } }$ ，每周 1 次，连续 13 周，停药恢复 4 周。结果显示，给药期及停药恢复期内，本品各组猴一般状态、体重、摄食量、体温、心电图、血压、呼吸频率、眼科检查、血液学、血生化、尿常规、补体、循环免疫球蛋白、淋巴细胞亚群、激素（T3、T4、TSH）、细胞因子（ $\mathrm { I F N } { - } \gamma$ 、TNF- $\mathfrak { a }$ 、IL-2、IL-6、IL-8、IL-10）、脏器重量、大体解剖观察及组织病理学检查未见与给药相关的毒性反应。

本品并未进行生殖毒性研究。已上市 PD-L1药物的非临床研究证明，通过保持母体对胎仔的免疫耐受来维持妊娠是 PD-1/PD-L1 通路的主要功能之一。阻断妊娠啮齿类动物模型的PD-L1 信号通路可破坏母体对胎仔的耐受性，导致胎仔丢失增加。妊娠期间给予本品有潜在的风险，包括流产或死胎的风险增加。基于本品的作用机制，胎仔暴露于本品可增加发生免疫介导紊乱或改变正常免疫应答的风险。根据 ICH《S9：抗肿瘤药物非临床评价指导原则》，本品通过文献评估存在引起生殖发育毒性风险，无需进行生殖发育毒性试验。

本品无溶血和凝聚反应，对注射部位血管及周围组织无刺激性。

本品尚未进行致癌、致畸和致突变的研究。
