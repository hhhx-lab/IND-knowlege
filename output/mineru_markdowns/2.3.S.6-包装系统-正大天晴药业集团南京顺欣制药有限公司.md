# 2.3.S.6 包装系统

# 1 概述

本资料提供TQB2858原液包装材料的选择依据及质量标准。考虑到原液包装应密封、防潮、避光，本产品原液采用冻融袋包装储存。选择 Merck公司的Mobius 冻融储液袋，冻融袋材质为超低密度聚乙烯（ULDPE）。

对 TQB2858 两 批 中 试 生 产 规 模 （ 1000L ） 的 原 液 （ 批 号 ：

20200303、20200405）进行加速和长期稳定性考察，目前结果表明质量无明显变化，本包装与 TQB2858 原液相容性良好，可以用于本品包装。建议原液-$3 5 ^ { \circ } \mathrm { C }$ 及以下避光保存。

# 2 TQB2858 原液直接接触制品的包装材料来源及质量标准

# 2.1 包装材料及来源

来源：Merck公司

包装材料：Mobius冻融储液袋

材质：超低密度聚乙烯（ULDPE）

# 2.2 质量标准

包装材料质量标准：厂家标准。

# 2.3 包装材料的选择根据

本 研 究 对 TQB2858 两 批 中 试 生 产 规 模 （ 1000L ） 的 原 液 （ 批 号 ：

20200303、20200405）进行了加速和长期稳定性考察。目前为止，已获得本品

的加速和长期稳定性研究 0 月及加速稳定性研究 1 月数据。

加速稳定性：在 $2 { \sim } 8 ^ { \circ } \mathrm { C }$ （加速稳定性试验）条件下，TQB2858 两批原液

（批号：20200303、20200405）放置 1 个月，质量均无明显变化。

综上所述，本包装与 TQB2858 原液相容性良好，可以用于本品包装。

以上是 TQB2858 原液与包材的初步相容性研究，全面系统的包材相容性研

究，计划在后续临床试验期间进行并于报产前完成。
