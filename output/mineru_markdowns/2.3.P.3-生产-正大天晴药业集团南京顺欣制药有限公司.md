# 2 生产工艺和工艺控制

根据国家《药品生产质量管理规范（2010年修订）》要求，初步确定本品中试生产工艺为：原液（冻存原液先化冻）混匀后，经 $0 . 2 2 \mu \mathrm { m }$ 滤器预过滤后检测微生物负荷，再经 $0 . 2 2 \mu \mathrm { m }$ 滤器终端过滤除菌，过滤后的药液进行连续无菌灌装。TQB2858 注射液标示装量为 $4 . 0 \mathrm { m l } /$ 瓶，根据《中华人民共和国药典》（2015年版通则0102）要求，拟定TQB2858注射液装量为 $4 . 2 \mathrm { m l }$ /瓶，控制装量差异限度为 $\pm 3 \%$ 以内，灌装后立即进行加塞和轧盖。成品经过 $1 0 0 \%$ 灯检后，贴签入库保存。

# （1）生产工艺

本品生产工艺过程包括： $\textcircled{1}$ 原液化冻混匀； $\textcircled{2}$ 除菌过滤； $\textcircled{3}$ 半成品分批及检定； $\textcircled{4}$ 分装； $\textcircled{5}$ 轧盖； $\textcircled{6}$ 灯检； $\textcircled{7}$ 包装。具体工艺如下。

$\textcircled{1}$ 原液化冻混匀：取原液，室温避光化冻，混匀。

$\textcircled{2}$ 除菌过滤：原液经两级 0.22µmPVDF滤器除菌过滤，得半成品。一级过滤后取样检测微生物负荷。

$\textcircled{3}$ 半成品分批及检定

按《中国药典》（2015年版三部）通则之“生物制品分批规程”进行分批，并取样检测 $\mathrm { p H }$ 值、渗透压摩尔浓度、蛋白质含量、细菌内毒素和无菌。

$\textcircled{4}$ 分装

按《中国药典》（2015年版三部）通则之“生物制品分装和冻干规程”分装，在无菌条件下灌装于中硼硅玻璃注射剂瓶中，每瓶灌装量为 $4 . 2 \mathrm { m l } \pm 3 \%$ ，灌装后立即加塞。

$\textcircled{5}$ 轧盖。  
$\textcircled{6}$ 灯检。  
$\textcircled{7}$ 包装

按《中国药典》（2015年版三部）通则之“生物制品包装规程”贴签、包装，抽样检定，入库， $2 { \sim } 8 ^ { \circ } \mathrm { C }$ 保存。
