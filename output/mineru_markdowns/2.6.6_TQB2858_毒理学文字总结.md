# 2.6.6.1 概要

本文所有的毒理学试验是按照中华人民共和国国家食品药品监督管理局《药物非临床研究质量管理规范》在 GLP 条件下完成的，未出现明显影响试验质量或完整性的因素。

本研究完成了本品食蟹猴单次给药毒性试验、食蟹猴 4 周重复给药毒性试验、家兔体外溶血试验、局部刺激性试验（伴随在食蟹猴 4 周重复给药毒性试验）。单次给药毒性试验、4 周重复给药毒性试验中均选择食蟹猴为评价动物，是本品的相关种属。

食蟹猴单次静脉注射本品的最大耐受剂量（MTD）为 $1 0 0 0 \mathrm { m g / k g _ { \circ } }$ 。

重 复 给 药 毒 性 试 验 中 ， 给 药 结 束 时 ， 本 品 60 、 $2 0 0 ~ \mathrm { m g / k g }$ 组 可 见 轻 度 以RBC、HGB、HCT 降低为主的贫血样改变以及 RET、RET%升高， $2 0 0 \mathrm { m g / k g }$ 组还可见骨髓涂片轻度红细胞系、晚幼红细胞比例升高。组织病理学检查骨髓及脾脏未见相关改变。TGF-β1 具有促进红系祖细胞增殖、分化，以及促进血红蛋白生成的生理作用。结合 TGF-β1 的生理作用及供试品的作用机制，上述改变与本品药理作用相关，其中RET、 $R { \mathrm { E T } } \%$ 、红细胞系、晚幼红细胞升高为代偿性改变。至恢复期结束， $2 0 0 \mathrm { m g / k g }$ 组 1 只雌猴 HGB、HCT 仍未见明显恢复，红细胞形态学见血红蛋白含量降低。上述症状恢复较慢，可能与该只猴给药后药理学效应（如 TGF- $\boldsymbol { \cdot } \beta \boldsymbol { 1 }$ 的变化情况）维持时间较长及恢复期结束时仍可检测到一定的药物浓度相关联。此外，各给药组其余猴上述指标均可见不同程度恢复。

重复给药毒性试验中，给药结束时， ${ \geqslant } 2 0 ~ \mathrm { m g / k g }$ 组甲状腺轻微单核细胞浸润，${ \geqslant } 6 0 \ \mathrm { m g / k g }$ 组肾脏轻微~轻度单核细胞浸润， $2 0 \mathrm { m g / k g }$ 及 $2 0 0 \mathrm { m g / k g }$ 组脑膜轻微单核细胞浸润。已上市 PD-L1 单抗研究资料中也报道了类似的多脏器单核细胞浸润，为免疫相关不良反应，故认为上述改变与本品相关。恢复期结束，除 $2 0 0 \mathrm { m g / k g }$ 组脑膜病变未见明显恢复外，其余脏器病变均可见一定程度恢复。

食 蟹 猴 静 脉 输 注 本 品 后 ， 各 剂 量 组 分 别 有 9（9/10 比 例 ） 、9（9/10 比 例 ）、3（3/10 比例）只动物给药后检出抗药抗体。最高非严重毒性剂量（HNSTD）为 200mg/kg，末次给药后雄、雌猴的暴露量分别为 210±40.8、209±86.7 h*mg/mL。

本品无明显的局部刺激性，在体外无溶血或红细胞聚集作用。

# 2.6.6.2 单次给药毒性

本试验共 6 只食蟹猴，每组 3 只，雌雄兼有，设 2 个组，分别单次静脉注射300、1000mg/kg 本品，观察 14 天。试验期间对一般状况和体重、摄食量、体温、II 导联心电图及血压、血液学、血液生化、尿液等指标进行检测，试验结束时进行大体解剖观察。

试验期间动物未出现死亡或濒死现象，各项指标未见异常，食蟹猴单次静脉注射本品的最大耐受剂量（MTD）为 $1 0 0 0 \mathrm { m g / k g _ { \circ } }$ 。

# 2.6.6.8 局部耐受性

# 2.6.6.8.1局部刺激性（伴随在食蟹猴静脉注射本品4周毒性试验进行）

本试验为伴随“食蟹猴静脉注射本品 4 周毒性试验”进行的血管刺激性研究，本试验设 4 个组，分别为对照组及本品 20、60、200 mg/kg 组，每组 10 只食蟹猴，雌雄各半。每周给药 1 次、连续 4 周（共给药 5 次），停药恢复 6 周。给药期及恢复期，每天观察各组猴一般状况及注射部位。末次给药后 7 天及恢复期结束，每组分别取 6、4只猴（雌雄各半）麻醉后施以安乐死，对注射部位及周围皮下组织进行大体解剖观察及组织病理学检查。

在本试验条件下，食蟹猴每周 1 次、连续 4 周（共 5 次）静脉注射 2、6、20mg/mL的本品（剂量分别为 20、60、200 mg/kg）并停药恢复 6 周，给药结束及恢复期结束，各给药组猴注射部位及周围皮下组织均未见与供试品相关的刺激性。

# 2.6.6.8.2兔红细胞的体外溶血试验

浓度为 $3 0 ~ \mathrm { m g / m L }$ 的本品在 3h 内所有观察时间点内对兔红细胞无溶血和凝聚作用体外溶血试验结果为阴性。

# 2.6.6.10 讨论和结论

在单次给药毒性试验中，食蟹猴单次静脉注射 300、1000mg/kg 本品的 MTD 为1000 mg/kg。

在重复给药毒性试验中，食蟹猴每周 1 次、连续 4 周，共给药 5 次，静脉注射20、60、 $2 0 0 \mathrm { m g / k g }$ 本品，停药恢复 6 周，HNSTD 为 $2 0 0 \mathrm { m g / k g } \mathrm { _ c }$ 。各剂量下均可见药理作用相关的轻度外周血 RBC、HGB、HCT 降低，胸腺轻微~轻度皮质淋巴细胞减少及脑膜、甲状腺和肾脏轻微单核细胞浸润，恢复期结束可见一定程度恢复。各剂量组均检出抗药抗体。

本品在体外无溶血或红细胞聚集作用，无明显的局部刺激性。
