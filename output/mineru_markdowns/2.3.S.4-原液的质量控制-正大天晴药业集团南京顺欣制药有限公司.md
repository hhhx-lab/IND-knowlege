# 2 分析方法

# 2.1 相对结合活性

本方法采用的是酶联免疫吸附测定（ELISA）方法。将 PD-L1 蛋白、TGF-β 蛋

白分别包被在微孔板上，清洗封闭，然后将参比品和供试品稀释至特定浓度，与包被蛋白孵育结合，清洗后加入HRP标记羊抗人IgG抗体孵育，洗涤微孔板以除去未结合的反应物，加入底物（TMB）进行显色反应，用终止液终止，在 $4 5 0 \mathrm { n m } /$ 650nm 的波长处测定吸光度。参比品、供试品和包被蛋白的结合量与吸光度值呈正相关。通过四参数 Logistic 曲线拟合获得参比品和供试品 EC50 值，并计算参比品和供试品 EC50 的比值，得出供试品抗 PD-L1 端与 PD-L1 蛋白、TGF-β 受体端与 TGF- $\cdot \beta$ 蛋白的相对结合活性。采用该方法完成了TQB2858三批原液（批号：20191227、20200303、20200405）的放行。

# 2.2 肽图

肽图分析方法的原理主要基于不同蛋白质的氨基酸序列（一级结构）不同，经蛋白酶酶切后产生不同的肽段，这些肽段在反相色谱柱中的保留时间不同，经梯度洗脱后依次被检出，从而得到具有特征肽段的肽谱图，根据肽谱图来鉴别蛋白质。本法依据《中国药典》（2015 年版）通则 3405，色谱柱为 ACQUITY UPLC®Peptide BEH300 C18（300Å,1.7μm,2.1*150mm）色谱柱或性能相当的色谱柱，柱温为 $4 0 ^ { \circ } \mathrm { C }$ ；以 A 相（ $\mathrm { \it { 0 . 1 \% } }$ 甲酸/水）、B 相（ $0 . 1 \%$ 甲酸/乙腈）为流动相；检测波长为

$2 1 4 \mathrm { n m } _ { \circ }$ 。供试品经变性、还原和烷基化，按 15:1（ $\left( \mathrm { m g / m g } \right)$ ）加入质谱级胞内蛋白酶（酶切缓冲液：0.05mol/L Tris-HCl）， $3 7 ^ { \circ } \mathrm { C }$ 孵育 $2 2 \small { \sim } 2 6 \mathrm { h }$ ，最后加入 $2 0 \%$ 甲酸溶液终止酶切。上样前 4℃、12000rpm 离心 $5 { \sim } 1 5 \mathrm { m i n }$ ，取上清液作为供试品溶液。取供试品溶液 $2 0 \mu \mathrm { l }$ 注入液相色谱仪，记录 peak1~peak3 的相对保留时间。并且采用该方法完成了 TQB2858 三批原液（批号：20191227、20200303、20200405）的放行检测。

# $2 . 3 \mathbf { p H }$ 值

依据《中国药典》（2015年版）通则0631进行pH值检测。取供试品，使用酸度计直接测定 pH 值，测定 2 次，取平均值为其 pH 值。采用该方法完成了TQB2858 三批原液（批号：20191227、20200303、20200405）的放行检测。

# 2.4 分子排阻色谱法

分子排阻色谱法是根据待测组分的分子大小进行分离的一种液相色谱技术。其分离原理是凝胶色谱柱的分子筛机制。药物分子进入色谱柱后，不同组分按其分子大小进入相应的孔径内，大分子物质在色谱柱中的滞留时间较短，出峰较早小分子物质在色谱柱中滞留时间较长，出峰较晚。本法依据《中国药典》（2015年版 ） 通 则 0514 ， 色 谱 柱 为 ACQUITY UPLC Protein BEH

SEC（200Å， $1 . 7 \mu \mathrm { m }$ ， $4 . 6 ^ { * } 3 0 0 \mathrm { m m }$ ）色谱柱或性能相当的色谱柱；流动相为50mmol/L 磷酸盐缓冲液-200mmol/L 氯化钠，pH7.0；流速为每分钟 $0 . 3 \mathrm { m l }$ ；检测波长为 $2 8 0 \mathrm { n m } _ { \circ }$ 。用流动相将供试品稀释至每 $\mathrm { l m l }$ 中约含蛋白质 $1 0 \mathrm { m g }$ ，作为供试品溶液，取供试品溶液 $2 \mu \mathrm { l }$ （ $2 0 \mu \ g$ 蛋白质）注入液相色谱仪。采用面积归一化法计算高分子杂质和免疫球蛋白单体含量。采用该方法完成了 TQB2858 三批原液（批号：20191227、20200303、20200405）的放行检测。

# 2.5 CE-SDS 还原电泳

依据《中国药典》（2015年版）通则3127，通过加入还原剂将抗体分子中的二硫键打开，进而分离蛋白的轻链（LC）、重链（HC）以及降解物等杂质。该方法使用面积归一化法计算峰面积百分比，进而计算降解物含量。采用该方法完成了TQB2858 三批原液（批号：20191227、20200303、20200405）的放行检测。

# 2.6 蛋白质 A 残留量

蛋白质A残留量检测采用夹心酶联免疫吸附试验（ELISA）的方法。在微孔板中，样品中的蛋白质A被板孔中包被的蛋白质A抗体捕获，随后与生物素标记的蛋白质A抗体结合形成一个固相的抗体-蛋白质A-生物素标记抗体的三明治

复合物，复合物再与辣根过氧化物酶（HRP）偶联的链霉亲和素结合形成活性酶标复合物。加入底物（TMB）进行显色反应，孵育适当时间后用终止液终止反应，在 $4 5 0 \mathrm { n m }$ 的波长处测定吸光度。吸光度值与样品中的蛋白质 A 的浓度呈正相关。采用该方法完成了 TQB2858 三批原液（批号：20191227、20200303、20200405）的放行。

# 2.7 宿主细胞DNA残留量

本实验采用磁珠法核酸抽提结合荧光定量PCR（qPCR）核酸定量的方法对TQB2858原液宿主细胞DNA残留量进行检测。用宿主细胞残留 DNA样本前处理试剂盒对样品中CHO细胞DNA进行提取，用CHO残留DNA检测试剂盒对CHO DNA标准品、经提取后的样品、阴性对照和质控品进行定量，并根据所得数据计算样品中宿主细胞DNA残留量。采用该方法完成了TQB2858三批原液（批号：20191227、20200303、20200405）的放行。

# 2.8 宿主细胞蛋白质残留量

宿主细胞蛋白质残留量检测采用夹心酶联免疫吸附试验（ELISA）的方法。在微孔板上，样品中的宿主细胞蛋白质（HCP）同时与游离的辣根过氧化物酶

（HRP）偶联，HCP多克隆抗体及板孔中包被的抗HCP捕获抗体相互作用，形成

一个固相的抗体-HCP-酶标抗体三明治复合物。加入底物（TMB）进行显色反应，

孵育适当时间后终止液终止反应，在 $4 5 0 \mathrm { n m } / 6 5 0 \mathrm { n m }$ 的波长处测定 OD 值。OD 值

与样品中 HCP 的浓度呈正相关。采用该方法完成了 TQB2858 三批原液（批号：

20191227、20200303、20200405）的放行

# 2.9 生物学活性

# 2.9.1 PD-L1

本实验采用报告基因的方法。根据 PD-1/PD-L1免疫疗法的作用机制来针对PD-1或PD-L1设计特定的抑制剂，阻止PD-1和PD-L1的识别过程，部分恢复T细胞功能，通过阻断PD-1/PD-L1信号通路使肿瘤细胞死亡。本实验以CHO-PD-L1-CD3L 作为靶细胞；Jurkat-PD-1-NFAT 作为效应细胞。PD-1 与 PD-L1 结合可以阻断CD3下游信号的转导，从而抑制萤光素酶的表达，当加入PD-1/PD-L1抗体时，这种Block效应被反转，萤光素酶可以表达，从而检测到萤光信号。采用该方法完成了 TQB2858 三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2.9.2 TGF-β

本方法采用的是酶联免疫吸附测定（ELISA）方法。将TGF- $\beta$ 蛋白包被在微孔板上，清洗封闭，然后将参比品和供试品稀释至特定浓度，与包被蛋白孵育结合，清洗后加入 HRP 标记羊抗人 IgG 抗体孵育，洗涤微孔板以除去未结合的反应物，加入底物（TMB）进行显色反应，用终止液终止，在 450nm/650nm 的波长处测定吸光度。参比品、供试品和包被蛋白的结合量与吸光度值呈正相关。通过四参数 Logistic 曲线拟合获得参比品和供试品 EC50 值，并计算参比品和供试品 EC50的比值，得出供试品 TGF- $\beta$ 受体端与 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 蛋白的相对结合活性。采用该方法完成了 TQB2858 三批注射液（批号：BP20200101、BP20200502、BP20200503）的放行。

# 2.10 蛋白质含量

本方法采用紫外-可见分光光度法对TQB2858样品进行蛋白质含量测定。由于蛋白质分子中含酪氨酸、色氨酸和苯丙氨酸等苯环结构，在 $2 8 0 \mathrm { n m }$ 波长附近有最大吸收峰，根据朗伯-比尔定律，蛋白溶液在某一固定紫外波长的吸光度值（A）与蛋白质的消光系数（ε）、蛋白质含量（C）以及光透过路径（L）存在一定关系：$_ { \mathrm { A } = \varepsilon \mathrm { C L } }$ 。因此，蛋白溶液的蛋白质含量可以通过固定波长的吸光度值、比色皿的

光径和该蛋白物质的消光系数计算获得。本法依据《中国药典》（2015年版）通则0401，用磷酸盐缓冲液（量取200mmol/L 磷酸氢二钠溶液 81ml、200mmol/L 磷酸二氢钠溶液 $1 9 \mathrm { m l }$ ，加水约 $8 0 0 \mathrm { m l }$ ，混匀，用磷酸调节 pH 值至 $7 . 3 { \pm } 0 . 1 $ ，加水至$1 0 0 0 \mathrm { m l }$ ，混匀）将供试品稀释至每 lml 中约含蛋白质 $0 . 5 \mathrm { m g }$ ，作为供试品溶液；以磷酸盐缓冲液作为空白，测定供试品溶液在波长 $2 8 0 \mathrm { n m }$ 处的吸光度，以消光系数为 $1 . 3 0 \mathrm { A U / m g / m l }$ 计算供试品溶液的蛋白质含量，再乘以稀释倍数即得。采用该方法完成了 TQB2858 三批原液（批号：20191227、20200303、20200405）的放行检测。

# 2.11 细菌内毒素检查

依据《中国药典》（2015年版）通则1143，内毒素与鲎试剂混合后可以引起鲎试剂发生一系列的反应，最终激活凝固酶，使鲎试剂中的凝固蛋白原变为凝固蛋白，产生凝集反应。凝集反应的程度与内毒素的含量是相关的。内毒素测定时首先需要确定最大有效稀释倍数。计算公式为：MVD（最大有效稀释倍数） ${ = } _ { \mathrm { C L } } / \lambda$ ，其中L为产品需控制的内毒素限值，c为样品溶液的浓度，λ为鲎试剂的灵敏度。将待测样品按上述方法计算的MVD进行稀释后与鲎试剂在 $3 7 ^ { \circ } \mathrm { C }$ 的条件下孵育

1小时，若混合液凝结坚实，可以达到翻转不掉落的程度，结果记为阳性 $^ { 6 6 } + \boldsymbol { ^ { 5 } }$ ，表明内毒素含量高于给定的限值，否则结果为阴性“-”，表明内毒素含量低于给定的 限 值 。 采 用 该 方 法 完 成 了 TQB2858 三 批 原 液 （ 批 号 ：20191227、20200303、20200405）的放行。

# 2.12 糖谱

糖基化修饰是糖链在糖基转移酶作用下结合到蛋白质特定氨基酸残基上，是存在于真核细胞中的一种重要蛋白质翻译后修饰。经过糖基化修饰的蛋白上的糖型种类及相对含量比例均能影响蛋白质的生物学活性、半衰期和结构稳定性。特定的糖型种类及含量关系到许多糖蛋白类药物的安全性和有效性。依法测定《中国药典》（2015 年版）通则 0512。色谱柱为 ACQUITY UPLC Glycan BEHAmide（ $( 2 . 1 ^ { * } 1 0 0 \mathrm { m m }$ ，130Å， $1 . 7 \mu \mathrm { m } )$ ）色谱柱或性能相当的色谱柱，柱温为 $6 0 ^ { \circ } \mathrm { C }$ ；以A相（100mmol/L pH4.5甲酸铵水溶液）、B相（乙腈）为流动相；检测波长：荧光激发波长为 $3 3 0 \mathrm { n m }$ 、发射波长为 $4 2 0 \mathrm { n m } _ { \circ }$ 。供试品经置换、变性，按 1:3000（mg:U）加入 PNGase F 酶， $3 7 ^ { \circ } \mathrm { C }$ 保温 $1 7 \mathrm { h } { \sim } 2 4 \mathrm { h }$ ，经固相萃取柱净化，真空干燥，2-AB标记试剂标记样品， $7 5 \mathrm { ^ \circ C }$ 保温 $1 . 0 \mathrm { h } \pm 0 . 5 \mathrm { h }$ ，依次加入水 $2 0 \mu \mathrm { l }$ 、乙腈 $3 0 \mu \mathrm { l }$ 稀释，以

4℃、12000rpm 离心 $1 0 \mathrm { { m i n } }$ ，取上清液作为供试品溶液。取供试品溶液 $2 \mu \mathrm { l }$ 注入液

相色谱仪，用面积归一化法计算 M5 糖型的含量。采用该方法完成了 TQB2858 三

批原液（批号：20191227、20200303、20200405）的放行检测。
