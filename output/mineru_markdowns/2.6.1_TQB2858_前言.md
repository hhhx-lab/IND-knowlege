# 2.6.1 前言

本品是一种双功能的融合蛋白（研发代号 TQB2858），临床拟用于晚期恶性实体/血液肿瘤患者静脉输注给药。拟定的临床起始剂量为 30mg/人 q3w，后续按剂量递增方案进行剂量爬坡，直至确定 MTD 或研究者认为无需继续剂量爬坡为止。

本品为 PD-L1/TGF- $\beta$ 双功能融合蛋白，由两个部分构成的。一端是能够识别结合 PD-L1 的抗体结构（Y），是本司的 TQB2450，处于临床 III 期的PD-L1 抗 体 ； 另 一 端 是 可 以 结 合 TGF- $\cdot \beta$ 的 TGF- $\beta$ 受 体 II 型 融 合 蛋 白（Trap），可以捕获游离的 TGF-β1。PD-L1 表达在多种肿瘤组织上，与 T 细胞上的受体 PD-1 结合负向调控免疫应答。多种因素可诱导肿瘤细胞上 PD-L1 的表 达 ， 同 时 表 达 的 PD-L1 有 利 于 肿 瘤 的 发 生 和 生 长 。 转 化 生 长 因 子 -$\beta$ （transforming growth factor- $\beta$ ，TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ ）是一种多功能细胞因子，共有 3 个亚型，即 TGF- $\boldsymbol { \cdot } \beta \boldsymbol { 1 }$ 、TGF- $\cdot \beta 2$ 、TGF- $\cdot \beta 3$ 。在晚期肿瘤阶段，肿瘤微环境中癌细胞、间质成纤维细胞和内皮细胞等大量释放的 TGF-β 能够使癌细胞具有侵袭性、扩散性、干细胞特性以及对治疗产生耐药性。本品能够同时阻断 PD-1/PD-L1 通路及 TGF- $\beta$ 通路从而促进免疫细胞对肿瘤细胞的杀伤作用。

# 2.6.1.2 药理作用及作用机制

本品为 PD-L1/TGF- $\beta$ 双功能融合蛋白，由两个部分构成的。一端是能够识别结合 PD-L1 的抗体结构（Y），是本司的 TQB2450，处于临床 III 期的PD-L1 抗 体 ； 另 一 端 是 可 以 结 合 TGF- $\cdot \beta$ 的 TGF- $\beta$ 受 体 II 型 融 合 蛋 白（Trap），可以捕获游离的 TGF- $\beta 1$ 。PD-L1 表达在多种肿瘤组织上，与 T 细胞上的受体 PD-1 结合负向调控免疫应答。多种因素可诱导肿瘤细胞上 PD-L1 的表 达 ， 同 时 表 达 的 PD-L1 有 利 于 肿 瘤 的 发 生 和 生 长 。 转 化 生 长 因 子 -$\beta$ （transforming growth factor- $\beta$ ，TGF- $\cdot \beta$ ）是一种多功能细胞因子，共有 3 个亚型，即 TGF- $\boldsymbol { \cdot } \beta \boldsymbol { 1 }$ 、TGF- $\cdot \beta 2$ 、TGF- $\cdot \beta 3$ 。在晚期肿瘤阶段，肿瘤微环境中癌细胞、间质成纤维细胞和内皮细胞等大量释放的 TGF-β 能够使癌细胞具有侵袭性、扩散性、干细胞特性以及对治疗产生耐药性。本品能够同时阻断 PD-1/PD-L1 通路及 TGF- $\beta$ 通路从而促进免疫细胞对肿瘤细胞的杀伤作用。

因此，在靶向中和肿瘤微环境的 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 基础上抑制 PD-1/PD-L1 通路，可以使 T 细胞恢复活性，增强免疫应答，更有效地提高抑制肿瘤发生和发展的效果。

# 2.6.1.3 药物临床拟用适应症及用法用量

【适应症】

用于晚期恶性实体/血液肿瘤患者。

【规格】

200mg（4ml）/瓶

【用法用量】

用法：将受试者所接受剂量的本品用生理盐水 $ { \mathrm { [ 0 . 9 \% ( w / v ) } }$ 氯化钠溶液，下同】稀释至 $2 0 0 \mathrm { m l }$ （最终浓度 $0 . 1 \mathrm { m g / m l } { \sim } 4 0 \mathrm { m g / m l }$ ），使用 $0 . 2 \mu \mathrm { m }$ 或 $0 . 2 2 \mu \mathrm { m }$ 的在线过滤器的输液器将静脉输液袋中的全部内容物经静脉输注入体内，输注时间为 $6 0 \pm 1 0 \mathrm { m i n }$ 。输注时间以开始滴注本品输注液为起点，本品输注液滴注结束且生理盐水冲管结束为终点（建议用 $2 0 \mathrm { m l }$ 生理盐水冲管）。

用量：拟定的临床起始剂量为 30mg/人 q3w，后续按剂量递增方案进行剂量爬坡，直至确定 MTD 或研究者认为无需继续剂量爬坡为止。
