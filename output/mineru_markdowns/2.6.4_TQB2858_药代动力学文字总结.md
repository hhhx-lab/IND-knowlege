# 2.6.4.1 概要

本项目在食蟹猴中检测了 TQB2858 注射液单次静脉注射的药代动力学特征。采用经方法学验证的 ELISA 法测定食蟹猴血清中本品的血药浓度。药代动力学生物分析方法学验证遵循 GLP 法规（与毒代动力学采用同一套方法验证），吸收试验为 Non-GLP条件。

药代动力学试验选择食蟹猴，是本品的相关种属，与毒理学研究的动物种属一致，给药制剂一致。

所有药代动力学试验均由成都华西海圻医药科技有限公司开展。项目严格按照相关试验方案以及标准操作规程（SOP）执行。

如下简要总结了药物动力学研究的主要发现。在第 2.6.4.3 节中，讨论了这些研究的具体设计和结果。第 2.6.4.9 节“讨论和结论”中对试验结果做了整体评价。试验研究的图表包含于模块 2.6.5 报告中。

# 2.6.4.2 分析方法（试验编号：A2020004-MV-BA001-01）

药代动力学试验采用验证过的 ELISA 方法检测两组动物的血清样本中的本品，并对方法的标准曲线与精密度、准确度、选择性、稀释线性、“HOOK”效应、样本稳定性、平行性以及特异性进行了验证，验证遵循 NMPA 的 GLP 规范（与毒代动力学共用同一套方法学验证），为毒理、药代试验生物分析提供论据。

结 果 表 明 ， 上 述 指 标 均 满 足 方 法 学 验 证 的 可 接 受 标 准 。 本 品 的 定 量 范 围 为$3 1 . 3 { \sim } 4 0 0 0 ~ \mathrm { n g / m L }$ ，定量下限为 $3 1 . 3 \mathrm { n g / m L }$ ；本品 $1 5 { \sim } 2 5 ~ ^ { \circ } \mathrm { C }$ 存放 17.6 小时、 $2 { \sim } 8 ~ ^ { \circ } \mathrm { C }$ 存放约 24.5 小时、4 次反复冻融（ ${ } ^ { - 6 6 } { } ^ { \circ } \mathrm { C }$ 以下 /室温反复冻融）、经处理后的血清样本于实验台放置约 11.7 小时/再于 $2 { \sim } 8 ~ ^ { \circ } \mathrm { C }$ 放置约 2.7 小时（此处以最短时间为准）、长期冻存32 及 79 天（已涵盖实际的样本储藏期）稳定性结果符合要求，表明样本在以上条件下存放稳定，可用于食蟹猴血清中本品浓度的测定。

# 2.6.4.3 吸收（试验编号：A2020004-K012-01、A2020004-T014-01）

本试验共用 18 只食蟹猴，分为 3 组，每组 6 只，雌雄各半，分别为本品单次静脉输注 1、10、60mg/kg。单次给药组于给药前 $0 \mathrm { h }$ ，给药开始后

1min、3h、8h、24h、48h、72h、120h、168h、216h、264h、336h、504h、672h 采血检测血药浓度。单次给药组于给药前 0h 及给药后 168 h、336h、672h 检测免疫原性。

ELISA 法测定食蟹猴血清中本品血药浓度和时间变化，ELISA 法测定血清中抗本品抗体的免疫原性，所有方法均经过方法学验证。

表 2.6.4:1 及图 2.6.4:1 结果显示：单次静脉输注 1、10 及 $6 0 \mathrm { m g / k g }$ 本品后，各给药组主要药代动力学参数无明显性别差异。血清药物浓度水平随给药剂量增加而增加，食蟹猴体内 $\mathrm { { C } _ { \mathrm { { m a x } } } }$ 的增长与剂量的增长成正比例。在 $1 { \sim } 1 0 \ \mathrm { m g / k g }$ 范围内，本品在血清中的暴露量（AUC）增加略高于剂量增加比例；在 $1 0 { \sim } 6 0 ~ \mathrm { m g / k g }$ 范围内，暴露量增加与剂量增加比例基本一致。

# 2.6.4.9 讨论和结论

单次静脉输注 1、10 及 $6 0 \mathrm { m g / k g }$ 本品后，各给药组主要药代动力学参数无明显性别差异。血清药物浓度水平随给药剂量增加而增加，食蟹猴体内 $\mathrm { { C } _ { \mathrm { { m a x } } } }$ 的增长与剂量的增长成正比例。在 $1 { \sim } 1 0 \ \mathrm { m g / k g }$ 范围内，本品在血清中的暴露量（AUC）增加略高于剂量增加比例；在 $1 0 { \sim } 6 0 ~ \mathrm { m g / k g }$ 范围内，暴露量增加与剂量增加比例基本一致。4 周重复给药毒性试验剂量与药代动力学剂量接近，因此重复给药的药代动力学结果参考毒代动力学结果。
