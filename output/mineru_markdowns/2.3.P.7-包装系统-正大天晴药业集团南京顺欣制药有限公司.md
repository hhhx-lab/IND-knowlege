等相关技术指导原则，制定了初步的药物与包材的相容性考察方案，考察包装

用材料容器对药物稳定性的影响，评定其在长期贮存过程中、在不同环境条件

下（如温度、光线）容器（材料）对药物的保护效果和药物包材相容性。并采

用色水法进行包装材料的气密性研究。

考察研究结果显示：（1）所选包装材料气密性良好，可以有效防止药品在

长期贮存过程中被污染。（2）已开展本品的加速和长期稳定性研究，目前已获

得加速稳定性研究1月数据。加速稳定性研究结果表明，在温度 $2 5 ^ { \circ } \mathsf { C } { \pm } 2 ^ { \circ } \mathsf { C } .$ 、相

对湿度 $6 0 \% \pm 1 0 \%$ （加速稳定性试验）条件下，TQB2858两批中试生产规模注射液（批号：BP20200502、BP20200503）放置1个月，高分子杂质含量和降解物含量略有增长，其余检测指标均无明显变化。分析是本品对温度敏感。将积累后续稳定性研究数据进行分析。（3）影响因素试验方案已拟定，计划取一批中试生产规模的注射液进行影响因素试验，即将进行研究。
