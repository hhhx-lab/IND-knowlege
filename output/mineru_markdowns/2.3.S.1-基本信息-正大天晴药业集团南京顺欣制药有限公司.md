# 2 结构

# 2.1 分子式和分子量

分子式： $\mathrm { C _ { 7 8 6 2 } H _ { 1 2 1 5 0 } N _ { 2 0 9 0 } O _ { 2 4 8 4 } S _ { 7 8 } }$ （无翻译后修饰）

分子量：178162Da（重链 N 末端环化，非糖基化）

# 2.2 结构式

本研究的TQB2858由1634个氨基酸组成，包含两条轻链和两条重链，其中重链第 $1 { \sim } 4 4 6$ 位区域和轻链为 PD-L1 抗体区域，重链第 $4 4 7 { \sim } 4 6 6$ 位区域为Linker 区域，重链第 $4 6 7 { \sim } 6 0 3$ 位区域为TGFβRII的胞外区，结构示意图见图2-1。

本研究的TQB2858重链含有4个理论的N-糖基化位点，分别为第296位、

第 514 位、第 538 位和第 598 位的天冬酰胺（N296、N514、N538 和 N598），

其中 N296 位于 PD-L1 抗体区域，N514、N538 和 N598 位于 TGFβRII 的胞外区

经特性鉴定研究，N296、N514 和 N538 三个位点均检测到 N-糖基化修饰，而

N598 位点未检测到 N-糖基化修饰。根据文献调研，N598 位点是否 N-糖基化对

于 TGF-β 的结合活性无影响，基于本研究中生物学活性研究结果，可认为 N598

未发生N-糖基化修饰的风险可控。

# 3 基本性质

TQB2858 是一种能同时靶向 PD-L1 通路和 TQF-β 通路的双特异性融合蛋白，

在哺乳动物细胞表达系统中通过重组DNA技术生产并通过包含特定病毒灭活和

移除的步骤纯化，由 1634个氨基酸组成，分子量为 178162Da（重链N末端环

化，非糖基化）。TQB2858原液为澄明液体、可带轻微乳光，对光照、温度比较敏感，为确保产品质量，在长期储存及运输过程中，需低温保存并注意避光。

PD-1/PD-L1 是一对免疫共刺激因子，PD-1/PD-L1 信号通路的激活可导致免疫抑制性肿瘤微环境形成，使肿瘤细胞逃避机体免疫监视和杀伤，而阻断 PD-1/PD-L1 信号通路可逆转肿瘤免疫微环境，增强内源性抗肿瘤免疫效应。PD-L1高表达不仅能促进肿瘤细胞的生长，还可诱导 T 淋巴细胞的凋亡。TGF-β 属于一类促进细胞生长和转化的细胞因子超家族。在癌细胞中具有多种生物学效应，可诱导免疫逃逸、促进血管形成、诱导上皮间质转化来促进肿瘤细胞的生长、增殖分化、浸润与转移。TQB2858 双抗同时靶向 PD-L1 及 TGF- $\boldsymbol { \cdot } \boldsymbol { \beta }$ 两条信号通路，可改善肿瘤免疫微环境，从而达到增强免疫治疗的疗效。
