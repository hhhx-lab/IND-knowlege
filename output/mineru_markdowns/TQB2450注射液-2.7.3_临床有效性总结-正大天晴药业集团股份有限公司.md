# 2.7.3 临床有效性总结

TQB2450 注射液（ $1 0 \mathrm { m L } { : } 1 0 0 \mathrm { m g }$ ）于2017 年01月申请临床试验研究，临床申请受理号为：CXSL1700018，已于 2017 年 10 月获得临床批件，临床批件号：2017L04914。此后，TQB2450 注射液于 2019 年 04 月提出新增规格（ $2 0 ~ \mathrm { m L }$ ： $6 0 0 \mathrm { m g }$ ）的补充申请，补充申请受理号为：CXSB1900008，该申请已于 2019 年 12 月获得临床批件，临床批件号：2019B04525。

盐酸安罗替尼胶囊 ALTER1202 研究采用随机（2:1）、双盲、安慰剂对照、多中心设计，入组经病理学确诊的、复治（既往接受过至少 2种化疗方案）的晚期小细胞肺癌患者。研究实际入组 120 例受试者，截止 2018 年 6 月 30 日，安罗替尼组和安慰剂组的中位 PFS分别为 4.07 月（95%CI：2.79-4.24）和 0.72 月（95%CI：0.69-0.82），与安慰剂相比，安罗替尼可使小细胞肺癌患者的中位 PFS 延长3.35 月，肿瘤复发风险降低 $8 0 . 8 \%$ ，差异有显著统计学意义（ $\scriptstyle \mathrm { H R = 0 } . 1 9 2$ ， $9 5 \%$ CI：0.117-0.315， $\mathrm { p { < } 0 . 0 0 0 1 }$ ）。截止 2018 年 6 月 30 日，明确死亡的终点事件共 53件（ $4 4 . 5 4 \%$ ），其中安慰剂组 21件（ $5 5 . 2 6 \%$ ），安罗替尼组32件 $( 3 9 . 5 1 \% )$ 。安罗替尼组和安慰剂组的中位 OS 分别为 7.29 月（ $9 5 \% \mathrm { C I }$ 为 6.50-10.51）和4.90 月（ $9 5 \% \mathrm { C I }$ 为2.56-6.67），两组间有统计学差异（ $\mathrm { p } { = } 0 . 0 2 1$ ）。基于本研究结果，CDE于2019年8月29日有条件批准安罗替尼用于既往至少接受过 2种化疗方案治疗后进展或复发的小细胞肺癌的治疗。

2020 年 ASCO 公布一项安罗替尼联合依托泊苷、顺铂/卡铂一线治疗广泛期小细胞肺癌的单臂、II期临床研究（摘要号 9066）。主要终点为 PFS 和ORR。推荐剂量如下：安罗替尼 12mg QD（连二停一）、依托泊苷（ $\mathrm { 1 0 0 m g / m ^ { 2 } , d 1 } { - 3 / } \mathrm { d } 2 1 )$ ）、顺铂（ $7 5 { - } 8 0 \mathrm { m g / m } ^ { 2 } , \mathrm { Q 3 W } )$ ）、卡铂（ ${ \bf A U C } = 5 { \cdot } 6$ ，Q3W）。安罗替尼联合化疗 4-6周期后，单药使用安罗替尼维持。截止2018 年 10 月至 2019 年 12 月，合计入组 27 例患者， $\mathrm { m P F S = 9 . 6 1 }$ 个月（ $9 5 \%$ Cl:7.80-11.42），$\mathrm { O R R } { = } 7 7 . 7 8 \%$ （21/27）， $\mathrm { D C R } { = } 9 6 . 3 \%$ （26/27）。安全性方面， $\geqslant 3$ 级不良事件包括中性粒细胞计数降低（ $2 2 \%$ ）、手足皮肤反应（ $1 5 \%$ ）、白细胞计数降低（ $1 1 \%$ ）、恶心（ $4 \%$ ）、口腔黏膜炎（ $4 \%$ ）等。本研究证实安罗替尼联合化疗一线治疗广泛期小细胞肺癌显著改善中位无进展生存期和客观缓解率，安全性可耐受，未见非预期不良事件。

TQB2450-Ib-04 研究是一项评估 TQB2450 联合安罗替尼治疗晚期实体瘤患者的 Ib 期临床试验，入组经标准治疗失败或无标准治疗方法的晚期实体瘤。TQB2450 固定剂量1200mg，安罗替尼进行 $1 0 \mathrm { m g / q d }$ 、 $1 2 \mathrm { m g / q d }$ 等剂量组爬坡。最终确定联合用药安全剂量为安罗替尼 $1 2 \mathrm { m g / q d }$ ，d1-14/Q3W +TQB2450 1200 mg，d1。本试验 2019 年 6 月启动，合计入组22 例受试者，其中 6 例为既往接受过至少 2 种化疗方案的晚期小细胞肺癌。截止 2020 年

4 月 10 日，整体肿瘤客观缓解率 ORR 为 $3 2 . 8 \%$ ，DCR 为 $8 1 . 8 \%$ ，6 例 SCLC 患者的 ORR为 $6 6 . 7 \%$ ，DCR 为 $8 3 . 3 3 \%$ 。本试验同时提示 TQB2450 联合安罗替尼有良好的耐受性和安全性，总体不良事件可控。
