#  非小细胞肺癌（NSCLC）

盐酸安罗替尼胶囊 ALTN-03-II 研究采用随机（1:1）、双盲、安慰剂对照、多中心设计，入组经病理学确诊的，既往至少接受过 2 线治疗失败的晚期NSCLC 患者。从2013年08月至2015年8月，全国 13家研究中心实际入组 117例受试者，安罗替尼试验组和安慰剂对照组的中位PFS 分别为 4.83月（ $9 5 \% \mathrm { C I } 3 . 4 7 { \sim } 6 . 4 0 $ ）和 1.23 月（ $9 5 \% \mathrm { C I } \ 0 . 7 0 { \sim } 1 . 6 0 )$ ，与安慰剂相比，安罗替尼可使 NSCLC 患者的中位 PFS 延长 3.6 个月，疾病进展风险降低$6 8 \%$ ，差异有统计学显著意义（ $\mathrm { H R } { = } 0 . 3 2$ ，95%CI 0.200~0.511， $\mathrm { p { < } 0 . 0 0 0 1 }$ ）。截止至2016年2 月，安罗替尼试验组和安慰剂对照组的中位 OS 分别为 9.33 月（ $9 5 \% \mathrm { C I } \ 6 . 8 0 { \sim } 1 5 . 0 7 .$ ）和6.30 月（ $9 5 \% \mathrm { C I } 4 . 2 7 { \sim } 1 0 . 5 3 $ ），与安慰剂相比，盐酸安罗替尼胶囊对OS 的延长虽然没有统计学意义，但还是使得NSCLC 患者的中位OS 延长3 个月，死亡风险降低 $2 3 \%$ （ $\mathrm { H R { = } } 0 . 7 7 6$ ，95%CI 0.511~1.176，p=0.2316）。

盐酸安罗替尼胶囊 ALTER0303 研究（ALTN-03-III）证实，与安慰剂相比盐酸安罗替尼胶囊用于晚期 NSCLC 三线及以上治疗，可以显著延长患者的 PFS 和 OS。ALTER0303研究采用随机（2:1）、双盲、安慰剂对照、多中心设计，入组经病理学确诊的，既往至少

接受过 2 线治疗失败的晚期 NSCLC 患者。研究实际入组 437 例受试者，安罗替尼试验组和安慰剂对照组的中位 PFS 分别为 5.4月（ $9 5 \%$ CI，4.4-5.6）和 1.4 月（ $9 5 \%$ CI，1.1-1.5），与安慰剂相比，安罗替尼可使 NSCLC 患者的中位 PFS 延长 4 个月，疾病进展风险降低$7 5 \%$ ，差异有统计学显著意义（ $\mathrm { H R } { = } 0 . 2 5$ ， $9 5 \% \mathrm { C I }$ ，0.19-0.31； $\mathrm { p } { < } 0 . 0 0 0 1$ ）。安罗替尼试验组和安慰剂对照组的中位 OS 分别为9.6个月（ $9 5 \%$ CI：8.2-10.6）和 6.3 月（ $9 5 \%$ CI，5.0-8.1），与安慰剂相比，盐酸安罗替尼胶囊可使 NSCLC 患者的中位OS 延长3.3个月，死亡风险降低 $3 2 \%$ ，差异有统计学意义（ $\mathrm { H R } { = } 0 . 6 8$ ，95%CI，0.54-0.87， $\mathrm { p { = } } 0 . 0 0 1 8$ ）。病理类型亚组分析结果显示，肺腺癌和肺鳞癌患者均可从盐酸安罗替尼治疗中获益。在肺腺癌患者中，盐酸安罗替尼组与安慰剂组相比 PFS 与 OS 均显著获益 6；在肺鳞癌患者中，盐酸安罗替尼组PFS 显著延长，OS 有延长趋势但无统计学意义。ALTER0303 研究中常见的不良反应包括高血压（ $6 7 . 3 5 \%$ ）、乏力（ $5 1 . 0 2 \%$ ）、TSH 升高（ $4 6 . 6 \%$ ）、食欲减退（ $4 5 . 2 4 \%$ ）、高三酰甘油血症（ $4 2 . 8 6 \%$ ）、高胆固醇血症（ $4 0 . 4 8 \%$ ）、手足皮肤反应（ $4 3 . 5 4 \%$ ）、咳嗽（ $3 7 . 4 1 \%$ ）、腹泻（ $3 5 . 0 3 \%$ ）、CGT升高（ $2 9 . 5 9 \%$ ）、蛋白尿（ $2 8 . 9 1 \%$ ）等。盐酸安罗替尼的不良反应多数可通过调整剂量、暂停给药及对症处理得到控制[1]。

#  肾细胞癌（RCC）

盐酸安罗替尼胶囊 ALTER0602A 研究（ALTN-06-IIA）已经证实，盐酸安罗替尼胶囊在初治的RCC 患者中具有良好的疗效和安全性。ALTER0602A 研究为一项随机（2:1）、舒尼替尼平行对照一线治疗晚期透明细胞型 RCC 的 II 期临床研究。自 2013 年 12 月起，由中国医学科学院肿瘤医院牵头，联合全国 17 家临床研究机构，共纳入 133例受试者。试验组给予安罗替尼（12 mg/次，给二停一方案），对照组给予舒尼替尼（ $5 0 ~ \mathrm { m g }$ /次，连四停二方案），受试者持续用药直至出现不能耐受的不良反应或治疗出现疾病进展后终止。以无进展生存期（PFS）为主要疗效终点指标。截止至 2015 年 5 月 15 日，安罗替尼组的中位PFS 为11.27个月，舒尼替尼组为 10.97个月，两组间 PFS 无显著差异（ $\mathrm { P = } 0 . 3 0 2 5$ ）。继续随访至 2016 年 4 月 15 日，安罗替尼组的中位 PFS 为 17.5 个月，舒尼替尼组为 16.6个月，两组间PFS 无显著差异（ $\mathrm { \cdot } \mathrm { P } { > } 0 . 5$ ）。继续随访至 2017年8月10 日，安罗替尼组的中位OS 为30.9个月，舒尼替尼组的中位 OS 为 30.5个月，两组间 OS 无显著差异（ $( \mathrm { P } { > } 0 . 5 )$ ）。两组ORR 分别为 $3 0 . 3 \%$ （ $9 5 \% \mathrm { C I } ~ 2 0 . 5 \% { \sim } 3 9 . 5 \%$ ）和 $2 7 . 9 \%$ （ $9 5 \% \mathrm { C I } ~ 1 4 . 5 \% { \sim } 4 1 . 3 \%$ ）。从有效性分析结果来看，与目前晚期 RCC 一线标准治疗舒尼替尼相比，安罗替尼具有相似的疗效。安全性数据显示，安罗替尼组的 3/4 级不良反应发生率显著低于舒尼替尼组（ $2 8 . 9 \%$ v $\phantom { } \cdot \phantom { } \phantom { } \phantom { } 5 . 5 5 . 8 \%$ ， $\mathrm { { P } { < } 0 . 0 1 }$ ）；在血小板减少和中性粒细胞减少等血液学毒性方面，安罗替尼几乎没有毒性，两组差异更加明显。安罗替尼组和舒尼替尼组更加严重的不良反应发生率分别为

$1 0 . 0 \%$ 和 $1 8 . 6 \%$ （ $\mathrm { P = } 0 . 1 8$ ）。两组均出现的最常见不良反应是高血压、手足综合征、腹泻、口腔炎和厌食症等。从安全性分析结果来看，安罗替尼的毒副作用明显低于舒尼替尼，尤其是血液学毒性[2]。基于此研究结果，2020年 CSCO 肾癌诊疗指南已将盐酸安罗替尼胶囊作为晚期RCC 一线治疗的 II 级推荐药物。

盐酸安罗替尼胶囊 ALTN-06-IIB 研究证实，盐酸安罗替尼胶囊在经索拉非尼和/或舒尼替尼治疗失败或不耐受的透明肾细胞癌患者中具有良好的疗效和安全性。ALTN-06-IIB研究为单臂、多中心 II 期临床研究，2013 年 11 月起，全国 17 家医院入组的 43 例受试者使用每天口服 $1 2 ~ \mathrm { m g }$ 盐酸安罗替尼胶囊，连 2 停 1（连续用药 2 周，停 1 周为一个治疗周期）方案，持续用药直至出现不能耐受的不良反应或治疗出现疾病进展，观察受试者的安全性和初步疗效，主要终点指标为无进展生存期（PFS）。截止 2016 年 5 月的统计时间，中位无进展生存期为14.1 月（ $9 5 \% \mathrm { C I } \ 8 . 3 3 { \sim } 2 2 . 5 0 $ ），平均用药时间达 13.8 月。6 周疾病控制率为 $9 0 . 5 \%$ （ $9 5 \%$ CI 77.4~97.3%），48 周疾病控制率为 $5 4 . 9 3 \%$ ，总客观缓解率（ $O \mathrm { R } \mathrm { R } { = } 1 9 . 0 5 \%$ ）。32 例既往接受 TKI 治疗后疾病进展的受试者，接受安罗替尼治疗后中位PFS 为8.6月，而既往因 TKI治疗不耐受的受试者，使用安罗替尼治疗后的中位 PFS 可达20个月。

#  软组织肉瘤

#  晚期肝细胞癌

2016 年 9 月至 2019 年 5 月在中国医学科学院肿瘤医院开展的一项安罗替尼在晚期HCC 的 II期单中心临床研究，分两个队列，队列一：既往未接受过系统治疗的晚期 HCC，队列二：索拉非尼治疗失败的晚期 HCC，Child-Pugh 肝功能评分 $\leqslant 8$ 分，安罗替尼 $1 2 \mathrm { m g }$ ，qd，连服 2 周停 1 周，3 周为一治疗周期，每 6 周进行一次疗效评估。共入组患者 49 例，其中一线患者 26 例，二线患者 23 例，疗效方面，一线患者 12 周 PFS 率 $8 0 . 8 \%$ ，mOS 为10.8 个月，mPFS 为 6.1 个月，二线患者 12 周 PFS 率 $6 5 . 2 \%$ ，mPFS 为 4.5 个月，mOS 暂未达到。

对于广泛期SCLC 一线治疗，也进行很多其它联合化疗方案的临床研究，但均未取代标准治疗方案。2019 年 3 月，基于 IMpower133 研究结果，FDA 批准 Atezolizumab 联合化疗用于广泛期 SCLC 一线治疗。IMpower133 研究是一项观察 Atezolizumab 联合 EC 一线治疗广泛期SCLC 的双盲、随机、安慰剂对照研究。入组了既往未接受过治疗的广泛期 SCLC患者，不要求进行 PD-L1 检测。患者 1:1 随机分配接受 4 个周期的卡铂 $^ +$ 依托铂苷联合Atezolizumab或安慰剂，21天为一个治疗周期，然后给予Atezolizumab或安慰剂维持治疗，直至疾病进展、不可耐受的毒性或不再有临床获益。共同主要终点为 OS 和研究者评估的PFS。研究结果提示，Atezo 组相对安慰剂组显著延长 SCLC 患者的mOS、mPFS，两组OS分别为 12.3 vs 10.3 个月，mPFS 分别为 5.2 vs 4.3 个月[3]。

CASPIAN 研究是一项关于广泛期 SCLC 患者一线治疗的随机、开放、全球多中心的Ⅲ期临床试验，旨在探索 Durvalumab 和 Tremelimumab 联合化疗在广泛期 SCLC 患者一线治疗中的疗效。该研究纳入了初治的广泛期 SCLC 患者，使用 Durvalumab±Tremelimumab

联合EP 方案（依托泊苷 $+$ 顺/卡铂类）化疗。主要研究终点为总生存期(OS)。最终，相较于单纯化疗组的 OS（10.3 个月），Durvalumab 联合化疗组的 OS 达到了 13.0 个月（ $\mathrm { \cdot } \mathrm { H R } { = } 0 . 7 3$ ，$\scriptstyle { \mathrm { p = } } 0 . 0 0 4 7$ ），降低 $2 7 \%$ 的死亡风险。单纯化疗组的 12 个月 OS 率为 $3 9 . 8 \%$ ，而 Durvalumab联合化疗组为 $5 3 . 7 \%$ ，提高了 $1 3 . 9 \%$ 。单纯化疗组的 ORR 为 $5 7 . 6 \%$ ，Durvalumab 联合化疗组为 $6 7 . 9 \%$ ，提高了 $1 0 . 3 \% ^ { [ 4 ] } \circ 2 0 2 0$ 年 3 月，基于 CASPIAN 研究结果，FDA 批准 Durvalumab联合化疗用于广泛期SCLC 一线治疗。

TQB2450-Ib-04 研究是一项评估 TQB2450 联合安罗替尼治疗晚期实体瘤患者的 Ib 期临床试验，入组经标准治疗失败或无标准治疗方法的晚期实体瘤。TQB2450 固定剂量$1 2 0 0 \mathrm { m g }$ ，安罗替尼进行 10mg/qd、 $1 2 \mathrm { m g / q d }$ 等剂量组爬坡。最终确定联合用药安全剂量为安罗替尼 $1 2 \mathrm { m g / q d }$ ，d1-14/Q3W +TQB2450 1200mg，d1。本试验 2019 年 6 月启动，合计入组 22 例受试者，其中 6 例为既往接受过至少 2 种化疗方案的晚期小细胞肺癌。截止 2020年4月10日，整体肿瘤客观缓解率 ORR 为 $3 2 . 8 \%$ ，DCR 为 $8 1 . 8 \%$ ，6 例 SCLC 患者的 ORR为 $6 6 . 7 \%$ ，DCR 为 $8 3 . 3 3 \%$ 。本试验同时提示 TQB2450 联合安罗替尼有良好的耐受性和安全性，总体不良事件可控。

安罗替尼药品注册批件的要求（2019 年 8 月 29 日，批件号 2019S00491/2/3）如下：“请开展确证性研究 TQB2450-III-04，递交阶段性研究报告及总结报告，将在达到主要终点后转为完全批准”。TQB2450 或安慰剂联合安罗替尼 $^ +$ 依托泊苷 $^ +$ 卡铂对比依托泊苷 $+$ 卡铂一线治疗广泛期小细胞肺癌的随机、双盲、平行对照、多中心 III期临床试验（方案编号：TQB2450-III-04）正在全国开展，第一例患者 2020 年 3 月 18 日随机入组，截止 2021 年 7月28日，本试验已入组 576例患者。
