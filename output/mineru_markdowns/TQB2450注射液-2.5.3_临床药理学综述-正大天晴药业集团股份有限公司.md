# 2.5.3.2 药效动力学

体外外周血单个核细胞（PBMC）的受体占位率（ ${ \mathrm { R O } } \%$ ）试验，选取 $0 . 0 0 3 { \sim } 0 . 3 \mu \mathrm { g / m L }$ 剂量，3~4 个供体，流式检测。结果显示， $0 . 0 6 2 5 \mu \mathrm { g / m L }$ 的 TQB2450 可使不同供体的 PBMC的 $\mathrm { R O \% }$ 均 $5 9 0 \%$ （均值 $9 9 . 0 8 \%$ ）。

I 期临床研究中（TQB2450-I-01）， $3 – 3 0 \mathrm { m g / k g }$ 和 $1 2 0 0 \mathrm { m g }$ 固定剂量共计 26 例受试者检测了给药后 $\mathrm { R O \% }$ 情况。给药21天后， $\mathrm { R O \% }$ 基本都在 $8 0 \%$ 以上，基本判定药物可维持饱和至下次给药前，以保证药效持续发挥，也说明了 3周给药一次的可行性。

# 2.5.3.3 免疫原性

I期临床研究中（TQB2450-I-01），入组 40例受试者，抗药抗体（ADA）检测阳性率为 $3 2 . 5 \%$ （13/40），随着剂量的增加，ADA 阳性率降低。ADA 阳性率的高低和检测方法的灵敏性及特异性密切相关，并且受多种因素的影响，包括分析方法、样本的处理方法、样本的收集时间、合并用药，以及患者的其他基础疾病等。因此，不同产品的 ADA 阳性率不适合直接比较。

ADA阳性受试者的血药浓度降低，随着剂量的增加影响程度减小。

ADA阳性受试者未见异常安全性信号。
